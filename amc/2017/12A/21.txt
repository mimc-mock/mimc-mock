<p>A set <img src="//latex.artofproblemsolving.com/a/d/2/ad28c83c99a8fd0dd2e2e594c9d02ee532765a0a.png" class="latex" alt="$S$" width="12" height="12" /> is constructed as follows. To begin, <img src="//latex.artofproblemsolving.com/d/f/7/df745ef918affeb99f94fa199a4dee661a68f08b.png" class="latex" alt="$S = \{0,10\}$" style="vertical-align: -4px" width="88" height="18" />. Repeatedly, as long as possible, if <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> is an integer root of some polynomial <img src="//latex.artofproblemsolving.com/8/2/d/82dfec03e116b484e8e8f33f3af5cabad4c70e18.png" class="latex" alt="$a_{n}x^n + a_{n-1}x^{n-1} + \dots + a_{1}x + a_0$" style="vertical-align: -2px" width="263" height="17" /> for some <img src="//latex.artofproblemsolving.com/6/a/a/6aaaca534ced32c0f4d583e9626459a21f489941.png" class="latex" alt="$n\geq{1}$" style="vertical-align: -2px" width="43" height="14" />, all of whose coefficients <img src="//latex.artofproblemsolving.com/4/1/2/412787c048e28774dc63fc27db42dc52ca858de7.png" class="latex" alt="$a_i$" style="vertical-align: -2px" width="14" height="10" /> are elements of <img src="//latex.artofproblemsolving.com/a/d/2/ad28c83c99a8fd0dd2e2e594c9d02ee532765a0a.png" class="latex" alt="$S$" width="12" height="12" />, then <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> is put into <img src="//latex.artofproblemsolving.com/a/d/2/ad28c83c99a8fd0dd2e2e594c9d02ee532765a0a.png" class="latex" alt="$S$" width="12" height="12" />. When no more elements can be added to <img src="//latex.artofproblemsolving.com/a/d/2/ad28c83c99a8fd0dd2e2e594c9d02ee532765a0a.png" class="latex" alt="$S$" width="12" height="12" />, how many elements does <img src="//latex.artofproblemsolving.com/a/d/2/ad28c83c99a8fd0dd2e2e594c9d02ee532765a0a.png" class="latex" alt="$S$" width="12" height="12" /> have?
</p><p><img src="//latex.artofproblemsolving.com/b/e/c/becd654280dcfeb340406a63fedc84f165957b55.png" class="latex" alt="$\textbf{(A)}\ 4 \qquad \textbf{(B)}\ 5 \qquad\textbf{(C)}\ 7 \qquad\textbf{(D)}\ 9 \qquad\textbf{(E)}\ 11$" style="vertical-align: -4px" width="357" height="17" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>At first, <img src="//latex.artofproblemsolving.com/b/0/5/b05d833d2b2ec928bdfe952d9d51ff8bff9bb1d4.png" class="latex" alt="$S=\{0,10\}$" style="vertical-align: -4px" width="88" height="18" />.
</p><p><img src="//latex.artofproblemsolving.com/0/a/3/0a3bc1888e104b19e3152897edf1b88fd3842a28.png" class="latexcenter" alt="\[\begin{tabular}{r c l c l} \(10x+10\) &amp; has root &amp; \(x=-1\) &amp; so now &amp; \(S=\{-1,0,10\}\) \\ \(-x^{10}-x^9-x^8-x^7-x^6-x^5-x^4-x^3-x^2-x+10\) &amp; has root &amp; \(x=1\) &amp; so now &amp; \(S=\{-1,0,1,10\}\) \\ \(x+10\) &amp; has root &amp; \(x=-10\) &amp; so now &amp; \(S=\{-10,-1,0,1,10\}\) \\ \(x^3+x-10\) &amp; has root &amp; \(x=2\) &amp; so now &amp; \(S=\{-10,-1,0,1,2,10\}\) \\ \(x+2\) &amp; has root &amp; \(x=-2\) &amp; so now &amp; \(S=\{-10,-2,-1,0,1,2,10\}\) \\ \(2x-10\) &amp; has root &amp; \(x=5\) &amp; so now &amp; \(S=\{-10,-2,-1,0,1,2,5,10\}\) \\ \(x+5\) &amp; has root &amp; \(x=-5\) &amp; so now &amp; \(S=\{-10,-5,-2,-1,0,1,2,5,10\}\) \end{tabular}\]" width="961" height="138" />
</p><p>At this point, no more elements can be added to <img src="//latex.artofproblemsolving.com/a/d/2/ad28c83c99a8fd0dd2e2e594c9d02ee532765a0a.png" class="latex" alt="$S$" width="12" height="12" />. To see this, let
</p><p><img src="//latex.artofproblemsolving.com/d/e/6/de65a5ecdb21fdb5846fa4fbcb7313835e7898a9.png" class="latexcenter" alt="\begin{align*} a_{n}x^n + a_{n-1}x^{n-1} + ... + a_{2}x^2 + a_{1}x + a_0 &amp;= 0 \\ x(a_{n}x^{n-1} + a_{n-1}x^{n-2} + ... + a_{2}x + a_{1}) + a_0 &amp;= 0 \\ x(a_{n}x^{n-1} + a_{n-1}x^{n-2} + ... + a_{2}x + a_{1}) &amp;= -a_0 \end{align*}" width="392" height="75" />
</p><p>with each <img src="//latex.artofproblemsolving.com/4/1/2/412787c048e28774dc63fc27db42dc52ca858de7.png" class="latex" alt="$a_i$" style="vertical-align: -2px" width="14" height="10" /> in <img src="//latex.artofproblemsolving.com/a/d/2/ad28c83c99a8fd0dd2e2e594c9d02ee532765a0a.png" class="latex" alt="$S$" width="12" height="12" />. <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> is a factor of <img src="//latex.artofproblemsolving.com/e/e/5/ee5c7c921b3334f6ecf58c241eeea42acf0ce71e.png" class="latex" alt="$a_0$" style="vertical-align: -2px" width="15" height="10" />, and <img src="//latex.artofproblemsolving.com/e/e/5/ee5c7c921b3334f6ecf58c241eeea42acf0ce71e.png" class="latex" alt="$a_0$" style="vertical-align: -2px" width="15" height="10" /> is in <img src="//latex.artofproblemsolving.com/a/d/2/ad28c83c99a8fd0dd2e2e594c9d02ee532765a0a.png" class="latex" alt="$S$" width="12" height="12" />, so <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> has to be a factor of some element in <img src="//latex.artofproblemsolving.com/a/d/2/ad28c83c99a8fd0dd2e2e594c9d02ee532765a0a.png" class="latex" alt="$S$" width="12" height="12" />. There are no such integers left, so there can be no more additional elements. <img src="//latex.artofproblemsolving.com/b/e/f/bef045639aa7cd47704534dfc2589b251b715b2d.png" class="latex" alt="$\{-10,-5,-2,-1,0,1,2,5,10\}$" style="vertical-align: -4px" width="238" height="18" /> has <img src="//latex.artofproblemsolving.com/b/f/2/bf2c9074b396e3af0dea52d792660eea1c77f10f.png" class="latex" alt="$9$" width="8" height="12" /> elements <img src="//latex.artofproblemsolving.com/d/1/c/d1c7e62aa15e0993abbccb54c86019990ff186c9.png" class="latex" alt="$\to \boxed{\textbf{(D)}}$" style="vertical-align: -10px" width="64" height="28" />
</p>
