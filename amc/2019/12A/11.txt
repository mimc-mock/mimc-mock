<p>For some positive integer <img src="//latex.artofproblemsolving.com/8/c/3/8c325612684d41304b9751c175df7bcc0f61f64f.png" class="latex" alt="$k$" width="9" height="12" />, the repeating base-<img src="//latex.artofproblemsolving.com/8/c/3/8c325612684d41304b9751c175df7bcc0f61f64f.png" class="latex" alt="$k$" width="9" height="12" /> representation of the (base-ten) fraction <img src="//latex.artofproblemsolving.com/7/0/c/70c01fbc67f37ea7e9d653ca7bdc5a81b149fa44.png" class="latex" alt="$\frac{7}{51}$" style="vertical-align: -13px" width="20" height="37" /> is <img src="//latex.artofproblemsolving.com/a/c/c/accdad507f69e05d5edf22370a7efa423ee49d54.png" class="latex" alt="$0.\overline{23}_k = 0.232323..._k$" style="vertical-align: -2px" width="155" height="17" />. What is <img src="//latex.artofproblemsolving.com/8/c/3/8c325612684d41304b9751c175df7bcc0f61f64f.png" class="latex" alt="$k$" width="9" height="12" />?
</p><p><img src="//latex.artofproblemsolving.com/6/9/7/69736c0dad97fe800934341e3dd15a9501331ffd.png" class="latex" alt="$\textbf{(A) } 13 \qquad\textbf{(B) } 14 \qquad\textbf{(C) } 15 \qquad\textbf{(D) } 16 \qquad\textbf{(E) } 17$" style="vertical-align: -4px" width="398" height="17" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>We can expand the fraction <img src="//latex.artofproblemsolving.com/2/5/1/25166078daa09b120d2ef195d4cd1b3696f07afc.png" class="latex" alt="$0.\overline{23}_k$" style="vertical-align: -2px" width="39" height="17" /> as follows: <img src="//latex.artofproblemsolving.com/3/d/d/3ddd5f37e8ea1ea9bacba39c51936e99f98d54f6.png" class="latex" alt="$0.\overline{23}_k = 2\cdot k^{-1} + 3 \cdot k^{-2} + 2 \cdot k^{-3} + 3 \cdot k^{-4} + \cdots$" style="vertical-align: -4px" width="380" height="20" />
</p><p>Notice that this is equivalent to 
<img src="//latex.artofproblemsolving.com/4/d/e/4de3f2df4de994ca42b8e04b79c72d320b9c58a8.png" class="latexcenter" alt="\[2( k^{-1} + k^{-3} + k^{-5} + ... ) + 3 (k^{-2} + k^{-4} + k^{-6} + \cdots )\]" width="406" height="20" />
</p><p>By summing the geometric series and simplifying, we have <img src="//latex.artofproblemsolving.com/f/6/1/f61baab88f88e280c09e4e114c20cf0a10a03eec.png" class="latex" alt="$\frac{2k+3}{k^2-1} = \frac{7}{51}$" style="vertical-align: -13px" width="99" height="38" />. Solving this quadratic equation (or simply testing the answer choices) yields the answer <img src="//latex.artofproblemsolving.com/8/8/9/8890d1843f86f20a4711e45593b4dad9e0094020.png" class="latex" alt="$k = \boxed{\textbf{(D) }16}$" style="vertical-align: -10px" width="100" height="28" />.
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>Let <img src="//latex.artofproblemsolving.com/7/6/4/764b81b942230a20855e999bc3c892881fbbc93f.png" class="latex" alt="$a = 0.2323\dots_k$" style="vertical-align: -2px" width="115" height="15" />. Therefore, <img src="//latex.artofproblemsolving.com/6/d/9/6d951455682b9261a789520061bc6d7465f510fe.png" class="latex" alt="$k^2a=23.2323\dots_k$" style="vertical-align: -2px" width="142" height="17" />.
</p><p>From this, we see that <img src="//latex.artofproblemsolving.com/7/4/9/7493ce553072d7eb83ed70a710b3ac09fd96f18b.png" class="latex" alt="$k^2a-a=23_k$" style="vertical-align: -2px" width="108" height="17" />, so <img src="//latex.artofproblemsolving.com/d/3/2/d32314dcd06aa9ac02ae166ace9c498f0518d60f.png" class="latex" alt="$a = \frac{23_k}{k^2-1} = \frac{2k+3}{k^2-1} = \frac{7}{51}$" style="vertical-align: -13px" width="210" height="38" />.
</p><p>Now, similar to in Solution 1, we can either test if <img src="//latex.artofproblemsolving.com/5/9/9/599d2c8baf99c8703253d7ccd03e2842d87ec64c.png" class="latex" alt="$2k+3$" style="vertical-align: -1px" width="50" height="14" /> is a multiple of 7 with the answer choices, or actually solve the quadratic, so that the answer is <img src="//latex.artofproblemsolving.com/a/d/6/ad6f66809be1add52e351bd651aedb1ad340b95d.png" class="latex" alt="$\boxed{\textbf{(D) }16}$" style="vertical-align: -10px" width="65" height="28" />.
</p>
<h2><span class="mw-headline" id="Solution_3">Solution 3</span></h2>
<p>Just as in Solution 1, we arrive at the equation <img src="//latex.artofproblemsolving.com/b/a/7/ba71703f6aa9bbf745681822c9ca12b8f8e6f439.png" class="latex" alt="$\frac{2k+3}{k^2-1}=\frac{7}{51}$" style="vertical-align: -13px" width="99" height="38" />.
</p><p>We can now rewrite this as <img src="//latex.artofproblemsolving.com/9/0/0/9004a40be8a073402d587eab4cdbe85ba95b0e7e.png" class="latex" alt="$\frac{2k+3}{(k-1)(k+1)}=\frac{7}{51}=\frac{7}{3\cdot 17}$" style="vertical-align: -17px" width="228" height="42" />. Notice that <img src="//latex.artofproblemsolving.com/9/0/0/900aca5b9bb75ad101006c1ec6badbae16833509.png" class="latex" alt="$2k+3=2(k+1)+1=2(k-1)+5$" style="vertical-align: -4px" width="290" height="18" />. As <img src="//latex.artofproblemsolving.com/b/7/6/b7616a230c77f0d9a577e5bca01aa06f1c35c457.png" class="latex" alt="$17$" style="vertical-align: 0px" width="18" height="12" /> is a prime, we therefore must have that one of <img src="//latex.artofproblemsolving.com/7/d/c/7dc6fab4c2fdb4e9046030fbfdcf6ea721b05602.png" class="latex" alt="$k-1$" style="vertical-align: 0px" width="40" height="13" /> and <img src="//latex.artofproblemsolving.com/f/0/5/f05f085f774fc8ab50676b778b86a1f1d1114fc8.png" class="latex" alt="$k+1$" style="vertical-align: -1px" width="40" height="14" /> is divisible by <img src="//latex.artofproblemsolving.com/b/7/6/b7616a230c77f0d9a577e5bca01aa06f1c35c457.png" class="latex" alt="$17$" style="vertical-align: 0px" width="18" height="12" />. Now, checking each of the answer choices, this gives <img src="//latex.artofproblemsolving.com/a/d/6/ad6f66809be1add52e351bd651aedb1ad340b95d.png" class="latex" alt="$\boxed{\textbf{(D) }16}$" style="vertical-align: -10px" width="65" height="28" />.
</p>
<h2><span class="mw-headline" id="Solution_4">Solution 4</span></h2>
<p>Assuming you are familiar with the rules for basic repeating decimals, <img src="//latex.artofproblemsolving.com/0/a/7/0a705278f6a0af91183875a5ae4e7c3da0b4b04b.png" class="latex" alt="$0.232323... = \frac{23}{99}$" style="vertical-align: -12px" width="128" height="37" />. Now we want our base, <img src="//latex.artofproblemsolving.com/8/c/3/8c325612684d41304b9751c175df7bcc0f61f64f.png" class="latex" alt="$k$" width="9" height="12" />, to conform to <img src="//latex.artofproblemsolving.com/3/2/e/32e93d1016ec8b999553a60eab708341a56571bb.png" class="latex" alt="$23\equiv7\pmod k$" style="vertical-align: -4px" width="123" height="18" /> and <img src="//latex.artofproblemsolving.com/c/e/b/ceb841943c0d832f45b5330512619ed03b184db1.png" class="latex" alt="$99\equiv51\pmod k$" style="vertical-align: -4px" width="132" height="18" />, the reason being that we wish to convert the number from base <img src="//latex.artofproblemsolving.com/f/c/6/fc606f7f1e530731ab4f1cc364c01dc64a4455ee.png" class="latex" alt="$10$" style="vertical-align: 0px" width="17" height="13" /> to base <img src="//latex.artofproblemsolving.com/8/c/3/8c325612684d41304b9751c175df7bcc0f61f64f.png" class="latex" alt="$k$" width="9" height="12" />. Given the first equation, we know that <img src="//latex.artofproblemsolving.com/8/c/3/8c325612684d41304b9751c175df7bcc0f61f64f.png" class="latex" alt="$k$" width="9" height="12" /> must equal 9, 16, 23, or generally, <img src="//latex.artofproblemsolving.com/1/d/7/1d70be966baaf9a9271c77e8a53e447c6ed5ca1e.png" class="latex" alt="$7n+2$" style="vertical-align: -1px" width="50" height="13" />. The only number in this set that is one of the multiple choices is <img src="//latex.artofproblemsolving.com/9/a/5/9a5b4928c8fe50ce3c2428da3bee3505e891b788.png" class="latex" alt="$16$" style="vertical-align: 0px" width="17" height="13" />. When we test this on the second equation, <img src="//latex.artofproblemsolving.com/c/e/b/ceb841943c0d832f45b5330512619ed03b184db1.png" class="latex" alt="$99\equiv51\pmod k$" style="vertical-align: -4px" width="132" height="18" />, it comes to be true. Therefore, our answer is <img src="//latex.artofproblemsolving.com/a/d/6/ad6f66809be1add52e351bd651aedb1ad340b95d.png" class="latex" alt="$\boxed{\textbf{(D) }16}$" style="vertical-align: -10px" width="65" height="28" />.
</p>
<h2><span class="mw-headline" id="Solution_5">Solution 5</span></h2>
<p>Note that the LHS equals <img src="//latex.artofproblemsolving.com/e/6/3/e633145b26207b95a685ad6e97b32eedbc1e9198.png" class="latexcenter" alt="\[\bigg(\frac{2}{k} + \frac{2}{k^3} + \cdots \bigg) + \bigg(\frac{3}{k^2} + \frac{3}{k^4} + \cdots \bigg) = \frac{\frac{2}{k}}{1 - \frac{1}{k^2}} + \frac{\frac{3}{k^2}}{1 - \frac{1}{k^2}} = \frac{2k+3}{k^2},\]" width="475" height="47" /> from which we see our equation becomes <img src="//latex.artofproblemsolving.com/d/3/7/d37369a222166fc6f32db16652a51411f4eb7e72.png" class="latexcenter" alt="\[\frac{2k+3}{k^2-1} = \frac{7}{51}, \ \ \implies \ \ 51(2k+3) = 7(k^2-1).\]" width="356" height="38" /> 
</p><p>Note that <img src="//latex.artofproblemsolving.com/b/7/6/b7616a230c77f0d9a577e5bca01aa06f1c35c457.png" class="latex" alt="$17$" style="vertical-align: 0px" width="18" height="12" /> therefore divides <img src="//latex.artofproblemsolving.com/9/8/9/989181fa60643f4cdb41800fea8f6ebd8d883341.png" class="latex" alt="$k^2 - 1,$" style="vertical-align: -3px" width="52" height="18" /> but as <img src="//latex.artofproblemsolving.com/b/7/6/b7616a230c77f0d9a577e5bca01aa06f1c35c457.png" class="latex" alt="$17$" style="vertical-align: 0px" width="18" height="12" /> is prime this therefore implies <img src="//latex.artofproblemsolving.com/d/0/a/d0a3308b2b35ac19238cdacb19d452697e79d9b1.png" class="latexcenter" alt="\[k \equiv \pm 1 \pmod{17}.\]" width="151" height="18" /> (Warning: This would not be necessarily true if <img src="//latex.artofproblemsolving.com/b/7/6/b7616a230c77f0d9a577e5bca01aa06f1c35c457.png" class="latex" alt="$17$" style="vertical-align: 0px" width="18" height="12" /> were composite.) Note that <img src="//latex.artofproblemsolving.com/2/0/b/20bc985c2d1c4c1f4eb5a6f7eabfb653df63a348.png" class="latex" alt="$\boxed{\textbf{(D)} 16 }$" style="vertical-align: -10px" width="59" height="28" /> is the only answer choice congruent satisfying this modular congruence, thus completing the problem. <img src="//latex.artofproblemsolving.com/b/6/c/b6cf65b1f2fbdf388e3daeff9b96b34d3399d777.png" class="latex" alt="$\square$" width="13" height="12" />
</p><p>~ Professor-Mom
</p>
