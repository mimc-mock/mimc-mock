<p>Let <img src="//latex.artofproblemsolving.com/8/a/c/8acdb05cb79a9c8717989c53e35bca752636986a.png" class="latex" alt="$\{a_k\}$" style="vertical-align: -4px" width="35" height="18" /> be a sequence of integers such that <img src="//latex.artofproblemsolving.com/2/b/0/2b02a491e6fc8db3a577b344b1a53b7a8e77e0f9.png" class="latex" alt="$a_1=1$" style="vertical-align: -2px" width="49" height="14" /> and <img src="//latex.artofproblemsolving.com/1/5/7/15769baac51bc344bdaec7747e04fce9324dfa50.png" class="latex" alt="$a_{m+n}=a_m+a_n+mn,$" style="vertical-align: -4px" width="181" height="14" /> for all positive integers <img src="//latex.artofproblemsolving.com/f/5/0/f5047d1e0cbb50ec208923a22cd517c55100fa7b.png" class="latex" alt="$m$" width="15" height="8" /> and <img src="//latex.artofproblemsolving.com/b/d/9/bd923e41635af40a1abe58f230732f2280b0723a.png" class="latex" alt="$n.$" width="14" height="8" /> Then <img src="//latex.artofproblemsolving.com/3/f/6/3f6bfb1f0a208187bed5a389462b7e45bb2a04c9.png" class="latex" alt="$a_{12}$" style="vertical-align: -2px" width="22" height="10" /> is
</p><p><img src="//latex.artofproblemsolving.com/8/b/a/8baf91f5ebb7dde68943923e36e20d09eb10a4fb.png" class="latex" alt="$\mathrm{(A) \ } 45\qquad \mathrm{(B) \ } 56\qquad \mathrm{(C) \ } 67\qquad \mathrm{(D) \ } 78\qquad \mathrm{(E) \ } 89$" style="vertical-align: -4px" width="387" height="18" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>When <img src="//latex.artofproblemsolving.com/a/4/b/a4b5800f5ad040d308609190ef18a9e0ca08a9b0.png" class="latex" alt="$m=1$" style="vertical-align: 0px" width="48" height="12" />, <img src="//latex.artofproblemsolving.com/e/4/4/e449dd3606058a687318b3a27773a91ce86791b7.png" class="latex" alt="$a_{n+1}=1+a_n+n$" style="vertical-align: -4px" width="142" height="16" />. Hence,
<img src="//latex.artofproblemsolving.com/0/a/e/0aef06417bf8b8b89391fea61e1cb98eb7a215eb.png" class="latexcenter" alt="\[a_{2}=1+a_1+1\]" width="119" height="15" />
<img src="//latex.artofproblemsolving.com/d/7/2/d72d1eaefc94ae603d8e632b918684a866018c0d.png" class="latexcenter" alt="\[a_{3}=1+a_2+2\]" width="120" height="15" />
<img src="//latex.artofproblemsolving.com/8/c/9/8c9295690bdd597f32271e5fc8ad8a8c475d8d1d.png" class="latexcenter" alt="\[a_{4}=1+a_3+3\]" width="120" height="15" />
<img src="//latex.artofproblemsolving.com/1/6/7/16762e47f22f53e2431935f211fa82570b46b6cb.png" class="latexcenter" alt="\[\dots\]" width="18" height="2" />
<img src="//latex.artofproblemsolving.com/1/a/0/1a0b1067f595a8c0e8a661368a8ca69a9a568797.png" class="latexcenter" alt="\[a_{12}=1+a_{11}+11\]" width="141" height="15" />
Adding these equations up, we have that <img src="//latex.artofproblemsolving.com/d/d/3/dd306deb8cf8ddbe260af7cb5f81b2d06ecefb14.png" class="latex" alt="$a_{12}=12+(1+2+3+...+11)=\boxed{\mathrm{(D) \ } 78}$" style="vertical-align: -10px" width="339" height="29" />
</p><p>~AopsUser101
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>Substituting <img src="//latex.artofproblemsolving.com/1/d/9/1d96ecc15129c5631ba4a2c6578e91e3d13e3baf.png" class="latex" alt="$n=1$" style="vertical-align: 0px" width="43" height="12" /> into <img src="//latex.artofproblemsolving.com/d/b/c/dbc309386e742de35e26e2f0b9fc4173a29818ff.png" class="latex" alt="$a_{m+n}=a_m+a_n+mn$" style="vertical-align: -4px" width="176" height="14" />: <img src="//latex.artofproblemsolving.com/f/0/b/f0b2ed054f50dbe1d17fe8ab7a08ee7bb7ddcd14.png" class="latex" alt="$a_{m+1}=a_m+a_{1}+m$" style="vertical-align: -4px" width="162" height="14" />.
</p><p>Since <img src="//latex.artofproblemsolving.com/b/1/1/b1171ce641826e9aca894162f9c15a57f6b46e09.png" class="latex" alt="$a_1 = 1$" style="vertical-align: -2px" width="49" height="14" />, <img src="//latex.artofproblemsolving.com/7/2/5/7253ecfe2492dbc6e97e6193ef3e63846a8f7a3b.png" class="latex" alt="$a_{m+1}=a_m+m+1$" style="vertical-align: -4px" width="154" height="16" />.
</p><p>Therefore, <img src="//latex.artofproblemsolving.com/d/7/6/d766ad9195de7b651f011dc4f4256b98ff40b05c.png" class="latex" alt="$a_m = a_{m-1} + m, a_{m-1}=a_{m-2}+(m-1), a_{m-2} = a_{m-3} + (m-2)$" style="vertical-align: -4px" width="511" height="18" />, and so on until <img src="//latex.artofproblemsolving.com/c/d/5/cd59b3f412ed32a236ae6309debc627eb12a4b24.png" class="latex" alt="$a_2 = a_1 + 2$" style="vertical-align: -2px" width="89" height="14" />.
</p><p>Adding the Left Hand Sides of all of these equations gives <img src="//latex.artofproblemsolving.com/6/4/a/64a4a425e895d1635544785759a5bf25d820d422.png" class="latex" alt="$a_m + a_{m-1} + a_{m-2} + a_{m-3} + \cdots + a_2$" style="vertical-align: -2px" width="287" height="13" />.
</p><p>Adding the Right Hand Sides of these equations gives 
</p><p><img src="//latex.artofproblemsolving.com/c/7/4/c744367d85b994b05a809b1ec72105d841ebf7f7.png" class="latex" alt="$(a_{m-1} + a_{m-2} + a_{m-3} + \cdots + a_1) + (m + (m-1) + (m-2) + \cdots + 2)$" style="vertical-align: -4px" width="551" height="18" />.
</p><p>These two expressions must be equal; hence <img src="//latex.artofproblemsolving.com/5/c/f/5cfca5939dad94e737b3ac4e3c2af81eab8daf3b.png" class="latex" alt="$a_m + a_{m-1} + a_{m-2} + a_{m-3} + \cdots + a_2 = (a_{m-1} + a_{m-2} + a_{m-3} + \cdots + a_1) + (m + (m-1) + (m-2) + \cdots + 2)$" style="vertical-align: -4px" width="865" height="18" /> and <img src="//latex.artofproblemsolving.com/7/d/b/7dbfbbbd0376cd1b7a3489c2f351b6f6d33ba3b8.png" class="latex" alt="$a_m = a_1 + (m + (m-1) + (m-2) + \cdots + 2)$" style="vertical-align: -4px" width="356" height="18" />.
</p><p>Substituting <img src="//latex.artofproblemsolving.com/b/1/1/b1171ce641826e9aca894162f9c15a57f6b46e09.png" class="latex" alt="$a_1 = 1$" style="vertical-align: -2px" width="49" height="14" />: <img src="//latex.artofproblemsolving.com/5/2/3/5230553cd9f1c72e9e6f3b38df9d9d392d7b3abd.png" class="latex" alt="$a_m = 1 + (m + (m-1) + (m-2) + \cdots + 2) = 1+2+3+4+ \cdots +m = \frac{(m+1)(m)}{2}$" style="vertical-align: -12px" width="677" height="38" />.
</p><p>Thus we have a general formula for <img src="//latex.artofproblemsolving.com/1/d/d/1ddf57a1507686a49e3d290b3542ccac565cd35a.png" class="latex" alt="$a_m$" style="vertical-align: -2px" width="21" height="10" /> and substituting <img src="//latex.artofproblemsolving.com/2/d/d/2dda80bd4f83db6b8ae1515c299c98ac3083af6b.png" class="latex" alt="$m=12$" style="vertical-align: 0px" width="57" height="12" />: <img src="//latex.artofproblemsolving.com/6/0/b/60b678c17218eb0bd5f3898cf6b8fa59b1ce2a47.png" class="latex" alt="$a_{12} = \frac{(13)(12)}{2} = (13)(6) = \boxed{\mathrm{(D)} 78}$" style="vertical-align: -12px" width="278" height="38" />.
</p>
<h2><span class="mw-headline" id="Solution_3">Solution 3</span></h2>
<p>We can literally just plug stuff in. No prerequisite is actually said in the sequence. Since <img src="//latex.artofproblemsolving.com/c/d/e/cde770f7c812419bc0a32b4572634e29865ec566.png" class="latex" alt="$a_{m+n} = a_m+a_n +mn$" style="vertical-align: -4px" width="176" height="14" />, we know <img src="//latex.artofproblemsolving.com/3/1/9/319b4414063907afa05ae8f662b945bb7e2ef505.png" class="latex" alt="$a_2=a_1+a_1+1\cdot1=1+1+1=3$" style="vertical-align: -2px" width="280" height="15" />. After this, we can use <img src="//latex.artofproblemsolving.com/5/e/9/5e97a8af68fbc8e357d3ee0eba452022b06c1875.png" class="latex" alt="$a_2$" style="vertical-align: -2px" width="15" height="10" /> to find <img src="//latex.artofproblemsolving.com/5/e/2/5e28850813a5321d2a8ae3c0510457a7337975ed.png" class="latex" alt="$a_4$" style="vertical-align: -2px" width="16" height="10" />. <img src="//latex.artofproblemsolving.com/e/b/8/eb8035207607946e30a4d7cc3734f8e4ce22cc70.png" class="latex" alt="$a_4=a_2+a_2+2\cdot 2 = 3+3+4 = 10$" style="vertical-align: -2px" width="289" height="15" />. Now, we can use <img src="//latex.artofproblemsolving.com/5/e/9/5e97a8af68fbc8e357d3ee0eba452022b06c1875.png" class="latex" alt="$a_2$" style="vertical-align: -2px" width="15" height="10" /> and <img src="//latex.artofproblemsolving.com/5/e/2/5e28850813a5321d2a8ae3c0510457a7337975ed.png" class="latex" alt="$a_4$" style="vertical-align: -2px" width="16" height="10" /> to find <img src="//latex.artofproblemsolving.com/3/4/a/34ace040475607683853fcfd4960a8f7fa6678c7.png" class="latex" alt="$a_6$" style="vertical-align: -2px" width="15" height="10" />, or <img src="//latex.artofproblemsolving.com/d/6/d/d6d62456e99eced58dd57ac88429cfd5eea50392.png" class="latex" alt="$a_6=a_4+a_2+4\cdot 2 = 10+3+8=21$" style="vertical-align: -2px" width="297" height="15" />. Lastly, we can use <img src="//latex.artofproblemsolving.com/3/4/a/34ace040475607683853fcfd4960a8f7fa6678c7.png" class="latex" alt="$a_6$" style="vertical-align: -2px" width="15" height="10" /> to find <img src="//latex.artofproblemsolving.com/3/f/6/3f6bfb1f0a208187bed5a389462b7e45bb2a04c9.png" class="latex" alt="$a_{12}$" style="vertical-align: -2px" width="22" height="10" />. <img src="//latex.artofproblemsolving.com/a/4/a/a4af4158c138378329bbd52201d0c3d012fe5981.png" class="latex" alt="$a_{12} = a_6+a_6+6\cdot 6 = 21+21+36= \boxed{\mathrm{(D) \ } 78}$" style="vertical-align: -10px" width="369" height="29" />
</p>
<h2><span class="mw-headline" id="Solution_4">Solution 4</span></h2>
<p>We can set <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> equal to <img src="//latex.artofproblemsolving.com/f/5/0/f5047d1e0cbb50ec208923a22cd517c55100fa7b.png" class="latex" alt="$m$" width="15" height="8" />, so we can say that 
<img src="//latex.artofproblemsolving.com/d/0/e/d0e11a0c3d648fe89dc2008c29e4b9dba61e4986.png" class="latexcenter" alt="\[a_{m + m} = a_m + a_m + m*m\]" width="205" height="15" />
<img src="//latex.artofproblemsolving.com/3/f/0/3f04f705a62096b5dbefa309be248f9cbbaa8df5.png" class="latexcenter" alt="\[a_{2m} = 2a_m + m^2\]" width="128" height="18" />
</p><p>We set <img src="//latex.artofproblemsolving.com/a/e/6/ae63293e58ad0185f715021c4ff800a48a50e2bc.png" class="latex" alt="$2m = 12$" style="vertical-align: 0px" width="67" height="12" />, we get <img src="//latex.artofproblemsolving.com/9/7/0/970c67d43e2cd7f3e62f1e9afeb5d18508c8dd34.png" class="latex" alt="$m = 6$" width="48" height="12" />.
<img src="//latex.artofproblemsolving.com/9/4/9/9491d260f1aeaafa41b3d0918bfb8c46a2a93e5e.png" class="latexcenter" alt="\[a_{12} = 2a_6 + 36\]" width="113" height="15" />
</p><p>We set <img src="//latex.artofproblemsolving.com/6/4/c/64cac487d22555cff8b098157426db871f16a571.png" class="latex" alt="$2m = 6$" width="58" height="12" />m, we get <img src="//latex.artofproblemsolving.com/2/7/b/27b00129906b721ebe808a705e83d5df749a4ba3.png" class="latex" alt="$m = 3$" width="48" height="12" />.
<img src="//latex.artofproblemsolving.com/b/2/7/b2744b22d14521cdb0e49a869119d46e7ad980ee.png" class="latexcenter" alt="\[a_6 = 2a_3 + 9\]" width="97" height="15" />
</p><p>Solving for <img src="//latex.artofproblemsolving.com/5/7/2/572b65ebc3f438b176b4ca4a890799f2f72564af.png" class="latex" alt="$a_3$" style="vertical-align: -2px" width="15" height="10" /> is easy, just direct substitution.
<img src="//latex.artofproblemsolving.com/3/5/5/3555dd02a3ab15fa5b5df91c8bd1e51015353b40.png" class="latexcenter" alt="\[a_2 = 1 + 1 + 1 = 3\]" width="145" height="15" />
<img src="//latex.artofproblemsolving.com/5/7/f/57faffab5e14d760fd35858286d6167e4b5866d8.png" class="latexcenter" alt="\[a_3 = a_{2 + 1} = 3 + 1 + 2 = 6\]" width="203" height="16" />
</p><p>Substituting, we get
<img src="//latex.artofproblemsolving.com/0/d/4/0d4bfa17bdbbea6f8a5f536ea0448b78fe1bbac8.png" class="latexcenter" alt="\[a_6 = 2(6) + 9 = 21\]" width="145" height="18" />
<img src="//latex.artofproblemsolving.com/1/f/8/1f8fdfa7efb5a5f72709a05d3755226952b7f8ec.png" class="latexcenter" alt="\[a_{12} = 2(21) + 36 = 78\]" width="171" height="18" />
</p><p>Thus, the answer is <img src="//latex.artofproblemsolving.com/5/5/b/55bdade53f471aab49f63d1a734979cc0c636640.png" class="latex" alt="$\boxed{D}$" style="vertical-align: -5px" width="27" height="24" />.
</p><p>~ euler123
</p>
<h2><span class="mw-headline" id="Solution_5">Solution 5</span></h2>
<p>Note that the sequence of triangular numbers <img src="//latex.artofproblemsolving.com/d/0/1/d0135d49ccd2c383d1ac1b4e5e19cf6af3cab32c.png" class="latex" alt="$T_n=1+2+3+...+n$" style="vertical-align: -2px" width="185" height="15" /> satisfies these conditions. It is immediately obvious that it satisfies <img src="//latex.artofproblemsolving.com/2/b/0/2b02a491e6fc8db3a577b344b1a53b7a8e77e0f9.png" class="latex" alt="$a_1=1$" style="vertical-align: -2px" width="49" height="14" />, and <img src="//latex.artofproblemsolving.com/d/b/c/dbc309386e742de35e26e2f0b9fc4173a29818ff.png" class="latex" alt="$a_{m+n}=a_m+a_n+mn$" style="vertical-align: -4px" width="176" height="14" /> can be visually proven with the diagram below.
</p><p><img src="//latex.artofproblemsolving.com/3/5/f/35fd346c6b8de4b5d49b16a163d49e70861d5e2f.png" class="latexcenter" alt="[asy] for(int i=5; i &gt; 0; --i) { for(int j=0; j &lt; i; ++j) { draw(circle((j+(5-i)/2,(5-i)*sqrt(3)/2),.2)); }; }; path m1 = brace((2,-.3),(0,-.3),.2); draw(m1); label(&quot;$m$&quot;,m1,S);  path n1 = brace((4,-.3),(3,-.3),.2); draw(n1); label(&quot;$n$&quot;,n1,S);   draw((-.2*sqrt(3),-.2)--(2+.2*sqrt(3),-.2)--(1,.4+sqrt(3))--cycle); label(&quot;$T_m$&quot;,(1,1/3*sqrt(3)));  draw((3-.2*sqrt(3),-.2)--(4+.2*sqrt(3),-.2)--(3.5,.4+.5*sqrt(3))--cycle); label(&quot;$T_n$&quot;,(3.5,.5/3*sqrt(3)));   path m2 = brace((2+.15*sqrt(3),.15+2*sqrt(3)),(3+.15*sqrt(3),.15+sqrt(3)),.2); draw(m2); label(&quot;$m$&quot;,m2,(.5*sqrt(3),.5));  path n2 = brace((1.5-.15*sqrt(3),.15+1.5*sqrt(3)),(2-.15*sqrt(3),.15+2*sqrt(3)),.2); draw(n2); label(&quot;$n$&quot;,n2,(-.5*sqrt(3),.5));   draw((2.5,-.4+.5*sqrt(3))--(3+.4/3*sqrt(3),sqrt(3))--(2,.4+2*sqrt(3))--(1.5-.4/3*sqrt(3),1.5*sqrt(3))--cycle); label(&quot;$mn$&quot;,(2.25,1.25*sqrt(3))); [/asy]" width="252" height="252" />
</p><p>This means that we can use the triangular number formula <img src="//latex.artofproblemsolving.com/c/3/c/c3c2697932cf8fa792ed50ee3b430726a8d8097a.png" class="latex" alt="$T_n = \frac{n(n+1)}{2}$" style="vertical-align: -12px" width="113" height="38" />, so the answer is <img src="//latex.artofproblemsolving.com/0/9/c/09c8a03b45eef10e256fd6a3b094cdc441c493e0.png" class="latex" alt="$T_{12} = \frac{12(12+1)}{2} = \boxed{\mathrm{(D) \ } 78}$" style="vertical-align: -12px" width="222" height="38" />.
</p><p><br />
