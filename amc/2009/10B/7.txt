<p>By inserting parentheses, it is possible to give the expression
<img src="//latex.artofproblemsolving.com/5/7/4/5749b941fd5ab876b3610ecff97cfbbf6ffd4981.png" class="latexcenter" alt="\[2\times3 + 4\times5\]" width="102" height="14" />
several values.  How many different values can be obtained?
</p><p><img src="//latex.artofproblemsolving.com/c/e/4/ce4e305858d2968a7a35e72d8e8d6f848ef4e8a5.png" class="latex" alt="$\text{(A) } 2 \qquad \text{(B) } 3 \qquad \text{(C) } 4 \qquad \text{(D) } 5 \qquad \text{(E) } 6$" style="vertical-align: -4px" width="330" height="17" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>The three operations can be performed on any of <img src="//latex.artofproblemsolving.com/4/d/b/4db3b59940e4f5cd42bbab2b7815c452a0197507.png" class="latex" alt="$3! = 6$" width="47" height="13" /> orders.  However, if the addition is performed either first or last, then multiplying in either order produces the same result. So at most four distinct values can be obtained.  It is easy to check that the values of the four expressions
<img src="//latex.artofproblemsolving.com/a/e/4/ae4e15557bcda228a1dbe886c24d7814e740971d.png" class="latexcenter" alt="\begin{align*} (2\times3) + (4\times5) &amp;= 26,\\ (2\times3 + 4)\times5 &amp;= 50,\\ 2\times(3 + 4\times5) &amp;= 46,\\ 2\times(3 + 4)\times5 &amp;= 70 \end{align*}" width="176" height="93" />
are in fact all distinct.  So the answer is <img src="//latex.artofproblemsolving.com/6/c/4/6c4c6c07150a0f6e56b0a6e198d1d19155dfe96c.png" class="latex" alt="$\boxed{4}$" style="vertical-align: -5px" width="20" height="23" />, which is choice <img src="//latex.artofproblemsolving.com/2/4/8/2484f88154fadcf9c781bba19becdae88696b77f.png" class="latex" alt="$\mathrm{(C)}$" style="vertical-align: -4px" width="25" height="18" />.
</p>
