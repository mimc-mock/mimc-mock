<p>What is the value of <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> if <img src="//latex.artofproblemsolving.com/d/f/7/df7dc189ae07b874946a12062a833c20d1d2c406.png" class="latex" alt="$|x-1|=|x-2|$" style="vertical-align: -4px" width="125" height="18" />?
</p><p><img src="//latex.artofproblemsolving.com/2/5/2/2528149de592f3ca8564c78b16c9d3c54150a6c5.png" class="latex" alt="$\mathrm{(A) \ } -\frac12 \qquad \mathrm{(B) \ } \frac12 \qquad \mathrm{(C) \ } 1 \qquad \mathrm{(D) \ } \frac32 \qquad \mathrm{(E) \ } 2$" style="vertical-align: -12px" width="376" height="37" />
</p>
<h2><span class="mw-headline" id="Solutions">Solutions</span></h2>
<h3><span class="mw-headline" id="Solution">Solution</span></h3>
<p><img src="//latex.artofproblemsolving.com/f/b/5/fb552300afddad49a8b28ca5d50402393a47616d.png" class="latex" alt="$|x-1|$" style="vertical-align: -4px" width="49" height="18" /> is the distance between <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> and <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" />; <img src="//latex.artofproblemsolving.com/1/f/b/1fb66ec93b6ea3313b9d1f313e7f49488c64a61b.png" class="latex" alt="$|x-2|$" style="vertical-align: -4px" width="49" height="18" /> is the distance between <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> and <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" />.
</p><p>Therefore, the given equation says <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> is equidistant from <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" /> and <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" />, so <img src="//latex.artofproblemsolving.com/a/8/c/a8c31ee2dfb8ea278390f17aff8ac3f279220440.png" class="latex" alt="$x=\frac{1+2}2=\frac{3}{2}\Rightarrow\boxed{\mathrm{(D)}\ \frac{3}{2}}$" style="vertical-align: -18px" width="203" height="48" />.
</p><p>Alternatively, we can solve by casework (a method which should work for any similar problem involving <a href="/wiki/index.php/Absolute_value" title="Absolute value">absolute values</a> of <a href="/wiki/index.php/Real_number" title="Real number">real numbers</a>).  If <img src="//latex.artofproblemsolving.com/b/9/a/b9afbaa16cfe6f80af3e0aad0641fcbe60dcae2a.png" class="latex" alt="$x \leq 1$" style="vertical-align: -2px" width="42" height="14" />, then <img src="//latex.artofproblemsolving.com/9/3/2/9320a881b9ee0bc9b1b2fa2e0d722017add88118.png" class="latex" alt="$|x - 1| = 1-x$" style="vertical-align: -4px" width="117" height="18" /> and <img src="//latex.artofproblemsolving.com/f/6/0/f609876686b271d76e1a77894df9942be7f9add2.png" class="latex" alt="$|x - 2| = 2 - x$" style="vertical-align: -4px" width="117" height="18" />, so we must solve <img src="//latex.artofproblemsolving.com/e/1/c/e1c54a1195777a6a1b1e0b47747f9d699528eaab.png" class="latex" alt="$1 - x = 2 - x$" style="vertical-align: 0px" width="107" height="12" />, which has no solutions.  Similarly, if <img src="//latex.artofproblemsolving.com/2/1/c/21cd6b8937bb07478e903e0186b41beefded34b8.png" class="latex" alt="$x \geq 2$" style="vertical-align: -2px" width="43" height="14" />, then <img src="//latex.artofproblemsolving.com/b/5/c/b5c53ff7435934dfb0512a85fe6ecc6bc67a67ee.png" class="latex" alt="$|x - 1| = x - 1$" style="vertical-align: -4px" width="116" height="18" /> and <img src="//latex.artofproblemsolving.com/4/3/4/43437de6ef081a3b6951dd156733cc58cda042f7.png" class="latex" alt="$|x - 2| = x - 2$" style="vertical-align: -4px" width="117" height="18" />, so we must solve <img src="//latex.artofproblemsolving.com/a/6/6/a66aefae4219c316baa963b88953a86a9ecc36f1.png" class="latex" alt="$x - 1 = x- 2$" style="vertical-align: 0px" width="107" height="12" />, which also has no solutions.  Finally, if <img src="//latex.artofproblemsolving.com/d/3/7/d3758d45bfc35e2180a37a86e6b112bd2a72845b.png" class="latex" alt="$1 \leq x \leq 2$" style="vertical-align: -2px" width="76" height="14" />, then <img src="//latex.artofproblemsolving.com/b/5/c/b5c53ff7435934dfb0512a85fe6ecc6bc67a67ee.png" class="latex" alt="$|x - 1| = x - 1$" style="vertical-align: -4px" width="116" height="18" /> and <img src="//latex.artofproblemsolving.com/1/3/2/132d6ae102412103a9c3b66629c7109d8f56c2cf.png" class="latex" alt="$|x - 2| = 2-x$" style="vertical-align: -4px" width="117" height="18" />, so we must solve <img src="//latex.artofproblemsolving.com/3/0/8/308a7602408fbcef7d924511d4c3f8a8593ad43c.png" class="latex" alt="$x - 1 = 2 - x$" style="vertical-align: 0px" width="107" height="12" />, which has the unique solution <img src="//latex.artofproblemsolving.com/6/8/f/68f5aa884318503b832c11d16203b4bb7042e7b4.png" class="latex" alt="$x = \frac32$" style="vertical-align: -12px" width="46" height="37" />.
</p>
<h3><span class="mw-headline" id="Solution_2">Solution 2</span></h3>
<p>We know that either <img src="//latex.artofproblemsolving.com/0/d/4/0d4cf8b3fbf613842fe93ff5ed8df37cf6610033.png" class="latex" alt="$x-1=x-2$" style="vertical-align: 0px" width="107" height="12" /> or <img src="//latex.artofproblemsolving.com/d/f/8/df8ef9811cfa09d907b96dea4e2321a4880865fe.png" class="latex" alt="$x-1=-(x-2)$" style="vertical-align: -4px" width="134" height="18" />. The first equation simplifies to <img src="//latex.artofproblemsolving.com/7/d/6/7d6147c7868f7a71e59918192adeff901fe2296f.png" class="latex" alt="$-1=-2$" style="vertical-align: 0px" width="70" height="12" />, which is false, so <img src="//latex.artofproblemsolving.com/d/f/8/df8ef9811cfa09d907b96dea4e2321a4880865fe.png" class="latex" alt="$x-1=-(x-2)$" style="vertical-align: -4px" width="134" height="18" />. Solving, we get <img src="//latex.artofproblemsolving.com/3/2/6/326a639f8b4eb069c92145427e7ea690890cebf6.png" class="latex" alt="$x=\frac{3}{2}\Rightarrow\boxed{\mathrm{(D)}\ \frac{3}{2}}$" style="vertical-align: -18px" width="134" height="48" />.
</p>
