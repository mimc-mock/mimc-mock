<p>Brenda and Sally run in opposite directions on a circular track, starting at <a href="/wiki/index.php/Diameter" title="Diameter">diametrically</a> opposite points.  They first meet after Brenda has run 100 meters.  They next meet after Sally has run 150 meters past their first meeting point.  Each girl runs at a constant speed.  What is the length of the track in meters?
</p><p><img src="//latex.artofproblemsolving.com/1/0/e/10e6378a4bebb98ac710142a128746b8247862a9.png" class="latex" alt="$\mathrm{(A) \ } 250 \qquad \mathrm{(B) \ } 300 \qquad \mathrm{(C) \ } 350 \qquad \mathrm{(D) \ }  400\qquad \mathrm{(E) \ } 500$" style="vertical-align: -4px" width="432" height="18" />
</p>
<h2><span class="mw-headline" id="Solutions">Solutions</span></h2>
<h3><span class="mw-headline" id="Solution_1">Solution 1</span></h3>
<p>Call the length of the race track <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" />. When they meet at the first meeting point, Brenda has run <img src="//latex.artofproblemsolving.com/e/5/9/e59e2c6e83eb78cca610a5fd4070ae01c8d4ae60.png" class="latex" alt="$100$" style="vertical-align: 0px" width="26" height="13" /> meters, while Sally has run <img src="//latex.artofproblemsolving.com/7/c/f/7cf8e40e3de5136d3608b4a595c42a68a3383195.png" class="latex" alt="$\frac{x}{2} - 100$" style="vertical-align: -12px" width="63" height="33" /> meters. By the second meeting point, Sally has run <img src="//latex.artofproblemsolving.com/f/5/e/f5e10e5f3aa7c766cb27c3ad319fa526f71588cd.png" class="latex" alt="$150$" style="vertical-align: 0px" width="26" height="13" /> meters, while Brenda has run <img src="//latex.artofproblemsolving.com/1/3/0/13080cccfe104afdb653a46b433ac5b264b59684.png" class="latex" alt="$x - 150$" style="vertical-align: 0px" width="59" height="13" /> meters. Since they run at a constant speed, we can set up a <a href="/wiki/index.php/Proportion" title="Proportion">proportion</a>: <img src="//latex.artofproblemsolving.com/7/c/9/7c953361a1967fdfaf6fb42f509be4123f9d0fa8.png" class="latex" alt="$\frac{100}{x- 150} = \frac{\frac{x}{2} - 100}{150}$" style="vertical-align: -13px" width="151" height="39" />. Cross-multiplying, we get that <img src="//latex.artofproblemsolving.com/3/4/9/34912a89f66017997181ca655013373341833b8c.png" class="latex" alt="$x = 350\Longrightarrow\boxed{\mathrm{(C)}\ 350}$" style="vertical-align: -10px" width="173" height="29" />.
</p>
<h3><span class="mw-headline" id="Solution_2">Solution 2</span></h3>
<p>The total distance the girls run between the start and the first meeting is one half of the track length. The total distance they run between the two meetings is the track length. As the girls run at constant speeds, the interval between the meetings is twice as long as the interval between the start and the first meeting. Thus between the meetings Brenda will run <img src="//latex.artofproblemsolving.com/8/e/2/8e2bb7680d28ee66dea4ae996852963d81596d65.png" class="latex" alt="$2\times100=200$" style="vertical-align: 0px" width="109" height="13" /> meters. Therefore the length of the track is <img src="//latex.artofproblemsolving.com/6/5/1/651c35215cdd7391e4dce51df444a9b7f398b2a3.png" class="latex" alt="$150 + 200 = 350$" style="vertical-align: -1px" width="127" height="14" /> meters <img src="//latex.artofproblemsolving.com/6/0/b/60b1fffe8d76fda076f145ede1abe8926012b087.png" class="latex" alt="$\Rightarrow\boxed{\mathrm{(C)}\ 350}$" style="vertical-align: -10px" width="95" height="29" />
</p><p><br />
</p>
