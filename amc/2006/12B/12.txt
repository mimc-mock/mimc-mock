<p>The <a href="/wiki/index.php/Parabola" title="Parabola">parabola</a> <img src="//latex.artofproblemsolving.com/8/4/c/84cea4f7bfc5284747e737af0d1a25e64afdd47e.png" class="latex" alt="$y=ax^2+bx+c$" style="vertical-align: -3px" width="132" height="18" /> has <a href="/wiki/index.php/Vertex" title="Vertex">vertex</a> <img src="//latex.artofproblemsolving.com/f/f/9/ff947434e8d911cc2c48214043c07d950858857d.png" class="latex" alt="$(p,p)$" style="vertical-align: -4px" width="39" height="18" /> and <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" />-intercept <img src="//latex.artofproblemsolving.com/4/7/8/4784770a0fbab87568fd865d911b4f472ddb1893.png" class="latex" alt="$(0,-p)$" style="vertical-align: -4px" width="52" height="18" />, where <img src="//latex.artofproblemsolving.com/c/8/9/c8912c8c77da38f911af5dad85ebdcec8dd731dc.png" class="latex" alt="$p\ne 0$" style="vertical-align: -4px" width="43" height="17" />. What is <img src="//latex.artofproblemsolving.com/8/1/3/8136a7ef6a03334a7246df9097e5bcc31ba33fd2.png" class="latex" alt="$b$" width="8" height="12" />?
</p><p><img src="//latex.artofproblemsolving.com/8/7/8/8789a47cdd80a8b6cf8cbd9b7ce6e9c9daca04ba.png" class="latex" alt="$\text {(A) } -p \qquad \text {(B) } 0 \qquad \text {(C) } 2 \qquad \text {(D) } 4 \qquad \text {(E) } p$" style="vertical-align: -4px" width="353" height="17" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>Substituting <img src="//latex.artofproblemsolving.com/4/7/8/4784770a0fbab87568fd865d911b4f472ddb1893.png" class="latex" alt="$(0,-p)$" style="vertical-align: -4px" width="52" height="18" />, we find that <img src="//latex.artofproblemsolving.com/7/6/6/7660845079668ec4eae411dce3f39a2a582353ce.png" class="latex" alt="$y = -p = a(0)^2 + b(0) + c = c$" style="vertical-align: -4px" width="237" height="19" />, so our parabola is <img src="//latex.artofproblemsolving.com/f/c/c/fcc9629cdc5b0fc8c93b281d8865e6d35682b562.png" class="latex" alt="$y = ax^2 + bx - p$" style="vertical-align: -3px" width="133" height="18" />. 
</p><p>The x-coordinate of the vertex of a parabola is given by <img src="//latex.artofproblemsolving.com/4/2/4/424644d3173e6f8d4cfb978c3d8e8fc1bbb3ce39.png" class="latex" alt="$x = p = \frac{-b}{2a} \Longleftrightarrow a = \frac{-b}{2p}$" style="vertical-align: -16px" width="195" height="41" />. Additionally, substituting <img src="//latex.artofproblemsolving.com/f/f/9/ff947434e8d911cc2c48214043c07d950858857d.png" class="latex" alt="$(p,p)$" style="vertical-align: -4px" width="39" height="18" />, we find that <img src="//latex.artofproblemsolving.com/7/6/7/767362eb6472cc8d00c0cec1e27a409109c6b394.png" class="latex" alt="$y = p = a(p)^2 + b(p) - p \Longleftrightarrow ap^2 + (b-2)p = \left(\frac{-b}{2p}\right)p^2 + (b-2)p = p\left(\frac b2-2\right) = 0$" style="vertical-align: -17px" width="667" height="43" />. Since it is given that <img src="//latex.artofproblemsolving.com/1/1/e/11e370369929d0c50bb247338e91eb7534b305ef.png" class="latex" alt="$p \neq 0$" style="vertical-align: -4px" width="43" height="17" />, then <img src="//latex.artofproblemsolving.com/e/7/9/e795cdf07a66d22aa053d74c19b6ea71a9317877.png" class="latex" alt="$\frac{b}{2} = 2 \Longrightarrow b = 4\ \mathrm{(D)}$" style="vertical-align: -12px" width="159" height="37" />.
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>A parabola with the given equation and with vertex <img src="//latex.artofproblemsolving.com/f/f/9/ff947434e8d911cc2c48214043c07d950858857d.png" class="latex" alt="$(p,p)$" style="vertical-align: -4px" width="39" height="18" /> must have equation <img src="//latex.artofproblemsolving.com/f/3/c/f3c90bcc212c044e77ad8563e4a78c609b3f4599.png" class="latex" alt="$y=a(x-p)^2+p$" style="vertical-align: -4px" width="138" height="19" />. Because the <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" />-intercept is <img src="//latex.artofproblemsolving.com/4/7/8/4784770a0fbab87568fd865d911b4f472ddb1893.png" class="latex" alt="$(0,-p)$" style="vertical-align: -4px" width="52" height="18" /> and <img src="//latex.artofproblemsolving.com/c/8/9/c8912c8c77da38f911af5dad85ebdcec8dd731dc.png" class="latex" alt="$p\ne 0$" style="vertical-align: -4px" width="43" height="17" />, it follows that <img src="//latex.artofproblemsolving.com/8/e/3/8e37cf2ee84a1937b8aa43111beb093a0897ca84.png" class="latex" alt="$a=-2/p$" style="vertical-align: -4px" width="75" height="18" />. Thus<img src="//latex.artofproblemsolving.com/6/c/3/6c348426a836de2ab7299ad1596ec81a24afbf84.png" class="latexcenter" alt="\[y=-\frac{2}{p}(x^2-2px+p^2)+p=-\frac{2}{p}x^2+4x-p,\]" width="360" height="40" /> so <img src="//latex.artofproblemsolving.com/f/e/0/fe0b8c03563be1aa2db6f0c86a7ff4216a2f000a.png" class="latex" alt="$\boxed{b=4}$" style="vertical-align: -5px" width="53" height="24" />.
</p>
