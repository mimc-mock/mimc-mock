<p>The function <img src="//latex.artofproblemsolving.com/b/b/2/bb2c93730dbb48558bb3c4738c956c4e8f816437.png" class="latex" alt="$f$" style="vertical-align: -3px" width="10" height="16" /> has the property that for each real number <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> in its domain, <img src="//latex.artofproblemsolving.com/d/1/7/d172d3d8333a9c1740b123ea13d5b44f22446869.png" class="latex" alt="$1/x$" style="vertical-align: -4px" width="28" height="18" /> is also in its domain and 
</p><p><img src="//latex.artofproblemsolving.com/e/d/c/edc79d79703691ce01f38e47e5cccfe30d173a54.png" class="latex" alt="$f(x)+f\left(\frac{1}{x}\right)=x$" style="vertical-align: -17px" width="146" height="43" />
</p><p>What is the largest set of real numbers that can be in the domain of <img src="//latex.artofproblemsolving.com/b/b/2/bb2c93730dbb48558bb3c4738c956c4e8f816437.png" class="latex" alt="$f$" style="vertical-align: -3px" width="10" height="16" />?
</p><p><img src="//latex.artofproblemsolving.com/c/7/b/c7beb8d12fa38c46ce55bec9431d4f16049b9356.png" class="latex" alt="$\mathrm{(A) \ } \{x|x\ne 0\}\qquad \mathrm{(B) \ } \{x|x&lt;0\}$" style="vertical-align: -4px" width="252" height="18" />
</p><p><img src="//latex.artofproblemsolving.com/8/e/9/8e944c7ed51c8eda299eb8d6dc48966c2c94ea20.png" class="latex" alt="$\mathrm{(C) \ } \{x|x&gt;0\}$" style="vertical-align: -4px" width="109" height="18" /><img src="//latex.artofproblemsolving.com/1/0/0/1000edba6af8b1601dc6ad929ce424b14c4063db.png" class="latex" alt="$\mathrm{(D) \ } \{x|x\ne -1\;\rm{and}\; x\ne 0\;\rm{and}\; x\ne 1\}$" style="vertical-align: -4px" width="288" height="18" />
</p><p><img src="//latex.artofproblemsolving.com/8/9/0/89091ba652dcccdb288832dbd8a33423cb9d2206.png" class="latex" alt="$\mathrm{(E) \ }  \{-1,1\}$" style="vertical-align: -4px" width="89" height="18" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>Quickly verifying by plugging in values verifies that <img src="//latex.artofproblemsolving.com/b/a/e/bae5aba07d37ff6ff813107e76260fb31ad5794e.png" class="latex" alt="$-1$" style="vertical-align: 0px" width="22" height="12" /> and <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" /> are in the domain.
</p><p><img src="//latex.artofproblemsolving.com/e/d/c/edc79d79703691ce01f38e47e5cccfe30d173a54.png" class="latex" alt="$f(x)+f\left(\frac{1}{x}\right)=x$" style="vertical-align: -17px" width="146" height="43" />
</p><p>Plugging in <img src="//latex.artofproblemsolving.com/7/9/a/79a0901419c277658469226431138dfb5ede474b.png" class="latex" alt="$\frac{1}{x}$" style="vertical-align: -12px" width="12" height="37" /> into the <a href="/wiki/index.php/Function" title="Function">function</a>: 
</p><p><img src="//latex.artofproblemsolving.com/d/0/2/d024a98c21d62bd8800c55d0427adc0a08e42d3a.png" class="latex" alt="$f\left(\frac{1}{x}\right)+f\left(\frac{1}{\frac{1}{x}}\right)=\frac{1}{x}$" style="vertical-align: -19px" width="169" height="45" />
</p><p><img src="//latex.artofproblemsolving.com/0/c/9/0c99e2f37a419f18bbafe08b121cf60499370bf8.png" class="latex" alt="$f\left(\frac{1}{x}\right)+ f(x)= \frac{1}{x}$" style="vertical-align: -17px" width="149" height="43" />
</p><p>Since <img src="//latex.artofproblemsolving.com/f/0/4/f047f1d60a202045343859e4327e55eaaa20482f.png" class="latex" alt="$f(x) + f\left(\frac{1}{x}\right)$" style="vertical-align: -17px" width="108" height="43" /> cannot have two values: 
</p><p><img src="//latex.artofproblemsolving.com/0/6/4/064fd95c7f6eb31fb2912713ac171020ffe26541.png" class="latex" alt="$x = \frac{1}{x}$" style="vertical-align: -12px" width="47" height="37" />
</p><p><img src="//latex.artofproblemsolving.com/8/3/5/835a72a392da3bdadab0dbcc275dee5242bc7c17.png" class="latex" alt="$x^2 = 1$" style="vertical-align: 0px" width="50" height="15" /> 
</p><p><img src="//latex.artofproblemsolving.com/6/0/f/60f897977fafd5029aa2100cafb1020d67922730.png" class="latex" alt="$x=\pm 1$" style="vertical-align: 0px" width="56" height="12" />
</p><p>Therefore, the largest <a href="/wiki/index.php/Set" title="Set">set</a> of <a href="/wiki/index.php/Real_number" title="Real number">real numbers</a> that can be in the <a href="/wiki/index.php/Domain" title="Domain">domain</a> of <img src="//latex.artofproblemsolving.com/b/b/2/bb2c93730dbb48558bb3c4738c956c4e8f816437.png" class="latex" alt="$f$" style="vertical-align: -3px" width="10" height="16" /> is <img src="//latex.artofproblemsolving.com/c/8/c/c8c4b9203525ec6a393bc36feabceff0f3ccfb84.png" class="latex" alt="$\{-1,1\} \Rightarrow E$" style="vertical-align: -4px" width="101" height="18" />
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>We know that <img src="//latex.artofproblemsolving.com/5/4/6/546685e9f8741e90d52e53fafe86aeb1f79328a1.png" class="latex" alt="$f(x) + f \left(\frac{1}{x}\right) = x.$" style="vertical-align: -17px" width="150" height="43" /> Plugging in <img src="//latex.artofproblemsolving.com/0/6/4/064fd95c7f6eb31fb2912713ac171020ffe26541.png" class="latex" alt="$x = \frac{1}{x}$" style="vertical-align: -12px" width="47" height="37" /> we get <img src="//latex.artofproblemsolving.com/7/7/1/7719eaa8dde229d3b29828b6b5045ea6194488a5.png" class="latexcenter" alt="\[f \left(\frac{1}{x}\right) + f \left(\frac{1}{\frac{1}{x}}\right) = \frac{1}{x}\]" width="168" height="45" /> <img src="//latex.artofproblemsolving.com/6/9/c/69c45495e2dce6fe7f6e9a0558605d11e88a9c10.png" class="latexcenter" alt="\[f \left(\frac{1}{x}\right) + f(x) = \frac{1}{x}.\]" width="153" height="43" />
</p><p>Also notice <img src="//latex.artofproblemsolving.com/2/d/6/2d6c78e7baaf12360805502ec97ade5f739e32cc.png" class="latexcenter" alt="\[f \left(\frac{1}{x}\right) + f(x) = x\]" width="146" height="43" /> by the commutative property(this is the same as the equation given in the problem. We are just rearranging). So we can set <img src="//latex.artofproblemsolving.com/2/a/5/2a564fb7d4622b2e8c5b71ed477d635f6379fa0e.png" class="latex" alt="$\frac{1}{x} = x$" style="vertical-align: -12px" width="48" height="37" /> which gives us <img src="//latex.artofproblemsolving.com/4/f/4/4f424f1753ba66a66c48a233c5afe322d41467a0.png" class="latex" alt="$x = \pm 1$" style="vertical-align: 0px" width="56" height="12" /> which is answer option <img src="//latex.artofproblemsolving.com/2/0/2/202e30ddc1b6129261aecbfe53c3007f111df7d3.png" class="latex" alt="$\boxed{\mathrm{(E) \ }  \{-1,1\}}.$" style="vertical-align: -10px" width="105" height="29" />
</p>
