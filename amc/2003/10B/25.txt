<p>How many distinct four-digit numbers are divisible by <img src="//latex.artofproblemsolving.com/7/c/d/7cde695f2e4542fd01f860a89189f47a27143b66.png" class="latex" alt="$3$" width="8" height="12" /> and have <img src="//latex.artofproblemsolving.com/9/a/1/9a1b2497e7b125cd721bab4c6e2385f954fe35cb.png" class="latex" alt="$23$" width="17" height="12" /> as their last two digits?
</p><p><img src="//latex.artofproblemsolving.com/d/3/d/d3d8a7b3467d065225a88974ff5b8ac09b87d158.png" class="latex" alt="$\textbf{(A) } 27 \qquad\textbf{(B) } 30 \qquad\textbf{(C) } 33 \qquad\textbf{(D) } 81 \qquad\textbf{(E) } 90$" style="vertical-align: -4px" width="398" height="17" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<h3><span id="Solution_1_(Slow)"></span><span class="mw-headline" id="Solution_1_.28Slow.29">Solution 1 (Slow)</span></h3>
<p>To test if a number is divisible by three, you add up the digits of the number. If the sum is divisible by three, then the original number is a multiple of three. If the sum is too large, you can repeat the process until you can tell whether it is a multiple of three. This is the basis for the solution. Since the last two digits are <img src="//latex.artofproblemsolving.com/9/a/1/9a1b2497e7b125cd721bab4c6e2385f954fe35cb.png" class="latex" alt="$23$" width="17" height="12" />, the sum of the digits is <img src="//latex.artofproblemsolving.com/a/e/6/ae64b029b2867360598c37c9a9eea87c395e76c6.png" class="latex" alt="$2+3 = 5$" style="vertical-align: -1px" width="73" height="14" /> (therefore it is not divisible by three). However certain numbers can be added to make the sum of the digits a multiple of three.
</p><p><img src="//latex.artofproblemsolving.com/a/c/2/ac22d1f35ff7eefacf568854f97cd448873c71ae.png" class="latex" alt="$5+1 = 6$" style="vertical-align: -1px" width="73" height="14" />,
</p><p><img src="//latex.artofproblemsolving.com/f/4/c/f4cb7edd91ff2a131ba974a2de02ba22084a2cce.png" class="latex" alt="$5+4 = 9$" style="vertical-align: -1px" width="73" height="14" />, and so on. 
</p><p>However since the largest four-digit number ending with <img src="//latex.artofproblemsolving.com/9/a/1/9a1b2497e7b125cd721bab4c6e2385f954fe35cb.png" class="latex" alt="$23$" width="17" height="12" /> is <img src="//latex.artofproblemsolving.com/0/8/0/080ec310025d9fcfe3bd047b3148ae58de8b29cc.png" class="latex" alt="$9923$" width="35" height="12" />, the maximum sum is
</p><p><img src="//latex.artofproblemsolving.com/f/b/1/fb18122be0fcfc0f2536591e1ebde12bf094a038.png" class="latex" alt="$5+18 = 23$" style="vertical-align: -1px" width="91" height="14" />.
</p><p>Using that process we can fairly quickly compile a list of the sum of the first two digits of the number. 
</p><p><img src="//latex.artofproblemsolving.com/6/4/d/64ddcf298f712b3a213eeb5a83e875865ccb8407.png" class="latexcenter" alt="\[\{1, 4, 7, 10, 13, 16\}\]" width="138" height="18" />
</p><p>Now we find all the two-digit numbers that have any of the sums shown above. We can do this by listing all the two digit numbers <img src="//latex.artofproblemsolving.com/3/f/7/3f79c3676b62fb47243e0d357c87115b6f3395e4.png" class="latex" alt="$xy$" style="vertical-align: -3px" width="19" height="11" /> in separate cases.
</p><p><img src="//latex.artofproblemsolving.com/9/4/3/943a6fca640b2ef0375212e22d8e53b886155d61.png" class="latexcenter" alt="\[I. x+y = 1, \{10\} = 1\]" width="166" height="18" />
<img src="//latex.artofproblemsolving.com/e/9/e/e9e1cddd5cb18b50d8e6a3fbf53287d9ebe3f467.png" class="latexcenter" alt="\[II. x+y = 4, \{13, 22, 31, 40\} = 4\]" width="255" height="18" />
<img src="//latex.artofproblemsolving.com/c/5/3/c53383c2670ef646e03ed75d328e01a70c39bba7.png" class="latexcenter" alt="\[III. x+y = 7, \{16, 25, 34, 43, 52, 61, 70\} = 7\]" width="343" height="18" />
<img src="//latex.artofproblemsolving.com/8/c/9/8c9642d529a637ed6678565dacbf2b8dd86221a9.png" class="latexcenter" alt="\[IV. x+y = 10, \{19, 28, 37, 46, 55, 64, 73, 82, 91\} = 9\]" width="397" height="18" />
<img src="//latex.artofproblemsolving.com/e/5/d/e5db4469f46fe2074b96fed15d09fef52f30de45.png" class="latexcenter" alt="\[V. x+y = 13, \{49, 58, 67, 76, 85, 94\} = 6\]" width="308" height="18" />
<img src="//latex.artofproblemsolving.com/a/f/a/afa411e8f5ae7e3ba4228cd38945cec13c3180c7.png" class="latexcenter" alt="\[VI. x+y = 16, \{79, 88, 97\} = 3\]" width="242" height="18" />
</p><p>And finally, we add the number of elements in each set.
</p><p><img src="//latex.artofproblemsolving.com/9/1/0/91018c437f8d8e47a97363f6d79fa803645ae6e7.png" class="latexcenter" alt="\[1+4+7+9+6+3 = \boxed{\textbf{(B)}\ 30}\]" width="252" height="28" />
</p>

</p><p><br />
Now we have that when <img src="//latex.artofproblemsolving.com/2/f/9/2f98b3f405113241659d8c8db2ea3480346db4d0.png" class="latex" alt="$a = 4$" style="vertical-align: 0px" width="43" height="12" /> this is the same case as <img src="//latex.artofproblemsolving.com/4/8/d/48d02444a59b917b64e8bd0963b88712fc7fa0a3.png" class="latex" alt="$a = 1$" style="vertical-align: 0px" width="42" height="12" /> as <img src="//latex.artofproblemsolving.com/d/6/b/d6b66fb4b4fb2505e89978ac022a12967b23e614.png" class="latex" alt="$4 \equiv 1 \pmod 3$" style="vertical-align: -5px" width="116" height="20" />. Hence if we count the above number of solutions and multiply by <img src="//latex.artofproblemsolving.com/7/c/d/7cde695f2e4542fd01f860a89189f47a27143b66.png" class="latex" alt="$3$" width="8" height="12" /> (as <img src="//latex.artofproblemsolving.com/8/2/4/824c8e3ed99d679ab928c999606df1c1e25a67ae.png" class="latex" alt="$1 \leq a \leq 9$" style="vertical-align: -2px" width="75" height="15" />), we are home free. There are <img src="//latex.artofproblemsolving.com/a/6/b/a6b6324c9b6712419dc32cab9b831daf58cefb60.png" class="latex" alt="$4 + 3 + 3 = 10$" style="vertical-align: -1px" width="113" height="14" /> solutions. Multiplying this by <img src="//latex.artofproblemsolving.com/7/c/d/7cde695f2e4542fd01f860a89189f47a27143b66.png" class="latex" alt="$3$" width="8" height="12" /> yields <img src="//latex.artofproblemsolving.com/6/c/a/6ca687be571fd4a2945b7915e61ea0f2509df801.png" class="latex" alt="$\boxed{30}$" style="vertical-align: -5px" width="30" height="23" />. <img src="//latex.artofproblemsolving.com/3/b/b/3bbb9e25e5be095e317771a9c5d00877ee9c8012.png" class="latex" alt="$\blacksquare$" width="13" height="12" />
</p>
