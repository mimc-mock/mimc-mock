<p>For <img src="//latex.artofproblemsolving.com/c/7/c/c7c39f49c7f2e326c3c701d02b78927ba6a42a2c.png" class="latex" alt="$135^\circ &lt; x &lt; 180^\circ$" style="vertical-align: 0px" width="127" height="13" />, points <img src="//latex.artofproblemsolving.com/5/4/7/5474841b314cafc479aff5fac9f111d1cbd43c39.png" class="latex" alt="$P=(\cos x, \cos^2 x), Q=(\cot x, \cot^2 x), R=(\sin x, \sin^2 x)$" style="vertical-align: -4px" width="441" height="20" /> and <img src="//latex.artofproblemsolving.com/b/d/1/bd1b60657efb27c678179aca63946c6889e22d7f.png" class="latex" alt="$S =(\tan x, \tan^2 x)$" style="vertical-align: -4px" width="144" height="19" /> are the vertices of a trapezoid. What is <img src="//latex.artofproblemsolving.com/9/b/0/9b0028aaec510244d7dbc28e2e12780485827085.png" class="latex" alt="$\sin(2x)$" style="vertical-align: -4px" width="54" height="18" />?
</p><p><img src="//latex.artofproblemsolving.com/7/0/e/70e393dbe67cf00283e126eadaf4f08fe2c18500.png" class="latex" alt="$\textbf{(A)} \ 2-2\sqrt{2}\qquad\textbf{(B)}\ 3\sqrt{3}-6\qquad\textbf{(C)}\ 3\sqrt{2}-5\qquad\textbf{(D)}\ -\frac{3}{4}\qquad\textbf{(E)}\ 1-\sqrt{3}$" style="vertical-align: -13px" width="590" height="38" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>Let <img src="//latex.artofproblemsolving.com/2/4/c/24cf94ab3d304e3e59936e20a525c1cc0e3f1a25.png" class="latex" alt="$f,g,h,j$" style="vertical-align: -3px" width="61" height="16" /> be <img src="//latex.artofproblemsolving.com/5/2/b/52b7b2cdde90dcbbc462b75339be09e39e65d651.png" class="latex" alt="$\sin, \cos, \tan, \cot$" style="vertical-align: -3px" width="120" height="15" /> (not respectively). Then we have four points <img src="//latex.artofproblemsolving.com/f/d/b/fdb31b7d9e057c76be6abd6e45fa32576c0de409.png" class="latex" alt="$(f,f^2),(g,g^2),(h,h^2),(j,j^2)$" style="vertical-align: -4px" width="217" height="19" />, and a pair of lines each connecting two points must be parallel (as we are dealing with a trapezoid). WLOG, take the line connecting the first two points and the line connecting the last two points to be parallel, so that <img src="//latex.artofproblemsolving.com/2/6/3/26361f1a4fd86556687be753fefc90656bd9e2f7.png" class="latex" alt="$\frac{g^2-f^2}{g-f} = \frac{j^2-h^2}{j-h}$" style="vertical-align: -16px" width="144" height="42" />, or <img src="//latex.artofproblemsolving.com/e/6/e/e6e620ee41c23017d7dec8e9e046881e5ba797d4.png" class="latex" alt="$g+f = j+h$" style="vertical-align: -3px" width="107" height="16" />.
</p><p>Now, we must find how to match up <img src="//latex.artofproblemsolving.com/5/2/b/52b7b2cdde90dcbbc462b75339be09e39e65d651.png" class="latex" alt="$\sin, \cos, \tan, \cot$" style="vertical-align: -3px" width="120" height="15" /> to <img src="//latex.artofproblemsolving.com/2/4/c/24cf94ab3d304e3e59936e20a525c1cc0e3f1a25.png" class="latex" alt="$f,g,h,j$" style="vertical-align: -3px" width="61" height="16" /> so that the above equation has a solution. On the interval <img src="//latex.artofproblemsolving.com/c/7/c/c7c39f49c7f2e326c3c701d02b78927ba6a42a2c.png" class="latex" alt="$135^\circ &lt; x &lt; 180^\circ$" style="vertical-align: 0px" width="127" height="13" />, we have <img src="//latex.artofproblemsolving.com/4/0/6/406651edcaa925be06d8dd53122b41528a64bfe5.png" class="latex" alt="$\cot x &lt;-1&lt;\cos x&lt;0&lt;\sin x$" style="vertical-align: 0px" width="240" height="13" />, and <img src="//latex.artofproblemsolving.com/b/e/a/bea2a6a6adbd8a7524648060b8b5fdb20cd88978.png" class="latex" alt="$\cot x &lt;-1&lt;\tan x&lt;0&lt;\sin x$" style="vertical-align: 0px" width="242" height="13" /> so the sum of the largest and the smallest is equal to the sum of the other two, namely, <img src="//latex.artofproblemsolving.com/c/0/e/c0eed6d9984b5e7e43cab00cb28eab3128e1c4e5.png" class="latex" alt="$\sin x+\cot x = \cos x+\tan x$" style="vertical-align: -1px" width="219" height="13" />.
</p><p>Now, we perform some algebraic manipulation to find <img src="//latex.artofproblemsolving.com/2/6/d/26d449b69d76a5b5011614a311ba6b14dede279c.png" class="latex" alt="$\sin (2x)$" style="vertical-align: -4px" width="54" height="18" />:
</p><p><img src="//latex.artofproblemsolving.com/5/c/8/5c8c315fc54edee90545130754a9b82996545e79.png" class="latex" alt="$\sin x+\cot x = \cos x+\tan x \\ \sin x - \cos x = \tan x - \cot x = (\sin x - \cos x) (\sin x + \cos x) / (\sin x \cos x) \\ \sin x \cos x = \sin x + \cos x \\ (\sin x \cos x)^2 = (\sin x + \cos x)^2 \\ (\sin x \cos x)^2 = \sin^2 x + 2\sin x \cos x + \cos^2 x \\ (\sin x \cos x)^2 - 2\sin x \cos x - 1 =0$" style="vertical-align: -4px" width="562" height="118" />
</p><p>Solve the quadratic to find <img src="//latex.artofproblemsolving.com/a/4/e/a4e571b7d0d25bdb365496158eccb843631fd79f.png" class="latex" alt="$\sin x \cos x = \frac{2 - 2\sqrt{2}}{2}$" style="vertical-align: -12px" width="168" height="41" />, so that <img src="//latex.artofproblemsolving.com/f/7/0/f701bf2e10c36704b294b0a3ce537d253115ea89.png" class="latex" alt="$\sin(2x) = 2 \sin x \cos x = \boxed{\textbf{(A)} \ 2 - 2\sqrt{2}}$" style="vertical-align: -10px" width="304" height="33" />.
</p>
