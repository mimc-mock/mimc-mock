<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>The sum of the set is <img src="//latex.artofproblemsolving.com/6/f/9/6f99719771f172747555530845812d5d66ae5bbf.png" class="latex" alt="$-7-5-3-2+2+4+6+13=8$" style="vertical-align: -1px" width="284" height="14" />, so if we could have the sum in each set of parenthesis be <img src="//latex.artofproblemsolving.com/c/7/c/c7cab1a05e1e0c1d51a6a219d96577a16b7abf9d.png" class="latex" alt="$4$" style="vertical-align: 0px" width="9" height="12" /> then the minimum value would be <img src="//latex.artofproblemsolving.com/c/5/b/c5b1943cb8baaa14874a6254b1e1ba6f9a44b886.png" class="latex" alt="$2(4^2)=32$" style="vertical-align: -4px" width="81" height="19" />. Considering the set of four terms containing <img src="//latex.artofproblemsolving.com/0/3/0/030e19e4e2f1c074127cb50581e8496470c5419f.png" class="latex" alt="$13$" style="vertical-align: 0px" width="17" height="13" />, this sum could only be even if it had two or four odd terms. If it had all four odd terms then it would be <img src="//latex.artofproblemsolving.com/b/b/7/bb7972eb3ecf65c50f8bcd7b107705501a4f1541.png" class="latex" alt="$13-7-5-3=-2$" style="vertical-align: 0px" width="159" height="13" />, and with two odd terms then its minimum value is <img src="//latex.artofproblemsolving.com/3/2/a/32a7484050a13cf1919423db85e4a630dd5fbb4b.png" class="latex" alt="$13-7+2-2=6$" style="vertical-align: -1px" width="145" height="14" />, so we cannot achieve two sums of <img src="//latex.artofproblemsolving.com/c/7/c/c7cab1a05e1e0c1d51a6a219d96577a16b7abf9d.png" class="latex" alt="$4$" style="vertical-align: 0px" width="9" height="12" />. The closest we could have to <img src="//latex.artofproblemsolving.com/c/7/c/c7cab1a05e1e0c1d51a6a219d96577a16b7abf9d.png" class="latex" alt="$4$" style="vertical-align: 0px" width="9" height="12" /> and <img src="//latex.artofproblemsolving.com/c/7/c/c7cab1a05e1e0c1d51a6a219d96577a16b7abf9d.png" class="latex" alt="$4$" style="vertical-align: 0px" width="9" height="12" /> is <img src="//latex.artofproblemsolving.com/7/c/d/7cde695f2e4542fd01f860a89189f47a27143b66.png" class="latex" alt="$3$" width="8" height="12" /> and <img src="//latex.artofproblemsolving.com/7/9/0/79069377f91364c2f87a64e5f9f562a091c8a6c1.png" class="latex" alt="$5$" width="8" height="12" />, which can be achieved through <img src="//latex.artofproblemsolving.com/9/8/c/98c35490ab57187506ffd67efc9536a4cefd7a66.png" class="latex" alt="$13-7-5+2$" style="vertical-align: -1px" width="111" height="14" /> and <img src="//latex.artofproblemsolving.com/8/7/9/8794dc5a8faad120291e1fdf14bdca348c609911.png" class="latex" alt="$6-3-2+4$" style="vertical-align: -1px" width="103" height="14" />. So the minimum possible value is <img src="//latex.artofproblemsolving.com/f/e/f/feff2cf074916c19e74963cfe1634789711a0546.png" class="latex" alt="$3^2+5^2=34\Rightarrow\boxed{\mathrm{C}}$" style="vertical-align: -5px" width="151" height="24" />.
</p>
