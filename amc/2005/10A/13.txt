<p>How many positive integers <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> satisfy the following condition:
</p><p><img src="//latex.artofproblemsolving.com/b/2/e/b2ea5c8fd0c94e54e90f22411acb9049ca06f629.png" class="latex" alt="$(130n)^{50} &gt; n^{100} &gt; 2^{200}\&#160;?$" style="vertical-align: -5px" width="192" height="21" />
</p><p><img src="//latex.artofproblemsolving.com/6/d/f/6df6b568b228be9bd794938ded09bbb887d48f91.png" class="latex" alt="$\textbf{(A) } 0\qquad \textbf{(B) } 7\qquad \textbf{(C) } 12\qquad \textbf{(D) } 65\qquad \textbf{(E) } 125$" style="vertical-align: -5px" width="392" height="18" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>We're given <img src="//latex.artofproblemsolving.com/6/5/d/65dbe012bf4a0b344c90567d4c90a1f5b22d494c.png" class="latex" alt="$(130n)^{50} &gt; n^{100} &gt; 2^{200}$" style="vertical-align: -4px" width="174" height="19" />, so
</p><p><img src="//latex.artofproblemsolving.com/8/1/1/8113297a6dcc5a47492dab19bb3eb95305ef2df9.png" class="latex" alt="$\sqrt[50]{(130n)^{50}} &gt; \sqrt[50]{n^{100}} &gt; \sqrt[50]{2^{200}}$" style="vertical-align: -10px" width="243" height="32" /> (because all terms are positive) and thus
</p><p><img src="//latex.artofproblemsolving.com/c/5/5/c554db2ac358ff83b197b53e5fa6d2a0c27bd94f.png" class="latex" alt="$130n &gt; n^2 &gt; 2^4$" style="vertical-align: 0px" width="120" height="15" /> 
</p><p><img src="//latex.artofproblemsolving.com/7/f/4/7f403deb1048b6d8bd457ce09d776709c93b51db.png" class="latex" alt="$130n &gt; n^2 &gt; 16$" style="vertical-align: 0px" width="122" height="15" />
</p><p>Solving each part separately: 
</p><p><img src="//latex.artofproblemsolving.com/2/1/6/216085670b959ae9774516d5876094671379e26b.png" class="latex" alt="$n^2 &gt; 16 \Longrightarrow n &gt; 4$" style="vertical-align: 0px" width="144" height="15" /> 
</p><p><img src="//latex.artofproblemsolving.com/6/6/2/6623ad47657455102a5b714cb4bab021005b1352.png" class="latex" alt="$130n &gt; n^2 \Longrightarrow 130 &gt; n$" style="vertical-align: 0px" width="182" height="15" /> 
</p><p>So <img src="//latex.artofproblemsolving.com/b/a/2/ba2a972ddc952987f08c524f16799f3832f5ce85.png" class="latex" alt="$4 &lt; n &lt; 130$" style="vertical-align: 0px" width="95" height="13" />. 
</p><p>Therefore the answer is the number of <a href="/wiki/index.php/Positive_integer" title="Positive integer">positive integers</a> over the interval <img src="//latex.artofproblemsolving.com/0/6/e/06e3346174f6070eb320a65c700bd63b8122c655.png" class="latex" alt="$(4,130)$" style="vertical-align: -5px" width="60" height="20" /> which is <img src="//latex.artofproblemsolving.com/4/8/b/48b23308d79b09defa5fb5af390843af6d2496e9.png" class="latex" alt="$129-5+1 = \boxed{\textbf{(E) }125}$" style="vertical-align: -11px" width="188" height="30" />
</p><p><br />
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>We're given <img src="//latex.artofproblemsolving.com/9/4/3/9430051004017b360b0bc200322865dbb366108b.png" class="latex" alt="$\left(130n\right)^{50}&gt;n^{100}&gt;2^{200}$" style="vertical-align: -5px" width="178" height="23" />.
</p><p>Alternatively to solution 1, first deal with the first half: <img src="//latex.artofproblemsolving.com/9/6/7/9670e27618faa74b19bb7296465b24b164337903.png" class="latex" alt="$\left(130n\right)^{50}&gt;\left(n^{2}\right)^{50}$" style="vertical-align: -7px" width="141" height="26" />. Because the exponents are equal, we can ignore them and solve for <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" />: <img src="//latex.artofproblemsolving.com/f/4/5/f45428ba7aa53207db831c00d745c2eeb9d092f7.png" class="latex" alt="$130n&gt;n^{2}$" style="vertical-align: -2px" width="83" height="18" />, or <img src="//latex.artofproblemsolving.com/e/6/5/e651b4e11f81c37e381ef1396de6da63052952e9.png" class="latex" alt="$n&lt;130$" style="vertical-align: -2px" width="64" height="15" />.
</p><p>The second half: <img src="//latex.artofproblemsolving.com/b/8/a/b8ad7bc253f500b1a262d4f9ed6c0eb6bc0d146f.png" class="latex" alt="$n^{100}&gt;2^{200}$" style="vertical-align: -2px" width="87" height="18" />, or <img src="//latex.artofproblemsolving.com/2/7/f/27fde44b2bf7609390ae0d9bf202a1e87192ea19.png" class="latex" alt="$n^{100}&gt;4^{100}$" style="vertical-align: -2px" width="87" height="18" />, which means <img src="//latex.artofproblemsolving.com/e/d/0/ed0ecf70e9e1af12c8ba1fd315b1e88e11679d44.png" class="latex" alt="$n&gt;4$" style="vertical-align: 0px" width="44" height="12" />.
</p><p>Therefore <img src="//latex.artofproblemsolving.com/7/d/f/7df316d756497c6b95efdc71d66f3be7c6ac8445.png" class="latex" alt="$4&lt;n&lt;130$" style="vertical-align: 0px" width="95" height="13" /> and <img src="//latex.artofproblemsolving.com/d/c/8/dc80c422fdab47dda5ccdecba7c7ce5d29957798.png" class="latex" alt="$n=\left\{5,\ 6,\ 7,\ \cdots ,\ 128,\ 129\right\}$" style="vertical-align: -5px" width="232" height="20" /> which contains the same number of elements as <img src="//latex.artofproblemsolving.com/6/9/f/69fda43ffa0d8dd9a11d410b5396be34b24fe899.png" class="latex" alt="$\left\{1,\ 2,\ 3,\ \cdots ,\ 124,\ 125\right\}$" style="vertical-align: -5px" width="197" height="20" /> which clearly contains <img src="//latex.artofproblemsolving.com/b/8/2/b829a6f0810a25e9ca9f85d690f2bafd0f2c1773.png" class="latex" alt="$125$" style="vertical-align: 0px" width="26" height="13" /> values or choice <img src="//latex.artofproblemsolving.com/f/e/3/fe3fe31646698300eb3e87fdc64e1355ef617c71.png" class="latex" alt="$\boxed{\textbf{(E) } 125}$" style="vertical-align: -11px" width="74" height="30" />.
</p><p>~JH. L
</p>
