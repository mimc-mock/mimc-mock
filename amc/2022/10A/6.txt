<p>Which expression is equal to <img src="//latex.artofproblemsolving.com/9/8/4/984209dd4e5f26c2736cb3fd5c2248f83f75fd36.png" class="latexcenter" alt="\[\left|a-2-\sqrt{(a-1)^2}\right|\]" width="150" height="44" /> for <img src="//latex.artofproblemsolving.com/6/f/5/6f537b3b21a6edd1fd0ffe37d1fb22a989df562b.png" class="latex" alt="$a&lt;0?$" style="vertical-align: -2px" width="53" height="15" />
</p><p><img src="//latex.artofproblemsolving.com/3/5/4/3543833138070a21b74029f2fecdd045cc6f8278.png" class="latex" alt="$\textbf{(A) } 3-2a \qquad \textbf{(B) } 1-a \qquad \textbf{(C) } 1 \qquad \textbf{(D) } a+1 \qquad \textbf{(E) } 3$" style="vertical-align: -5px" width="460" height="18" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>We have 
<img src="//latex.artofproblemsolving.com/4/1/3/4132eefb93f7cd74539174068e167d9127ac7f41.png" class="latexcenter" alt="\begin{align*} \left|a-2-\sqrt{(a-1)^2}\right| &amp;= \left|a-2-|a-1|\right| \\ &amp;=\left|a-2-(1-a)\right| \\ &amp;=\left|2a-3\right| \\ &amp;=\boxed{\textbf{(A) } 3-2a}. \end{align*}" width="303" height="129" />
~MRENTHUSIASM
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>Assume that <img src="//latex.artofproblemsolving.com/e/2/0/e20f40b858f202848232ab948fca5c19ded2a1fd.png" class="latex" alt="$a=-1.$" style="vertical-align: -2px" width="64" height="15" /> Then, the given expression simplifies to <img src="//latex.artofproblemsolving.com/7/9/0/79069377f91364c2f87a64e5f9f562a091c8a6c1.png" class="latex" alt="$5$" width="8" height="12" />:
<img src="//latex.artofproblemsolving.com/5/8/e/58e98265468169e13e3fb84f034344aa3a62cadf.png" class="latexcenter" alt="\begin{align*} \left|a-2-\sqrt{(a-1)^2}\right| &amp;= \left|-1-2-\sqrt{(-1-1)^2}\right| \\ &amp;= \left|-1-2-\sqrt{4}\right| \\ &amp;= \left|-1-2-2\right| \\ &amp;= 5. \end{align*}" width="357" height="128" />
Then, we test each of the answer choices to see which one is equal to <img src="//latex.artofproblemsolving.com/7/9/0/79069377f91364c2f87a64e5f9f562a091c8a6c1.png" class="latex" alt="$5$" width="8" height="12" />:
</p><p><img src="//latex.artofproblemsolving.com/e/b/5/eb5894989301eac86cb0536f0a8ae23256a8a972.png" class="latex" alt="$\textbf{(A) } 3-2a = 3-2\cdot(-1) = 3+2 = 5.$" style="vertical-align: -5px" width="306" height="20" /> 
</p><p><img src="//latex.artofproblemsolving.com/4/c/6/4c6994fab310684015062119ba8f2cb9e6ddd8ca.png" class="latex" alt="$\textbf{(B) } 1-a = 1-(-1) = 2 \neq 5.$" style="vertical-align: -5px" width="242" height="20" />
</p><p><img src="//latex.artofproblemsolving.com/a/c/e/ace458ac6016ee4cee38b85fadc07367a56edca2.png" class="latex" alt="$\textbf{(C) } 1 \neq 5.$" style="vertical-align: -5px" width="84" height="18" />
</p><p><img src="//latex.artofproblemsolving.com/0/b/3/0b3321b64a1046dbb17cb91352a58c9866e5e9f3.png" class="latex" alt="$\textbf{(D) } a+1 = -1+1 = 0 \neq 5.$" style="vertical-align: -5px" width="229" height="18" />
</p><p><img src="//latex.artofproblemsolving.com/0/1/d/01d4319582cba97c0cc53120851fee972f9bc7a5.png" class="latex" alt="$\textbf{(E) } 3 \neq 5.$" style="vertical-align: -5px" width="83" height="18" />
</p><p>The only answer choice equal to <img src="//latex.artofproblemsolving.com/7/9/0/79069377f91364c2f87a64e5f9f562a091c8a6c1.png" class="latex" alt="$5$" width="8" height="12" /> for <img src="//latex.artofproblemsolving.com/8/1/7/817555099cf3c1ef68bd2e5b1f4bd27d48cbbb6c.png" class="latex" alt="$a=-1$" style="vertical-align: 0px" width="56" height="12" /> is <img src="//latex.artofproblemsolving.com/e/2/f/e2f0350aa68d5fc851e14cefa01444cf039b1716.png" class="latex" alt="$\boxed{\textbf{(A) } 3-2a}.$" style="vertical-align: -11px" width="104" height="30" />
</p><p>-MathWizard09
</p>
