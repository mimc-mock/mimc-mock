<h3><span class="mw-headline" id="Solution_3">Solution 3</span></h3>
<p>Note that <img src="//latex.artofproblemsolving.com/b/6/f/b6f89cc57902f8005b15b224428dfb302f440e77.png" class="latex" alt="$( x^3 + y^3 ) ( \frac{1}{y^2} + \frac{1}{x^2} ) = x + \frac{x^3}{y^2} + \frac{y^3}{x^2} + y.$" style="vertical-align: -16px" width="308" height="42" /> Now, we only need to find the values of <img src="//latex.artofproblemsolving.com/4/4/6/446930c357814b554e45bce0229906b2cb75fae7.png" class="latex" alt="$x^3 + y^3$" style="vertical-align: -3px" width="55" height="18" /> and <img src="//latex.artofproblemsolving.com/9/e/3/9e3a38ede295e51e815d30fa097453f825390a35.png" class="latex" alt="$\frac{1}{y^2} + \frac{1}{x^2}.$" style="vertical-align: -16px" width="68" height="40" /> <br />

<h3><span id="Solution_4_(Bashing)"></span><span class="mw-headline" id="Solution_4_.28Bashing.29">Solution 4 (Bashing)</span></h3>
<p>This is basically bashing using Vieta's formulas to find <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> and <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" /> (which I highly do not recommend, I only wrote this solution for fun).
</p><p>We use Vieta's to find a quadratic relating <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> and <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" />. We set <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> and <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" /> to be the roots of the quadratic <img src="//latex.artofproblemsolving.com/6/6/8/66833390e06cd04fca02064efc8e5b0987040210.png" class="latex" alt="$Q ( n ) = n^2 - 4n - 2$" style="vertical-align: -4px" width="155" height="19" /> (because <img src="//latex.artofproblemsolving.com/5/4/f/54f37629dc5b0db0dc7655a6ef18988faa0ae552.png" class="latex" alt="$x + y = 4$" style="vertical-align: -3px" width="75" height="15" />, and <img src="//latex.artofproblemsolving.com/c/1/5/c15410e783f1eb41b13e995560105e8e832f3189.png" class="latex" alt="$xy = -2$" style="vertical-align: -3px" width="66" height="15" />). We can solve the quadratic to get the roots <img src="//latex.artofproblemsolving.com/6/5/2/65289f11b0da767979a5f70c315d67fd7c471afc.png" class="latex" alt="$2 + \sqrt{6}$" style="vertical-align: -1px" width="56" height="18" /> and <img src="//latex.artofproblemsolving.com/7/3/6/7363644d545274aed189e597abd078029116e486.png" class="latex" alt="$2 - \sqrt{6}$" style="vertical-align: -1px" width="56" height="18" />. <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> and <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" /> are "interchangeable", meaning that it doesn't matter which solution <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> or <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" /> is, because it'll return the same result when plugged in. So we plug in <img src="//latex.artofproblemsolving.com/6/5/2/65289f11b0da767979a5f70c315d67fd7c471afc.png" class="latex" alt="$2 + \sqrt{6}$" style="vertical-align: -1px" width="56" height="18" /> for <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> and <img src="//latex.artofproblemsolving.com/7/3/6/7363644d545274aed189e597abd078029116e486.png" class="latex" alt="$2 - \sqrt{6}$" style="vertical-align: -1px" width="56" height="18" /> and get <img src="//latex.artofproblemsolving.com/a/c/1/ac1322b27e9f4aee317bab57a2ef8a2af7af6f5a.png" class="latex" alt="$\boxed{\textbf{(D)}\ 440}$" style="vertical-align: -10px" width="74" height="28" /> as our answer.

<h3><span id="Solution_5_(Bashing_Part_2)"></span><span class="mw-headline" id="Solution_5_.28Bashing_Part_2.29">Solution 5 (Bashing Part 2)</span></h3>
<p>This usually wouldn't work for most problems like this, but we're lucky that we can quickly expand and factor this expression in this question.
</p><p>We first change the original expression to <img src="//latex.artofproblemsolving.com/2/5/e/25eff3ec4318531fb50b452dae9478d4a3393fdf.png" class="latex" alt="$4 + \frac{x^5 + y^5}{x^2 y^2}$" style="vertical-align: -16px" width="91" height="42" />, because <img src="//latex.artofproblemsolving.com/5/4/f/54f37629dc5b0db0dc7655a6ef18988faa0ae552.png" class="latex" alt="$x + y = 4$" style="vertical-align: -3px" width="75" height="15" />. This is equal to <img src="//latex.artofproblemsolving.com/9/f/4/9f48a59ae75e6ffb924758b81065ea519539d35a.png" class="latex" alt="$4 + \frac{(x + y)(x^4 - x^3 y + x^2 y^2 - x y^3 + y^4)}{4} = x^4 + y^4 - x^3 y - x y^3 + 8$" style="vertical-align: -13px" width="530" height="39" />. We can factor and reduce <img src="//latex.artofproblemsolving.com/0/f/e/0fe9050d4555951d92a7afe716b5e11d24f95ba0.png" class="latex" alt="$x^4 + y^4$" style="vertical-align: -3px" width="56" height="18" /> to <img src="//latex.artofproblemsolving.com/c/6/f/c6f847009fb87b4ec6a3d37531554e4ac5b8d914.png" class="latex" alt="$(x^2 + y^2)^2 - 2 x^2 y^2 = ((x + y)^2 - 2xy)^2 - 8 = 400 - 8 = 392$" style="vertical-align: -4px" width="470" height="19" />. Now our expression is just <img src="//latex.artofproblemsolving.com/2/9/e/29efda76518d4db48f408917b70d2c07c76609ed.png" class="latex" alt="$400 - (x^3 y + x y^3)$" style="vertical-align: -4px" width="139" height="19" />. We factor <img src="//latex.artofproblemsolving.com/c/d/4/cd42c53a0209eeffad6019b25643ca108ff52893.png" class="latex" alt="$x^3 y + x y^3$" style="vertical-align: -3px" width="75" height="18" /> to get <img src="//latex.artofproblemsolving.com/2/c/b/2cb98b4a0574c4ed5714ed22992d902881380b3e.png" class="latex" alt="$(xy)(x^2 + y^2) = -40$" style="vertical-align: -4px" width="161" height="19" />. So the answer would be <img src="//latex.artofproblemsolving.com/0/e/4/0e4ae95c05852642993e0b36aaeb2864487d6d81.png" class="latex" alt="$400 - (-40)  = \boxed{\textbf{(D)} 440}$" style="vertical-align: -10px" width="188" height="28" />.

<h3><span id="Solution_6_(Complete_Binomial_Theorem)"></span><span class="mw-headline" id="Solution_6_.28Complete_Binomial_Theorem.29">Solution 6 (Complete Binomial Theorem)</span></h3>
<p>We first simplify the expression to <img src="//latex.artofproblemsolving.com/f/a/0/fa0a4d3ca3a7a8bb3dcab7a9673704a8a60c54d2.png" class="latexcenter" alt="\[x + y + \frac{x^5 + y^5}{x^2y^2}.\]" width="128" height="42" />
Then, we can solve for <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> and <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" /> given the system of equations in the problem.
Since <img src="//latex.artofproblemsolving.com/8/4/1/841bb6db55e6608832da14733ada215aa6fde89a.png" class="latex" alt="$xy = -2,$" style="vertical-align: -3px" width="71" height="15" /> we can substitute <img src="//latex.artofproblemsolving.com/a/a/8/aa838c8d2a6aee32e95aba0736613b277d956dd0.png" class="latex" alt="$\frac{-2}{x}$" style="vertical-align: -12px" width="25" height="37" /> for <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" />. 
Thus, this becomes the equation <img src="//latex.artofproblemsolving.com/b/1/e/b1e78655012111ea3aadc704573854c903b6a58e.png" class="latexcenter" alt="\[x - \frac{2}{x} = 4.\]" width="83" height="37" />
Multiplying both sides by <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" />, we obtain <img src="//latex.artofproblemsolving.com/7/d/1/7d1d0211357dcd417333cd66474435adec41d241.png" class="latex" alt="$x^2 - 2 = 4x,$" style="vertical-align: -3px" width="97" height="18" /> or 
<img src="//latex.artofproblemsolving.com/c/0/2/c02188679dae5ae073e7e910ed83a11327d7adc3.png" class="latexcenter" alt="\[x^2 - 4x - 2 = 0.\]" width="127" height="15" />
By the quadratic formula we obtain <img src="//latex.artofproblemsolving.com/f/d/4/fd49c8506ea89307b9e8a593d6b3c8a4ef3f3a76.png" class="latex" alt="$x = 2 \pm \sqrt{6}$" style="vertical-align: -1px" width="90" height="18" />. 
We also easily find that given <img src="//latex.artofproblemsolving.com/f/d/4/fd49c8506ea89307b9e8a593d6b3c8a4ef3f3a76.png" class="latex" alt="$x = 2 \pm \sqrt{6}$" style="vertical-align: -1px" width="90" height="18" />, <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" /> equals the conjugate of <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" />. 
Thus, plugging our values in for <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> and <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" />, our expression equals

By the binomial theorem, we observe that every second terms of the expansions <img src="//latex.artofproblemsolving.com/0/7/7/077791531216b77f911a9821f320c1924efafb7e.png" class="latex" alt="$x^5$" width="16" height="15" /> and <img src="//latex.artofproblemsolving.com/1/1/e/11ef1e403c620831fb2177c856a58586354a5806.png" class="latex" alt="$y^5$" style="vertical-align: -3px" width="15" height="18" /> will cancel out (since a positive plus a negative of the same absolute value equals zero). We also observe that the other terms not canceling out are doubled when summing the expansions of <img src="//latex.artofproblemsolving.com/5/e/c/5ec85f200708dc5943e999d5514ff04ac494fcd0.png" class="latex" alt="$x^5 + y^5$" style="vertical-align: -3px" width="55" height="18" />.

which equals
<img src="//latex.artofproblemsolving.com/1/c/e/1cea39dd245da59b8113ac079c7a6aefb280e797.png" class="latexcenter" alt="\[4 + \frac{2(872)}{4}\]" width="84" height="39" />

<h3><span class="mw-headline" id="Solution_8">Solution 8</span></h3>
<p>We can use Newton Sums to solve this problem.
We start by noticing that we can rewrite the equation as <img src="//latex.artofproblemsolving.com/8/8/7/887f7cd6cf73932840a0faa710853e1deebdc87a.png" class="latex" alt="$\frac{x^3}{y^2} + \frac{y^3}{x^2} + x + y.$" style="vertical-align: -16px" width="134" height="42" />
Then, we know that <img src="//latex.artofproblemsolving.com/8/2/4/8246e37b44c1ee51caccfa5a3c9edaf5f64f825c.png" class="latex" alt="$x + y = 4,$" style="vertical-align: -3px" width="79" height="15" /> so we have <img src="//latex.artofproblemsolving.com/0/e/5/0e594f7c408606f8491b9a5e7dc03f22580e26a9.png" class="latex" alt="$\frac{x^3}{y^2} + \frac{y^3}{x^2} + 4.$" style="vertical-align: -16px" width="100" height="42" />

Next, we can plug in these values of <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> and <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" /> to get <img src="//latex.artofproblemsolving.com/1/f/4/1f4950c043c3f974420546fa354a4b720ef9494e.png" class="latex" alt="$\frac{x^3}{y^2} + \frac{y^3}{x^2} = \frac{x^5}{4} + \frac{y^5}{4},$" style="vertical-align: -16px" width="159" height="42" /> which is the same as <img src="//latex.artofproblemsolving.com/9/8/7/987c9b378d902f8d0e810bbb3623aab4326118d6.png" class="latexcenter" alt="\[\frac{x^3}{y^2} + \frac{y^3}{x^2} = \frac{x^5 + y^5}{4}.\]" width="152" height="42" />

<img src="//latex.artofproblemsolving.com/f/1/6/f1668acc070546f7da5d344d79e2acddf64da5a8.png" class="latexcenter" alt="\[P_1 = S_1\]" width="59" height="15" />

<img src="//latex.artofproblemsolving.com/7/4/6/746fbb0d1ddd2884148a21fd504d1a8677f41e79.png" class="latexcenter" alt="\[P_1 = S_1 = 4\]" width="94" height="15" />

<img src="//latex.artofproblemsolving.com/a/5/6/a56c22c3975fc1ba4e48fea5c2e48e4353306df7.png" class="latexcenter" alt="\[P_3 = P_2 S_1 - P_1 S_2 = 80 - (-8) = 88\]" width="284" height="18" />
<img src="//latex.artofproblemsolving.com/5/d/e/5de4166b710b70a1f4d095b64105369d14577ab9.png" class="latexcenter" alt="\[P_4 = P_3 S_1 - P_2 S_2 = 88 \cdot 4 - (-40) = 392\]" width="324" height="18" />
<img src="//latex.artofproblemsolving.com/4/0/2/4028bc1ee380fcd0129c8f6a28adecb4d9e24f06.png" class="latexcenter" alt="\[P_5 = P_4 S_1 - P_3 S_2 = 392 \cdot 4 - (-2) \cdot 88 = 1744.\]" width="368" height="18" />
