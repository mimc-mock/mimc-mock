<p>The acronym AMC is shown in the rectangular grid below with grid lines spaced <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" /> unit apart. In units, what is the sum of the lengths of the line segments that form the acronym AMC<img src="//latex.artofproblemsolving.com/4/a/8/4a8960d40d67144344fa86ca431cc1d6491690c5.png" class="latex" alt="$?$" width="8" height="12" />
</p><p><img src="//latex.artofproblemsolving.com/6/6/4/6649eb6b62018218d4086bde7597f2da9b4b10e3.png" class="latexcenter" alt="[asy] import olympiad; unitsize(25); for (int i = 0; i &lt; 3; ++i) { for (int j = 0; j &lt; 9; ++j) { pair A = (j,i);  } } for (int i = 0; i &lt; 3; ++i) { for (int j = 0; j &lt; 9; ++j) { if (j&#160;!= 8) { draw((j,i)--(j+1,i), dashed); } if (i&#160;!= 2) { draw((j,i)--(j,i+1), dashed); } } } draw((0,0)--(2,2),linewidth(2)); draw((2,0)--(2,2),linewidth(2)); draw((1,1)--(2,1),linewidth(2)); draw((3,0)--(3,2),linewidth(2)); draw((5,0)--(5,2),linewidth(2)); draw((4,1)--(3,2),linewidth(2)); draw((4,1)--(5,2),linewidth(2)); draw((6,0)--(8,0),linewidth(2)); draw((6,2)--(8,2),linewidth(2)); draw((6,0)--(6,2),linewidth(2)); [/asy]" width="338" height="88" />
</p><p><img src="//latex.artofproblemsolving.com/1/9/1/1919ebef6abaaa0fec0589dd23fef0cc980fce9f.png" class="latex" alt="$\textbf{(A) } 17 \qquad \textbf{(B) } 15 + 2\sqrt{2} \qquad \textbf{(C) } 13 + 4\sqrt{2} \qquad \textbf{(D) } 11 + 6\sqrt{2} \qquad \textbf{(E) } 21$" style="vertical-align: -4px" width="564" height="21" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>Each of the straight line segments have length <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" /> and each of the slanted line segments have length <img src="//latex.artofproblemsolving.com/2/c/4/2c4dc1409208ff6a3039f4871d85c16763334983.png" class="latex" alt="$\sqrt{2}$" style="vertical-align: -1px" width="24" height="18" /> (this can be deducted using <img src="//latex.artofproblemsolving.com/2/2/2/22231a5cdd4fe05a11f4b577e3b81221f3017a85.png" class="latex" alt="$45-45-90$" style="vertical-align: 0px" width="98" height="13" />, pythag, trig, or just sense)
</p><p>There area a total of <img src="//latex.artofproblemsolving.com/0/3/0/030e19e4e2f1c074127cb50581e8496470c5419f.png" class="latex" alt="$13$" style="vertical-align: 0px" width="17" height="13" /> straight lines segments and <img src="//latex.artofproblemsolving.com/c/7/c/c7cab1a05e1e0c1d51a6a219d96577a16b7abf9d.png" class="latex" alt="$4$" style="vertical-align: 0px" width="9" height="12" /> slanted line segments. The sum is <img src="//latex.artofproblemsolving.com/e/a/1/ea195e2a18770b5749c8a5f4bad1af4ad6531dc8.png" class="latex" alt="$\boxed{\textbf{(C) } 13 + 4\sqrt{2}}$" style="vertical-align: -11px" width="122" height="35" /> 
</p><p>~quacker88
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>Either count the straight or diagonals and deduce from the answers that the only answer possible is <img src="//latex.artofproblemsolving.com/e/a/1/ea195e2a18770b5749c8a5f4bad1af4ad6531dc8.png" class="latex" alt="$\boxed{\textbf{(C) } 13 + 4\sqrt{2}}$" style="vertical-align: -11px" width="122" height="35" />.
