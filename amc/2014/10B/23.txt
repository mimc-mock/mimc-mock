<p>A sphere is inscribed in a truncated right circular cone as shown. The volume of the truncated cone is twice that of the sphere. What is the ratio of the radius of the bottom base of the truncated cone to the radius of the top base of the truncated cone?
</p><p><img src="//latex.artofproblemsolving.com/6/e/d/6ed55192447af710c595ba632ef42c8a98251f94.png" class="latex" alt="$\text{(A) } \dfrac32 \quad \text{(B) } \dfrac{1+\sqrt5}2 \quad \text{(C) } \sqrt3 \quad \text{(D) } 2 \quad \text{(E) } \dfrac{3+\sqrt5}2$" style="vertical-align: -12px" width="383" height="41" />
</p><p><img src="//latex.artofproblemsolving.com/b/c/a/bcae66453a62181e1866ee687a7aed449a6594da.png" class="latexcenter" alt="[asy] real r=(3+sqrt(5))/2; real s=sqrt(r); real Brad=r; real brad=1; real Fht = 2*s; import graph3; import solids; currentprojection=orthographic(1,0,.2); currentlight=(10,10,5); revolution sph=sphere((0,0,Fht/2),Fht/2); draw(surface(sph),green+white+opacity(0.5)); //triple f(pair t) {return (t.x*cos(t.y),t.x*sin(t.y),t.x^(1/n)*sin(t.y/n));} triple f(pair t) { triple v0 = Brad*(cos(t.x),sin(t.x),0); triple v1 = brad*(cos(t.x),sin(t.x),0)+(0,0,Fht); return (v0 + t.y*(v1-v0)); } triple g(pair t) { return (t.y*cos(t.x),t.y*sin(t.x),0); } surface sback=surface(f,(3pi/4,0),(7pi/4,1),80,2); surface sfront=surface(f,(7pi/4,0),(11pi/4,1),80,2); surface base = surface(g,(0,0),(2pi,Brad),80,2); draw(sback,gray(0.9)); draw(sfront,gray(0.5)); draw(base,gray(0.9)); draw(surface(sph),gray(0.4));[/asy]" width="252" height="188" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>First, we draw the vertical cross-section passing through the middle of the frustum.
Let the top base have a diameter of <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" />, and the bottom base have a diameter of <img src="//latex.artofproblemsolving.com/0/a/5/0a5745df50c8607bdc9b8de4968315633a3047dc.png" class="latex" alt="$2r$" width="17" height="12" />.
</p><p><img src="//latex.artofproblemsolving.com/3/4/b/34b3cd62c08642105b975a53c232c20e13bc53c4.png" class="latexcenter" alt="[asy] size(7cm); pair A,B,C,D; real r = (3+sqrt(5))/2; real s = sqrt(r); A = (-r,0); B = (r,0); C = (1,2*s); D = (-1,2*s); draw(A--B--C--D--cycle); pair O = (0,s); draw(shift(O)*scale(s)*unitcircle); dot(O); pair X,Y; X = (0,0); Y = (0,2*s); draw(X--Y); label(&quot;$r-1$&quot;,(r/2+1/2,0),S); label(&quot;$1$&quot;,(Y+C)/2,N); label(&quot;$s$&quot;,(O+Y)/2,W); label(&quot;$s$&quot;,(O+X)/2,W); draw(B--C--(1,0)--cycle,blue+1bp); pair P = 0.73*C+0.27*B; draw(O--P); dot(P); label(&quot;$1$&quot;,(C+P)/2,NE); label(&quot;$r$&quot;,(B+P)/2,NE); [/asy]" width="332" height="248" />
</p><p>Then using the Pythagorean theorem we have:
<img src="//latex.artofproblemsolving.com/e/a/9/ea93cc1977198893b2a5a61564364d7877dab0e4.png" class="latex" alt="$(r+1)^2=(2s)^2+(r-1)^2$" style="vertical-align: -4px" width="207" height="19" />, 
which is equivalent to:
<img src="//latex.artofproblemsolving.com/7/7/a/77ab66312f957f5bad1c74f1295b0021c8e0bc7f.png" class="latex" alt="$r^2+2r+1=4s^2+r^2-2r+1$" style="vertical-align: -1px" width="245" height="16" />.
Subtracting <img src="//latex.artofproblemsolving.com/f/2/4/f2443c3a311121b07c45b465114d975d2463f30e.png" class="latex" alt="$r^2-2r+1$" style="vertical-align: -1px" width="86" height="16" /> from both sides,
<img src="//latex.artofproblemsolving.com/f/8/a/f8ae0616ddb27d77b602943da58c40543421cfe9.png" class="latex" alt="$4r=4s^2$" style="vertical-align: 0px" width="65" height="15" />, and solving for <img src="//latex.artofproblemsolving.com/f/3/7/f37bba504894945c07a32f5496d74299a37aa51c.png" class="latex" alt="$s$" width="8" height="8" />, we end up with
<img src="//latex.artofproblemsolving.com/6/d/f/6dfde7da597d8e7793a419e59f3bd49a686e64c5.png" class="latexcenter" alt="\[s=\sqrt{r}.\]" width="59" height="18" />
Next, we can find the volume of the frustum (truncated cone) and of the sphere. Since we know <img src="//latex.artofproblemsolving.com/c/b/a/cba5581ebf308590a2936a23565e30160a3beb55.png" class="latex" alt="$V_{\text{frustum}}=2V_{\text{sphere}}$" style="vertical-align: -5px" width="137" height="17" />, we can solve for <img src="//latex.artofproblemsolving.com/f/3/7/f37bba504894945c07a32f5496d74299a37aa51c.png" class="latex" alt="$s$" width="8" height="8" />
using <img src="//latex.artofproblemsolving.com/0/0/b/00b0b5f4ac00b5acffd024417ae55e9435c90385.png" class="latex" alt="$V_{\text{frustum}}=\frac{\pi h}{3}(R^2+r^2+Rr)$" style="vertical-align: -12px" width="223" height="37" />
we get:
<img src="//latex.artofproblemsolving.com/c/2/4/c24cd9f71d863e02bc76f9a299e0ebc537573a20.png" class="latexcenter" alt="\[V_{\text{frustum}}=\frac{\pi\cdot2\sqrt{r}}{3}(r^2+r+1)\]" width="232" height="39" />
Using  <img src="//latex.artofproblemsolving.com/a/a/8/aa8a2b7650e080923dc1b9640985a2035c31caee.png" class="latex" alt="$V_{\text{sphere}}=\dfrac{4s^{3}\pi}{3}$" style="vertical-align: -12px" width="111" height="39" />, we get 
<img src="//latex.artofproblemsolving.com/8/2/d/82da68e37e5b3401b8de5a8588b92760176358cd.png" class="latexcenter" alt="\[V_{\text{sphere}}=\dfrac{4(\sqrt{r})^{3}\pi}{3}\]" width="139" height="39" />
so we have:
<img src="//latex.artofproblemsolving.com/2/3/d/23da0dd79a5b1c17a4012126e13a3723da1778f6.png" class="latexcenter" alt="\[\frac{\pi\cdot2\sqrt{r}}{3}(r^2+r+1)=2\cdot\dfrac{4(\sqrt{r})^{3}\pi}{3}.\]" width="271" height="39" />
Dividing by <img src="//latex.artofproblemsolving.com/3/0/9/3095d616de3b74af707f828ec376a4a4e765be6e.png" class="latex" alt="$\frac{2\pi\sqrt{r}}{3}$" style="vertical-align: -12px" width="46" height="39" />, we get
<img src="//latex.artofproblemsolving.com/e/f/f/eff48a7e2a483ec769c53595e6616aae042ab70e.png" class="latexcenter" alt="\[r^2+r+1=4r\]" width="119" height="16" />
which is equivalent to <img src="//latex.artofproblemsolving.com/d/b/2/db29bc3b20d4947107253c8633c1e77dec9c4443.png" class="latexcenter" alt="\[r^2-3r+1=0\]" width="120" height="16" />
by the Quadratic Formula, <img src="//latex.artofproblemsolving.com/2/3/f/23f05cd0c8ea696585fd057809431e2dfa41cf09.png" class="latex" alt="$r=\dfrac{3\pm\sqrt{(-3)^2-4\cdot1\cdot1}}{2\cdot1}$" style="vertical-align: -13px" width="205" height="41" />
, so
<img src="//latex.artofproblemsolving.com/4/6/3/463b0f30cd0d1f995e8334d04cc9497d32cf8e76.png" class="latexcenter" alt="\[r=\dfrac{3+\sqrt{5}}{2} \longrightarrow \boxed{\textbf{(E)}}\]" width="170" height="41" />
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>Similar to above, draw a smaller cone top with the base of the smaller circle with radius <img src="//latex.artofproblemsolving.com/a/d/c/adc143c3f887fa7e92fd80d91bb22af51b4abd77.png" class="latex" alt="$r_1$" style="vertical-align: -2px" width="13" height="10" /> and height <img src="//latex.artofproblemsolving.com/8/1/8/8189a5b5a0917b8c93350827be4038af1839139d.png" class="latex" alt="$h$" width="10" height="12" />. The smaller right triangle is similar to the blue highlighted one in Solution <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" />. Then <img src="//latex.artofproblemsolving.com/5/6/2/562142e71e698edbed4d919fcb0e8ed082abbdc3.png" class="latex" alt="$\frac{r_1}{r_2-r_1}=\frac{h}{2R}$" style="vertical-align: -15px" width="107" height="40" /> where <img src="//latex.artofproblemsolving.com/e/f/f/eff43e84f8a3bcf7b6965f0a3248bc4d3a9d0cd4.png" class="latex" alt="$R$" width="14" height="12" /> is the radius of the sphere. Then <img src="//latex.artofproblemsolving.com/1/a/c/1ac0a49aaa78157bf97b0767097ce090e900afc0.png" class="latex" alt="$h=\frac{2Rr_1}{r_2-r_1}$" style="vertical-align: -15px" width="90" height="40" />.
</p><p>From the Pythagorean theorem on the blue triangle in Solution <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" />, we get similarly that <img src="//latex.artofproblemsolving.com/f/8/2/f82016ef4ffdac94f4f0c22c31e35fb6bebace2e.png" class="latex" alt="$R=\sqrt{r_2r_1}$" style="vertical-align: -5px" width="85" height="18" />.
</p><p>From the volume requirements, we get that <img src="//latex.artofproblemsolving.com/e/b/1/eb14e1117fa28249ed42ef8fcd29d16540c281ee.png" class="latex" alt="$\frac{8}{3}\pi R^3=\frac{\pi r_2^2(h+2R)-\pi r_1^2h}{3}$" style="vertical-align: -12px" width="228" height="39" /> which yields <img src="//latex.artofproblemsolving.com/c/2/c/c2cef3031c109b1dcbc78d908759dd05751a45d2.png" class="latex" alt="$8R^3=r_2^2(h+2R)-r_1^2h$" style="vertical-align: -4px" width="189" height="19" />.
</p><p>The small right triangle on top is similar to the big right triangle of the entire big cone. So <img src="//latex.artofproblemsolving.com/1/f/4/1f428a8e7147da69c03bae55ddfe8c6482773dcb.png" class="latex" alt="$\frac{r_1}{r_2}=\frac{h}{h+2R}\implies h+2R=\frac{r_2h}{r_1}$" style="vertical-align: -15px" width="261" height="40" />. 
</p><p>Substituting yields <img src="//latex.artofproblemsolving.com/2/7/5/275105e78e498f69a3625b9f385ee14e53782d14.png" class="latex" alt="$8R^3=\frac{h(r_2^3-r_1^3)}{r_1}$" style="vertical-align: -15px" width="136" height="41" />. 
</p><p>Substituting <img src="//latex.artofproblemsolving.com/f/8/2/f82016ef4ffdac94f4f0c22c31e35fb6bebace2e.png" class="latex" alt="$R=\sqrt{r_2r_1}$" style="vertical-align: -5px" width="85" height="18" /> and <img src="//latex.artofproblemsolving.com/1/a/c/1ac0a49aaa78157bf97b0767097ce090e900afc0.png" class="latex" alt="$h=\frac{2Rr_1}{r_2-r_1}$" style="vertical-align: -15px" width="90" height="40" /> yields <img src="//latex.artofproblemsolving.com/c/e/9/ce985a8e5abfe2e467f37fc9d38ca2b207de5524.png" class="latex" alt="$8(r_2r_1)^{\frac{3}{2}}=2(r_2r_1)^{\frac{1}{2}}(r_2^2+r_2r_1+r_1^2)$" style="vertical-align: -4px" width="275" height="22" /> which yields <img src="//latex.artofproblemsolving.com/8/2/0/8206c070be522cdc9a57e6e318f0faa13df01092.png" class="latex" alt="$r_2^2-3r_2r_1+r_1^2=0$" style="vertical-align: -4px" width="150" height="19" />.
</p><p>Solving in <img src="//latex.artofproblemsolving.com/0/b/4/0b4f3f4b698ce35667432041e168f7653533fa7b.png" class="latex" alt="$r_2$" style="vertical-align: -2px" width="14" height="10" /> yields <img src="//latex.artofproblemsolving.com/f/f/f/fff1bac7729785fb01b91b2998b401202f2bc462.png" class="latex" alt="$r_2=\frac{3r_1\pm r_1\sqrt{5}}{2}$" style="vertical-align: -13px" width="133" height="43" /> so <img src="//latex.artofproblemsolving.com/d/6/d/d6db020b392d46c1b97d22e11bedd209cc3af39b.png" class="latex" alt="$\frac{r_2}{r_1}=\frac{3+\sqrt{5}}{2}$" style="vertical-align: -15px" width="102" height="43" />.
</p>
<h2><span class="mw-headline" id="Solution_3:_Another_Way_of_Simplifying_the_Equation_in_Solution_2">Solution 3: Another Way of Simplifying the Equation in Solution 2</span></h2>
<p>Similar to the previous solutions, we let <img src="//latex.artofproblemsolving.com/e/f/f/eff43e84f8a3bcf7b6965f0a3248bc4d3a9d0cd4.png" class="latex" alt="$R$" width="14" height="12" />, <img src="//latex.artofproblemsolving.com/b/5/5/b55ca7a0aa88ab7d58f4fc035317fdac39b17861.png" class="latex" alt="$r$" width="8" height="8" />, and <img src="//latex.artofproblemsolving.com/f/3/7/f37bba504894945c07a32f5496d74299a37aa51c.png" class="latex" alt="$s$" width="8" height="8" /> be the radii of the larger base, smaller base, and the sphere, respectively. By the Pythagorean Theorem, we have <img src="//latex.artofproblemsolving.com/3/3/0/3307a5303445f3ca34578290f66a17f55a01209b.png" class="latex" alt="$s=\sqrt{Rr}$" style="vertical-align: -1px" width="71" height="18" />. Let <img src="//latex.artofproblemsolving.com/b/1/9/b1902d279ba37d60bdce4e0e987b7cd19d48974e.png" class="latex" alt="$H$" width="16" height="12" />, <img src="//latex.artofproblemsolving.com/8/1/8/8189a5b5a0917b8c93350827be4038af1839139d.png" class="latex" alt="$h$" width="10" height="12" /> be the heights of the large and small cones, respectively, then we have <img src="//latex.artofproblemsolving.com/b/8/9/b894d7c5e742ac7aeca115172c9c7509b536c1ac.png" class="latex" alt="$H=h+2s$" style="vertical-align: -1px" width="90" height="14" /> and
<img src="//latex.artofproblemsolving.com/2/9/0/29032063feade5565b0327ccf124236cb78c9173.png" class="latexcenter" alt="\[h=2s\left(\frac{r}{R-r}\right)==\frac{2sr}{R-r}=\frac{2\left(\sqrt{Rr}\right)r}{R-r}\]" width="326" height="56" /> 
Thus we have the equation 
<img src="//latex.artofproblemsolving.com/a/3/5/a3593a0984c377f7c3bf20d44dee6a78cf9918af.png" class="latexcenter" alt="\[V_{large cone}-V_{small cone}=2V_{sphere}\]" width="246" height="17" />
<img src="//latex.artofproblemsolving.com/0/7/5/075e62d591e66bcdb2c457901d2960651c517a41.png" class="latexcenter" alt="\[\frac{\pi R^2H}{3}-\frac{\pi r^2h}{3}=2\left(\frac{4\pi s^3}{3}\right)\]" width="214" height="43" />
<img src="//latex.artofproblemsolving.com/c/a/9/ca9a57a1a42a61965acff51c373d488f9c4b4329.png" class="latexcenter" alt="\[\pi R^2H-\pi r^2h=8\pi s^3\]" width="167" height="15" />
<img src="//latex.artofproblemsolving.com/b/2/9/b29fa217ae7a5628222734a6805e9d88fd56fec8.png" class="latexcenter" alt="\[R^2H-r^2h=8s^3\]" width="134" height="15" />
Substituting the expressions for <img src="//latex.artofproblemsolving.com/b/1/9/b1902d279ba37d60bdce4e0e987b7cd19d48974e.png" class="latex" alt="$H$" width="16" height="12" />, <img src="//latex.artofproblemsolving.com/8/1/8/8189a5b5a0917b8c93350827be4038af1839139d.png" class="latex" alt="$h$" width="10" height="12" />, <img src="//latex.artofproblemsolving.com/f/3/7/f37bba504894945c07a32f5496d74299a37aa51c.png" class="latex" alt="$s$" width="8" height="8" /> in order into the equation yields
<img src="//latex.artofproblemsolving.com/4/9/2/49266a7e0792f6eb6a0c9efce1cbd4be344709ae.png" class="latexcenter" alt="\[R^2\left(h+2s\right)+r^2h=8s^3\]" width="185" height="20" />
<img src="//latex.artofproblemsolving.com/0/1/e/01e5f4fa70267902d48ce9e18948bd5dde71098d.png" class="latexcenter" alt="\[R^2\left(\left(\frac{2sr}{R-r}\right)+2s\right)+r^2\left(\frac{2sr}{R-r}\right)=8s^3\]" width="329" height="43" />
<img src="//latex.artofproblemsolving.com/f/3/a/f3ae0419a037bea36ac67ce7a53570ed6c2712fd.png" class="latexcenter" alt="\[R^2\left(\frac{2sR}{R-r}\right)+r^2\left(\frac{2sr}{R-r}\right)=8s^3\]" width="263" height="43" />
<img src="//latex.artofproblemsolving.com/9/0/3/90362fd9d27903670419486ea6ee4356d05823c1.png" class="latexcenter" alt="\[\frac{2sR^3-2sr^3}{R-r}=8s^3\]" width="144" height="39" />
<img src="//latex.artofproblemsolving.com/2/d/e/2de22b6c7b6f434a3d86104458359c6660ef62a0.png" class="latexcenter" alt="\[\frac{R^3-r^3}{R-r}=4s^2\]" width="109" height="39" />
The numerator <img src="//latex.artofproblemsolving.com/3/1/3/313c0c946d76566a0313808f7959ca429051c1a0.png" class="latex" alt="$R^3-r^3$" width="58" height="15" /> can be factored to yield
<img src="//latex.artofproblemsolving.com/f/f/0/ff0088d02884e364d6e685db2f495aaa0c0fd36e.png" class="latexcenter" alt="\[\frac{\left(R-r\right)\left(R^2+Rr+r^2\right)}{R-r}=4s^2\]" width="230" height="39" />
<img src="//latex.artofproblemsolving.com/b/5/d/b5dabe94ab92df3078036b40f110ef508c27cdd0.png" class="latexcenter" alt="\[R^2+Rr+r^2{R-r}=4s^2\]" width="197" height="16" />
<img src="//latex.artofproblemsolving.com/a/4/a/a4ace363ef2319cae02c99c9d914835806a9ed6e.png" class="latexcenter" alt="\[R^2+Rr+r^2=4\left(\sqrt{Rr}\right)^2\]" width="205" height="35" />
<img src="//latex.artofproblemsolving.com/7/0/0/700063ae1557a52c1eabdbc056bf694052674a20.png" class="latexcenter" alt="\[R^2+Rr+r^2=4Rr\]" width="159" height="16" />
<img src="//latex.artofproblemsolving.com/b/f/8/bf85a15135abbf313881d3f33a6ef707bdd41c6b.png" class="latexcenter" alt="\[R^2-3Rr+r^2=0\]" width="146" height="16" />
Solving the quadratic to obtain the ratio <img src="//latex.artofproblemsolving.com/7/c/7/7c772bc9a6f786072666799d191448033c175a8d.png" class="latex" alt="$\frac{R}{r}=\frac{3+\sqrt{5}}{2}$" style="vertical-align: -12px" width="100" height="41" /> <img src="//latex.artofproblemsolving.com/6/b/f/6bfc0829f3f29bbf35d0a861644e00cfef0f50ca.png" class="latex" alt="$\boxed{\textbf{(E)}}$" style="vertical-align: -10px" width="39" height="28" />
</p><p>~ Nafer
