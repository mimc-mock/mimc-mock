<p>A sphere is inscribed in a truncated right circular cone as shown. The volume of the truncated cone is twice that of the sphere. What is the ratio of the radius of the bottom base of the truncated cone to the radius of the top base of the truncated cone?
</p><p><img src="//latex.artofproblemsolving.com/d/c/a/dca3b808f0caf802ec275e44346fe25e3e4792f1.png" class="latexcenter" alt="[asy] real r=(3+sqrt(5))/2; real s=sqrt(r); real Brad=r; real brad=1; real Fht = 2*s; import graph3; import solids; currentprojection=orthographic(1,0,.2); currentlight=(10,10,5); revolution sph=sphere((0,0,Fht/2),Fht/2); //draw(surface(sph),green+white+opacity(0.5)); //triple f(pair t) {return (t.x*cos(t.y),t.x*sin(t.y),t.x^(1/n)*sin(t.y/n));} triple f(pair t) { triple v0 = Brad*(cos(t.x),sin(t.x),0); triple v1 = brad*(cos(t.x),sin(t.x),0)+(0,0,Fht); return (v0 + t.y*(v1-v0)); } triple g(pair t) { return (t.y*cos(t.x),t.y*sin(t.x),0); } surface sback=surface(f,(3pi/4,0),(7pi/4,1),80,2); surface sfront=surface(f,(7pi/4,0),(11pi/4,1),80,2); surface base = surface(g,(0,0),(2pi,Brad),80,2); draw(sback,gray(0.9)); draw(sfront,gray(0.5)); draw(base,gray(0.9)); draw(surface(sph),gray(0.4)); [/asy]" width="252" height="188" />
</p><p><img src="//latex.artofproblemsolving.com/6/e/d/6ed55192447af710c595ba632ef42c8a98251f94.png" class="latex" alt="$\text{(A) } \dfrac32 \quad \text{(B) } \dfrac{1+\sqrt5}2 \quad \text{(C) } \sqrt3 \quad \text{(D) } 2 \quad \text{(E) } \dfrac{3+\sqrt5}2$" style="vertical-align: -12px" width="383" height="41" />
</p>
<h2><span class="mw-headline" id="Solutions">Solutions</span></h2>
<h3><span class="mw-headline" id="Solution_1">Solution 1</span></h3>
<p>First, we draw the vertical cross-section passing through the middle of the frustum.
let the top base equal 2 and the bottom base to be equal to 2r
<img src="//latex.artofproblemsolving.com/0/1/6/016c1f2f2e25e86be1364ec138a943e0955b8e6d.png" class="latexcenter" alt="[asy] size(7cm); pair A,B,C,D; real r = (3+sqrt(5))/2; real s = sqrt(r); A = (-r,0); B = (r,0); C = (1,2*s); D = (-1,2*s); draw(A--B--C--D--cycle); pair O = (0,s); draw(shift(O)*scale(s)*unitcircle); dot(O); pair X,Y; X = (0,0); Y = (0,2*s); draw(X--Y); label(&quot;$r-1$&quot;,(X+B)/2,S); label(&quot;$1$&quot;,(Y+C)/2,N); label(&quot;$s$&quot;,(O+Y)/2,W); label(&quot;$s$&quot;,(O+X)/2,W); draw(B--C--(1,0)--cycle,blue+1bp); pair P = 0.73*C+0.27*B; draw(O--P); dot(P); label(&quot;$1$&quot;,(C+P)/2,NE); label(&quot;$r$&quot;,(B+P)/2,NE); [/asy]" width="332" height="248" />
</p><p>then using the Pythagorean theorem we have:
<img src="//latex.artofproblemsolving.com/e/a/9/ea93cc1977198893b2a5a61564364d7877dab0e4.png" class="latex" alt="$(r+1)^2=(2s)^2+(r-1)^2$" style="vertical-align: -4px" width="207" height="19" />
which is equivalent to:
<img src="//latex.artofproblemsolving.com/7/7/a/77ab66312f957f5bad1c74f1295b0021c8e0bc7f.png" class="latex" alt="$r^2+2r+1=4s^2+r^2-2r+1$" style="vertical-align: -1px" width="245" height="16" />
subtracting <img src="//latex.artofproblemsolving.com/f/2/4/f2443c3a311121b07c45b465114d975d2463f30e.png" class="latex" alt="$r^2-2r+1$" style="vertical-align: -1px" width="86" height="16" /> from both sides
<img src="//latex.artofproblemsolving.com/f/8/a/f8ae0616ddb27d77b602943da58c40543421cfe9.png" class="latex" alt="$4r=4s^2$" style="vertical-align: 0px" width="65" height="15" />
solving for s we get:
<img src="//latex.artofproblemsolving.com/7/6/1/761f734737f3d130b9c1f3714b65c6062426b51c.png" class="latexcenter" alt="\[s=\sqrt{r}\]" width="56" height="18" />
next we can find the volume of the frustum and of the sphere and we know <img src="//latex.artofproblemsolving.com/c/b/a/cba5581ebf308590a2936a23565e30160a3beb55.png" class="latex" alt="$V_{\text{frustum}}=2V_{\text{sphere}}$" style="vertical-align: -5px" width="137" height="17" /> so we can solve for <img src="//latex.artofproblemsolving.com/f/3/7/f37bba504894945c07a32f5496d74299a37aa51c.png" class="latex" alt="$s$" width="8" height="8" />
using <img src="//latex.artofproblemsolving.com/2/a/f/2afe51f13641fc794ffe4f6025bf85c9e49f41ab.png" class="latex" alt="$V_{\text{frustum}}=\frac{\pi*h}{3}(R^2+r^2+Rr)$" style="vertical-align: -12px" width="241" height="37" />
we get:
<img src="//latex.artofproblemsolving.com/d/7/e/d7eef8d7362d698016a192e684324b236ecda676.png" class="latexcenter" alt="\[V_{\text{frustum}}=\frac{\pi*2\sqrt{r}}{3}(r^2+r+1)\]" width="236" height="39" />
using  <img src="//latex.artofproblemsolving.com/8/e/e/8eee5c6ae8433f258f3c15dcb0d6113ff3c31d05.png" class="latex" alt="$V_{\text{sphere}}=\dfrac{4r^{3}\pi}{3}$" style="vertical-align: -12px" width="111" height="39" />
we get 
<img src="//latex.artofproblemsolving.com/8/2/d/82da68e37e5b3401b8de5a8588b92760176358cd.png" class="latexcenter" alt="\[V_{\text{sphere}}=\dfrac{4(\sqrt{r})^{3}\pi}{3}\]" width="139" height="39" />
so we have:
<img src="//latex.artofproblemsolving.com/7/3/1/731ee8c7ba48dd60c6b5cba6156967cd2d371542.png" class="latexcenter" alt="\[\frac{\pi*2\sqrt{r}}{3}(r^2+r+1)=2*\dfrac{4(\sqrt{r})^{3}\pi}{3}\]" width="274" height="39" />
dividing by <img src="//latex.artofproblemsolving.com/2/9/2/29253aff60588ed9f4ad7d05054cd379e6ab3cf6.png" class="latex" alt="$\frac{2\pi*\sqrt{r}}{3}$" style="vertical-align: -12px" width="63" height="39" />
we get <img src="//latex.artofproblemsolving.com/e/f/f/eff48a7e2a483ec769c53595e6616aae042ab70e.png" class="latexcenter" alt="\[r^2+r+1=4r\]" width="119" height="16" />
which is equivalent to <img src="//latex.artofproblemsolving.com/d/b/2/db29bc3b20d4947107253c8633c1e77dec9c4443.png" class="latexcenter" alt="\[r^2-3r+1=0\]" width="120" height="16" />
<img src="//latex.artofproblemsolving.com/a/9/f/a9fc6b6929ee1fb1d4ef525cd209e1c882e19d5d.png" class="latex" alt="$r=\dfrac{3\pm\sqrt{(-3)^2-4*1*1}}{2*1}$" style="vertical-align: -13px" width="213" height="41" />
so <img src="//latex.artofproblemsolving.com/d/4/3/d4392ed60ab25c1ad1394398225bf6abdf959a50.png" class="latexcenter" alt="\[r=\dfrac{3+\sqrt{5}}{2}\longrightarrow \boxed{E}\]" width="157" height="41" />
</p>
<h3><span id="Solution_2(ADD_DIAGRAM)"></span><span class="mw-headline" id="Solution_2.28ADD_DIAGRAM.29">Solution 2(ADD DIAGRAM)</span></h3>
<p>Let's once again look at the cross section of the frustum. Let the angle from the center of the sphere to a point on the circumference of the bottom circle be <img src="//latex.artofproblemsolving.com/7/9/6/79607326260cf94d825a16e94c1c1b1a8af24db6.png" class="latex" alt="$\theta.$" width="12" height="13" /> This implies that the angle from the center of the sphere to a point on the circumference of the top circle is <img src="//latex.artofproblemsolving.com/2/9/b/29bd79e07a236a1a45f73de0ff53312359579f72.png" class="latex" alt="$90 - \theta.$" width="53" height="13" /> Hence the bottom radius is <img src="//latex.artofproblemsolving.com/3/f/5/3f53f6fc52bec8bf977f45df418f89f6747334d3.png" class="latex" alt="$r\tan{\theta}$" width="49" height="13" /> and the top radius is <img src="//latex.artofproblemsolving.com/5/9/b/59b8a07207a04e4b881cc78e8bc2ad9b0805f04e.png" class="latex" alt="$\frac {r}{\tan {\theta}}.$" style="vertical-align: -12px" width="46" height="33" /> This means that the radio between the bottom radius and top radius is <img src="//latex.artofproblemsolving.com/1/6/a/16a2168826620f75cf9d08bfd4a5e4b8b96fc172.png" class="latex" alt="$(\tan {\theta})^2.$" style="vertical-align: -5px" width="67" height="21" /> Using the frustum volume formula, we find that the are of this figure is <img src="//latex.artofproblemsolving.com/a/0/b/a0b84b0c7a50614b76714d50f66b76281d621010.png" class="latex" alt="$\frac{2\pi r}{3}(r^2(\tan {\theta})^2 + r^2 + \frac {r^2} {(\tan {\theta})^2}).$" style="vertical-align: -17px" width="251" height="43" /> We can equate this to <img src="//latex.artofproblemsolving.com/3/0/2/302472866b79d973760418e911c8401b40ae4693.png" class="latex" alt="$\frac {8\pi*r^3} 3.$" style="vertical-align: -12px" width="61" height="39" /> Simplifying, we are left with a quadratic conveniently in <img src="//latex.artofproblemsolving.com/1/6/a/16a2168826620f75cf9d08bfd4a5e4b8b96fc172.png" class="latex" alt="$(\tan {\theta})^2.$" style="vertical-align: -5px" width="67" height="21" /> The quadratic is <img src="//latex.artofproblemsolving.com/6/2/5/6250aadb0ac7c3c606b30c81def132dacb0f8e75.png" class="latex" alt="$(\tan {\theta})^4 - 3(\tan {\theta})^2 + 1 = 0.$" style="vertical-align: -4px" width="219" height="19" /> This gives us <img src="//latex.artofproblemsolving.com/c/0/0/c00d6086cd43bb66eb1a59988886a588a25039ac.png" class="latex" alt="$(\tan {\theta})^2 = \dfrac{3+\sqrt{5}}{2}\longrightarrow \boxed{E}$" style="vertical-align: -12px" width="209" height="41" />
</p><p>~NeeNeeMath
</p>
