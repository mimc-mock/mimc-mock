<p>Each square in a <img src="//latex.artofproblemsolving.com/1/9/7/197151a067f49bd4daa1952d1a4fac322409f0a6.png" class="latex" alt="$5 \times 5$" width="40" height="12" /> grid is either filled or empty, and has up to eight adjacent neighboring squares, where neighboring squares share either a side or a corner. The grid is transformed by the following rules:
</p>
<ul><li>Any filled square with two or three filled neighbors remains filled.</li></ul>
<ul><li>Any empty square with exactly three filled neighbors becomes a filled square.</li></ul>
<ul><li>All other squares remain empty or become empty.</li></ul>
<p>A sample transformation is shown in the figure below.
<img src="//latex.artofproblemsolving.com/0/0/3/00333423a7ddd483233c145023df02bdb469a3b9.png" class="latexcenter" alt="[asy]         import geometry;         unitsize(0.6cm);          void ds(pair x) {             filldraw(x -- (1,0) + x -- (1,1) + x -- (0,1)+x -- cycle,mediumgray,invisible);         }          ds((1,1));         ds((2,1));         ds((3,1));         ds((1,3));          for (int i = 0; i &lt;= 5; ++i) {             draw((0,i)--(5,i));             draw((i,0)--(i,5));         }          label(&quot;Initial&quot;, (2.5,-1));         draw((6,2.5)--(8,2.5),Arrow);          ds((10,2));         ds((11,1));         ds((11,0));          for (int i = 0; i &lt;= 5; ++i) {             draw((9,i)--(14,i));             draw((i+9,0)--(i+9,5));         }          label(&quot;Transformed&quot;, (11.5,-1)); [/asy]" width="398" height="182" />
Suppose the <img src="//latex.artofproblemsolving.com/1/9/7/197151a067f49bd4daa1952d1a4fac322409f0a6.png" class="latex" alt="$5 \times 5$" width="40" height="12" /> grid has a border of empty squares surrounding a <img src="//latex.artofproblemsolving.com/d/2/7/d278803d72fcc493afb1559174a483aa7f41d143.png" class="latex" alt="$3 \times 3$" width="40" height="12" /> subgrid. How many initial configurations will lead to a transformed grid consisting of a single filled square in the center after a single transformation? (Rotations and reflections of the same configuration are considered different.)
<img src="//latex.artofproblemsolving.com/2/5/3/253c2691ed603d5d600e2b1becee941d5268c900.png" class="latexcenter" alt="[asy]         import geometry;         unitsize(0.6cm);          void ds(pair x) {             filldraw(x -- (1,0) + x -- (1,1) + x -- (0,1)+x -- cycle,mediumgray,invisible);         }          for (int i = 1; i &lt; 4; ++ i) {             for (int j = 1; j &lt; 4; ++j) {                 label(&quot;?&quot;,(i + 0.5, j + 0.5));             }         }          for (int i = 0; i &lt;= 5; ++i) {             draw((0,i)--(5,i));             draw((i,0)--(i,5));         }          label(&quot;Initial&quot;, (2.5,-1));         draw((6,2.5)--(8,2.5),Arrow);          ds((11,2));          for (int i = 0; i &lt;= 5; ++i) {             draw((9,i)--(14,i));             draw((i+9,0)--(i+9,5));         }          label(&quot;Transformed&quot;, (11.5,-1)); [/asy]" width="398" height="182" />
<img src="//latex.artofproblemsolving.com/c/c/0/cc04fa4d612e3cb256e4e121720b3fbfeb920e61.png" class="latex" alt="$\textbf{(A)}\ 14 \qquad\textbf{(B)}\ 18 \qquad\textbf{(C)}\ 22 \qquad\textbf{(D)}\ 26 \qquad\textbf{(E)}\ 30$" style="vertical-align: -5px" width="397" height="18" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>There are two cases for the initial configuration:
</p>
<ol style="margin-left: 1.5em;">
  <li>The center square is filled. </li><p>
Exactly two of the eight adjacent neighboring squares of the center are filled. Clearly, the only possibility is that the squares along one diagonal are filled, as shown below:</p><p>
<img src="//latex.artofproblemsolving.com/8/d/9/8d9ca05cb188cd4c1903e7c6ced4882797e1db84.png" class="latexcenter" alt="[asy] import geometry; unitsize(0.6cm);  void ds(pair x) { 	filldraw(x -- (1,0) + x -- (1,1) + x -- (0,1)+x -- cycle,mediumgray,invisible); }  ds((1,3)); ds((2,2)); ds((3,1));  for (int i = 0; i &lt;= 5; ++i) { 	draw((0,i)--(5,i));     draw((i,0)--(i,5));     }  label(&quot;$2$ Configurations&quot;, (2.5,-1)); [/asy]" width="148" height="185" />
In this case, there are <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" /> possible initial configurations. All rotations and reflections are considered.</p><p class="mw-empty-elt"> 
  </p><li>The center square is empty.</li><p>
Exactly three of the eight adjacent neighboring squares of the center are filled. The possibilities are shown below:</p><p>
<img src="//latex.artofproblemsolving.com/9/a/f/9af0bc0ac9bf851c3078feeda3696d8c643a3b6d.png" class="latexcenter" alt="[asy] import geometry; unitsize(0.6cm);  void ds(pair x) { 	filldraw(x -- (1,0) + x -- (1,1) + x -- (0,1)+x -- cycle,mediumgray,invisible); }  ds((1,3)); ds((3,3)); ds((1,1));  for (int i = 0; i &lt;= 5; ++i) { 	draw((0,i)--(5,i));     draw((i,0)--(i,5));     }  ds((10,3)); ds((12,3)); ds((11,1));  for (int i = 0; i &lt;= 5; ++i) {     draw((9,i)--(14,i));     draw((i+9,0)--(i+9,5));     }  ds((19,3)); ds((20,1)); ds((21,2));  for (int i = 0; i &lt;= 5; ++i) {     draw((18,i)--(23,i));     draw((i+18,0)--(i+18,5));     }  ds((28,3)); ds((29,1)); ds((30,1));  for (int i = 0; i &lt;= 5; ++i) {     draw((27,i)--(32,i));     draw((i+27,0)--(i+27,5));     }      label(&quot;$4$ Configurations&quot;, (2.5,-1)); label(&quot;$4$ Configurations&quot;, (11.5,-1)); label(&quot;$4$ Configurations&quot;, (20.5,-1)); label(&quot;$8$ Configurations&quot;, (29.5,-1)); [/asy]" width="912" height="185" />
In this case, there are <img src="//latex.artofproblemsolving.com/7/a/6/7a6309944db1b047f8490d83eae974a2713d8177.png" class="latex" alt="$4+4+4+8=20$" style="vertical-align: -2px" width="147" height="15" /> possible initial configurations. All rotations and reflections are considered.</p><p class="mw-empty-elt">
</p></ol>
<p>Together, the answer is <img src="//latex.artofproblemsolving.com/e/0/6/e065c8e443f05a356b11631f3e8dcc57f8b64704.png" class="latex" alt="$2+20=\boxed{\textbf{(C)}\ 22}.$" style="vertical-align: -11px" width="144" height="30" />
</p><p>~mathboy100 ~MRENTHUSIASM
</p>
