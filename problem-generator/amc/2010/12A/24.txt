<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>The question asks for the number of disjoint open intervals, which means we need to find the number of disjoint intervals such that the function is defined within them.
</p><p>We note that since all of the <img src="//latex.artofproblemsolving.com/6/f/b/6fb300dacfcde1b57ad070409262b1743e0cd966.png" class="latex" alt="$\sin$" width="22" height="12" /> factors are inside a logarithm, the function is undefined where the inside of the logarithm is less than or equal to <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" />.
</p><p>First, let us find the number of zeros of the inside of the logarithm.
</p><p><img src="//latex.artofproblemsolving.com/9/b/4/9b4632ec77ebe12ddb444ccfcbb70ec384edb137.png" class="latexcenter" alt="\begin{align*}\sin(\pi x) \cdot \sin(2 \pi x) \cdot \sin (3 \pi x) \cdots \sin(8 \pi x) &amp;= 0\\ \sin(\pi x) &amp;= 0\\ x &amp;= 0, 1\\ \sin(2 \pi x) &amp;= 0\\ x &amp;= 0, \frac{1}{2}, 1\\ \sin(3 \pi x) &amp;= 0\\ x &amp;= 0, \frac{1}{3}, \frac{2}{3}, 1\\ \sin(4 \pi x) &amp;= 0\\ x &amp;= 0, \frac{1}{4}, \frac{2}{4}, \frac{3}{4}, 1\\ &amp;\cdots\end{align*}" width="424" height="294" />
</p><p>After counting up the number of zeros for each factor and eliminating the excess cases we get <img src="//latex.artofproblemsolving.com/9/a/1/9a1b2497e7b125cd721bab4c6e2385f954fe35cb.png" class="latex" alt="$23$" width="17" height="12" /> zeros and <img src="//latex.artofproblemsolving.com/1/d/1/1d111a5e00ee5ea6700bb629994dc629874c505b.png" class="latex" alt="$22$" width="17" height="12" /> intervals.
</p><p>In order to find which intervals are negative, we must first realize that at every zero of each factor, the sign changes. We also have to be careful, as some zeros are doubled, or even tripled, quadrupled, etc.
</p><p>The first interval <img src="//latex.artofproblemsolving.com/2/9/d/29d593aab4bc6a1861db505b3a048bb79a1a5ab0.png" class="latex" alt="$\left(0, \frac{1}{8}\right)$" style="vertical-align: -18px" width="58" height="45" /> is obviously positive. This means the next interval <img src="//latex.artofproblemsolving.com/b/4/d/b4dea4da8e7ae036ebe2aedbe3ab17746510d79b.png" class="latex" alt="$\left(\frac{1}{8}, \frac{1}{7}\right)$" style="vertical-align: -18px" width="62" height="45" /> is negative. Continuing the pattern and accounting for doubled roots (which do not flip sign), we realize that there are <img src="//latex.artofproblemsolving.com/7/9/0/79069377f91364c2f87a64e5f9f562a091c8a6c1.png" class="latex" alt="$5$" width="8" height="12" /> negative intervals from <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" /> to <img src="//latex.artofproblemsolving.com/2/f/9/2f960094315d60883495f9b74148e17487ee9584.png" class="latex" alt="$\frac{1}{2}$" style="vertical-align: -12px" width="11" height="37" />. Since the function is symmetric, we know that there are also <img src="//latex.artofproblemsolving.com/7/9/0/79069377f91364c2f87a64e5f9f562a091c8a6c1.png" class="latex" alt="$5$" width="8" height="12" /> negative intervals from <img src="//latex.artofproblemsolving.com/2/f/9/2f960094315d60883495f9b74148e17487ee9584.png" class="latex" alt="$\frac{1}{2}$" style="vertical-align: -12px" width="11" height="37" /> to <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" />.
</p><p>And so, the total number of disjoint open intervals is <img src="//latex.artofproblemsolving.com/a/3/9/a390f62151275c7a63a056f4f8f2b7f75c2e89ce.png" class="latex" alt="$22 - 2\cdot{5} = \boxed{12\ \textbf{(B)}}$" style="vertical-align: -10px" width="159" height="28" />
</p>
