<p>The set of all real numbers <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> for which
</p><p><img src="//latex.artofproblemsolving.com/e/f/a/efaf7382f309efe8361302a7c57315c0e92e4adb.png" class="latexcenter" alt="\[\log_{2004}(\log_{2003}(\log_{2002}(\log_{2001}{x})))\]" width="257" height="18" />
</p><p>is defined is <img src="//latex.artofproblemsolving.com/0/f/8/0f8cf718eeeda414b47579ea618b460e5a38cf5f.png" class="latex" alt="$\{x\mid x &gt; c\}$" style="vertical-align: -5px" width="88" height="20" />. What is the value of <img src="//latex.artofproblemsolving.com/3/3/7/3372c1cb6d68cf97c2d231acc0b47b95a9ed04cc.png" class="latex" alt="$c$" width="8" height="8" />?
</p><p><img src="//latex.artofproblemsolving.com/b/4/2/b421f3f99b7951e75eda6fae877ba772b69a6160.png" class="latex" alt="$\textbf {(A) } 0\qquad \textbf {(B) }2001^{2002} \qquad \textbf {(C) }2002^{2003} \qquad \textbf {(D) }2003^{2004} \qquad \textbf {(E) }2001^{2002^{2003}}$" style="vertical-align: -5px" width="597" height="24" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>For all real numbers <img src="//latex.artofproblemsolving.com/f/3/b/f3b28547efd22be1c4a05fa193cd7038fc65d71b.png" class="latex" alt="$a,b,$" style="vertical-align: -3px" width="29" height="16" /> and <img src="//latex.artofproblemsolving.com/3/3/7/3372c1cb6d68cf97c2d231acc0b47b95a9ed04cc.png" class="latex" alt="$c$" width="8" height="8" /> such that <img src="//latex.artofproblemsolving.com/2/2/1/2210e1c13767dc4524e3c1415bacd736c6a76317.png" class="latex" alt="$b&gt;1,$" style="vertical-align: -3px" width="45" height="16" /> note that:
</p>
<ol style="margin-left: 1.5em;">
  <li><img src="//latex.artofproblemsolving.com/2/a/b/2aba777fd992fd3f377364f9ee87b24eb98e89b2.png" class="latex" alt="$\log_b a$" style="vertical-align: -4px" width="42" height="17" /> is defined if and only if <img src="//latex.artofproblemsolving.com/3/4/7/3470668bb8a39eebcb9ed22863d4c71245731ed9.png" class="latex" alt="$a&gt;0.$" style="vertical-align: 0px" width="46" height="13" /></li><p class="mw-empty-elt">
  </p><li><img src="//latex.artofproblemsolving.com/8/c/f/8cfcf23e7e527cec27f683a1adeb3c40096677ec.png" class="latex" alt="$\log_b a&gt;c$" style="vertical-align: -5px" width="77" height="19" /> if and only if <img src="//latex.artofproblemsolving.com/2/b/8/2b8cb101d61e55686abe4187d5a33ba9de1b2943.png" class="latex" alt="$a&gt;b^c.$" style="vertical-align: -2px" width="55" height="15" /></li><p class="mw-empty-elt">
</p></ol>
<p>Therefore, we have
<img src="//latex.artofproblemsolving.com/f/0/4/f04993e6f16dda48f411ac332c7a79c9732f07fb.png" class="latexcenter" alt="\begin{align*} \log_{2004}(\log_{2003}(\log_{2002}(\log_{2001}{x}))) \text{ is defined} &amp;\implies \log_{2003}(\log_{2002}(\log_{2001}{x}))&gt;0 \\ &amp;\implies \log_{2002}(\log_{2001}{x})&gt;1 \\ &amp;\implies \log_{2001}{x}&gt;2002 \\ &amp;\implies x&gt;2001^{2002}, \end{align*}" width="608" height="94" />
from which <img src="//latex.artofproblemsolving.com/f/1/0/f108ad9ca3fa9531ab71aacfb115e03803d1c694.png" class="latex" alt="$c=\boxed{\textbf {(B) }2001^{2002}}.$" style="vertical-align: -11px" width="149" height="32" />
</p><p>~Azjps ~MRENTHUSIASM
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>Let
<img src="//latex.artofproblemsolving.com/e/e/9/ee94d03c2e342ff065d2b84da09ce7e8c99e8180.png" class="latexcenter" alt="\begin{align*} x &amp;= 2001^a, \\ a &amp;= 2002^b, \\ b &amp;= 2003^c, \\ c &amp;= 2004^d. \end{align*}" width="82" height="94" />
It follows that <img src="//latex.artofproblemsolving.com/1/a/7/1a738e63b7eb591f9d89a7e126bd9bbecca05d85.png" class="latexcenter" alt="\[x = 2001^{2002^{2003^{2004^d}}}.\]" width="155" height="25" />
The smallest value of <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> occurs when <img src="//latex.artofproblemsolving.com/a/b/3/ab35f073b9be6e22f899013a7f67984f25f6cc89.png" class="latex" alt="$d\rightarrow -\infty,$" style="vertical-align: -4px" width="77" height="18" /> so this expression becomes
<img src="//latex.artofproblemsolving.com/2/9/3/293dc68e4ee23b057f7cf9a5c01ed157b2351d72.png" class="latexcenter" alt="\[x = 2001^{2002^{2003^0}} = 2001^{2002^1} = \boxed{\textbf {(B) }2001^{2002}}.\]" width="359" height="31" />
</p>
