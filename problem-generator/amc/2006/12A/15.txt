<p>Suppose <img src="//latex.artofproblemsolving.com/9/5/c/95c6d0453dd9438d52267937c0742af65c5e0ccc.png" class="latex" alt="$\cos x=0$" width="70" height="12" /> and <img src="//latex.artofproblemsolving.com/3/0/2/3026b28b8ab79344a1de2554c78e44bf3be55133.png" class="latex" alt="$\cos (x+z)=1/2$" style="vertical-align: -4px" width="131" height="18" />. What is the smallest possible positive value of <img src="//latex.artofproblemsolving.com/b/1/3/b13f21416d84e13708696f34dea81026cda583c9.png" class="latex" alt="$z$" width="8" height="8" />?
</p><p><img src="//latex.artofproblemsolving.com/d/d/4/dd48cfdb542a30eb0130e6282364ebac5efd8a4d.png" class="latex" alt="$\mathrm{(A) \ } \frac{\pi}{6}\qquad \mathrm{(B) \ } \frac{\pi}{3}\qquad \mathrm{(C) \ } \frac{\pi}{2}\qquad \mathrm{(D) \ } \frac{5\pi}{6} \quad \mathrm{(E) \ } \frac{7\pi}{6}$" style="vertical-align: -12px" width="372" height="37" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<ul><li>For <img src="//latex.artofproblemsolving.com/e/8/b/e8b5e42382dc69750f3a649ca44f9ddcec1ef6c5.png" class="latex" alt="$\cos x = 0$" width="70" height="12" />, x must be in the form of <img src="//latex.artofproblemsolving.com/3/7/3/373b9daf50d39fe915ef332c968a190b29276c0c.png" class="latex" alt="$\frac{\pi}{2} + \pi n$" style="vertical-align: -12px" width="58" height="33" />, where <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> denotes any <a href="/wiki/index.php/Integer" title="Integer">integer</a>.</li>
<li>For <img src="//latex.artofproblemsolving.com/7/6/3/763b92f23f7ba2744eefc7896026f7fd0a75d022.png" class="latex" alt="$\cos (x+z) = 1 / 2$" style="vertical-align: -4px" width="131" height="18" />, <img src="//latex.artofproblemsolving.com/3/7/f/37fedd041985400f4d214e2f0f464d75d7ecec11.png" class="latex" alt="$x + z = \frac{\pi}{3} +2\pi n, \frac{5\pi}{3} + 2\pi n$" style="vertical-align: -12px" width="219" height="37" />.</li></ul>
<p>The smallest possible value of <img src="//latex.artofproblemsolving.com/b/1/3/b13f21416d84e13708696f34dea81026cda583c9.png" class="latex" alt="$z$" width="8" height="8" /> will be that of <img src="//latex.artofproblemsolving.com/5/f/4/5f44352075ec0ab1a02f183eba9a3bf256085852.png" class="latex" alt="$\frac{5\pi}{3} - \frac{3\pi}{2} = \frac{\pi}{6} \Rightarrow \mathrm{(A)}$" style="vertical-align: -12px" width="164" height="37" />.
</p>
