<p>Jacob uses the following procedure to write down a sequence of numbers. First he chooses the first term to be 6. To generate each succeeding term, he flips a fair coin. If it comes up heads, he doubles the previous term and subtracts 1. If it comes up tails, he takes half of the previous term and subtracts 1. What is the probability that the fourth term in Jacob's sequence is an <a href="/wiki/index.php/Integer" title="Integer">integer</a>?
</p><p><img src="//latex.artofproblemsolving.com/4/b/8/4b877c82cebfbaf3560a166c058de2db3ef4d502.png" class="latex" alt="$\mathrm{(A)}\ \frac{1}{6}\qquad\mathrm{(B)}\ \frac{1}{3}\qquad\mathrm{(C)}\ \frac{1}{2}\qquad\mathrm{(D)}\ \frac{5}{8}\qquad\mathrm{(E)}\ \frac{3}{4}$" style="vertical-align: -13px" width="361" height="38" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>We construct a tree showing all possible outcomes that Jacob may get after <img src="//latex.artofproblemsolving.com/7/c/d/7cde695f2e4542fd01f860a89189f47a27143b66.png" class="latex" alt="$3$" width="8" height="12" /> flips; we can do this because there are only <img src="//latex.artofproblemsolving.com/8/4/5/8455f3b5cb3b4880b8c9d782a5c1f0334db819eb.png" class="latex" alt="$8$" width="8" height="12" /> possibilities:
<img src="//latex.artofproblemsolving.com/9/1/1/911a43419f7882cf84ca9d4df37cfe828a993b92.png" class="latexcenter" alt="\[6\quad\begin{cases}  \ \text{H}: 11 &amp;\quad  \begin{cases}  \ \text{H}: 21 &amp;\quad \begin{cases} \ \text{H}: \boxed{41}\\ \ \text{T}: 9.5 \end{cases}\\  \ \text{T}: 4.5 &amp;\quad \begin{cases} \ \text{H}: \boxed{8}\\ \ \text{T}: 1.25 \end{cases}  \end{cases}\\  \ \text{T}: 2 &amp;\quad  \begin{cases}  \ \text{H}: 3 &amp;\qquad\! \begin{cases} \ \text{H}: \boxed{5}\\ \ \text{T}: 0.5 \end{cases}\\  \ \text{T}: 0 &amp;\qquad\! \begin{cases} \ \text{H}: \boxed{-1}\\ \ \text{T}: \boxed{-1} \end{cases} \end{cases} \end{cases}\]" width="316" height="213" />
There is a <img src="//latex.artofproblemsolving.com/f/4/3/f4344e4ce1c2a266b885d9b160544e3036de15f8.png" class="latex" alt="$\frac{5}{8}$" style="vertical-align: -12px" width="11" height="37" /> chance that the fourth term in Jacob's sequence is an integer, so the answer is <img src="//latex.artofproblemsolving.com/b/f/b/bfbf4df7c25681bb178ddc50ee0c0d656aa25b5e.png" class="latex" alt="$\mathrm{(D)}$" style="vertical-align: -4px" width="26" height="18" />.
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>We can see that as long as the last flip is heads, it will be an integer, so there is a <img src="//latex.artofproblemsolving.com/4/a/4/4a4a7df1100efe353cc612d5d1c289c22d5bf8c1.png" class="latex" alt="$\dfrac{1}{2}$" style="vertical-align: -12px" width="11" height="37" /> chance of this happening.
</p><p>Doing a little casework, we see that the only possibility when the last flip is tails is when the third flip brings it to 0. There is a <img src="//latex.artofproblemsolving.com/0/e/4/0e423a59340510ead85744f33de388fd7ac11874.png" class="latex" alt="$\dfrac{1}{8}$" style="vertical-align: -12px" width="11" height="37" /> chance of this happening.
</p><p>Therefore, there is a <img src="//latex.artofproblemsolving.com/f/2/6/f2645861a6cdf4ee7eeb407eebe86e7cc9a98850.png" class="latex" alt="$\dfrac{1}{2}+\dfrac{1}{8}=\boxed{(D)\dfrac{5}{8}}$" style="vertical-align: -19px" width="129" height="49" /> possibility of ending with an integer.
</p><p>~nezha33
</p>
