<p>How many <img src="//latex.artofproblemsolving.com/e/0/a/e0a0db32027a732ac57d37ef2ae9bb150f65b108.png" class="latex" alt="$7$" width="9" height="12" />-digit palindromes (numbers that read the same backward as forward) can be formed using the digits <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" />, <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" />, <img src="//latex.artofproblemsolving.com/7/c/d/7cde695f2e4542fd01f860a89189f47a27143b66.png" class="latex" alt="$3$" width="8" height="12" />, <img src="//latex.artofproblemsolving.com/7/c/d/7cde695f2e4542fd01f860a89189f47a27143b66.png" class="latex" alt="$3$" width="8" height="12" />, <img src="//latex.artofproblemsolving.com/7/9/0/79069377f91364c2f87a64e5f9f562a091c8a6c1.png" class="latex" alt="$5$" width="8" height="12" />, <img src="//latex.artofproblemsolving.com/7/9/0/79069377f91364c2f87a64e5f9f562a091c8a6c1.png" class="latex" alt="$5$" width="8" height="12" />, <img src="//latex.artofproblemsolving.com/7/9/0/79069377f91364c2f87a64e5f9f562a091c8a6c1.png" class="latex" alt="$5$" width="8" height="12" />?
</p><p><img src="//latex.artofproblemsolving.com/2/5/e/25e1fa94f1ed557cbcad42e8a623e0d7bd8f214e.png" class="latex" alt="$\text{(A) } 6 \qquad \text{(B) } 12 \qquad \text{(C) } 24 \qquad \text{(D) } 36 \qquad \text{(E) } 48$" style="vertical-align: -4px" width="366" height="17" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>A seven-digit palindrome is a number of the form <img src="//latex.artofproblemsolving.com/3/1/1/3119be0cfee713da1d1a1d9003b7a5d8d307898c.png" class="latex" alt="$\overline{abcdcba}$" width="60" height="16" />. Clearly, <img src="//latex.artofproblemsolving.com/9/6/a/96ab646de7704969b91c76a214126b45f2b07b25.png" class="latex" alt="$d$" width="9" height="12" /> must be <img src="//latex.artofproblemsolving.com/7/9/0/79069377f91364c2f87a64e5f9f562a091c8a6c1.png" class="latex" alt="$5$" width="8" height="12" />, as we have an odd number of fives. We are then left with <img src="//latex.artofproblemsolving.com/6/6/d/66d5da31c8dee8743914deff32bbd28e96454615.png" class="latex" alt="$\{a,b,c\} = \{2,3,5\}$" style="vertical-align: -4px" width="144" height="18" />. There are <img src="//latex.artofproblemsolving.com/6/8/5/6859713adc973d3fbe97fafef9ce829757e54b12.png" class="latex" alt="$3!$" width="12" height="13" /> permutations of these three numbers, since each is reflected over the midpoint we only have to count the first there. Each of the <img src="//latex.artofproblemsolving.com/a/7/0/a7092fa22bfc58692c6ccd13c1a49d883dab6c4e.png" class="latex" alt="$\boxed{(A)6}$" style="vertical-align: -10px" width="48" height="29" /> permutations of the set <img src="//latex.artofproblemsolving.com/5/7/8/5785b403681355d362f5f0bdf549d253f6504772.png" class="latex" alt="$\{2,3,5\}$" style="vertical-align: -4px" width="60" height="18" /> will give us one palindrome.
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>Say we have a 2 first. Then, we have a 2 pinned as the last digit, so we have to fill in the remaining digits with only 3's and 5's. We have 2 options for the second digit then, and the rest is fixed. This means that we have <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" /> ways for this case.
</p><p>Say we have a 3 first. By symmetry, this is the same as the 2 cases, so we have <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" /> ways.
</p><p>Say we have a 5 first. We then have a 5 in the middle. We can either have a 2 second or a 3 second. So we have <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" /> ways.
</p><p>This means that our answer is <img src="//latex.artofproblemsolving.com/7/c/5/7c5ffa1e4312e9abb1ca1993a68a25218ae39692.png" class="latex" alt="$2+2+2=\boxed{\textbf{(A)}6}$" style="vertical-align: -10px" width="146" height="28" />
</p><p><br />
~yofro
</p>
