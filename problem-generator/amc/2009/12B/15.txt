<p>Assume <img src="//latex.artofproblemsolving.com/6/1/8/618a918d963a7a6c2934db2fd337c7c56ac8fea8.png" class="latex" alt="$0 &lt; r &lt; 3$" style="vertical-align: 0px" width="75" height="13" />.  Below are five equations for <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" />.  Which equation has the largest solution <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" />?
</p><p><img src="//latex.artofproblemsolving.com/8/7/6/87670fecb4a24fee91f26ccf345d7501b1aa43c3.png" class="latex" alt="$\textbf{(A)}\ 3(1 + r)^x = 7\qquad \textbf{(B)}\ 3(1 + r/10)^x = 7\qquad \textbf{(C)}\ 3(1 + 2r)^x = 7$" style="vertical-align: -4px" width="520" height="18" />
<img src="//latex.artofproblemsolving.com/5/c/a/5cabe7a6d7e790c55efb7aa1663cfa7e5fe079f5.png" class="latex" alt="$\textbf{(D)}\ 3(1 + \sqrt {r})^x = 7\qquad \textbf{(E)}\ 3(1 + 1/r)^x = 7$" style="vertical-align: -4px" width="344" height="19" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p><b>(B)</b> Intuitively, <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> will be largest for that option for which the value in the parentheses is smallest.
</p><p>Formally, first note that each of the values in parentheses is larger than <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" />. Now, each of the options is of the form <img src="//latex.artofproblemsolving.com/f/0/e/f0eb8eddef90a515326003d01a6671f0769d7646.png" class="latex" alt="$3f(r)^x = 7$" style="vertical-align: -4px" width="84" height="18" />. This can be rewritten as <img src="//latex.artofproblemsolving.com/a/e/8/ae8b0fc901a3def6150afa2e923eafa54276fc8d.png" class="latex" alt="$x\log f(r) = \log\frac 73$" style="vertical-align: -12px" width="136" height="37" />. As <img src="//latex.artofproblemsolving.com/7/6/d/76d109a3f715d746523840b4603534398e37a78c.png" class="latex" alt="$f(r)&gt;1$" style="vertical-align: -4px" width="66" height="18" />, we have <img src="//latex.artofproblemsolving.com/4/5/7/45739a2b1d04ed93b5eba47aadb6f36484beb1e7.png" class="latex" alt="$\log f(r)&gt;0$" style="vertical-align: -4px" width="93" height="18" />. Thus <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> is the largest for the option for which <img src="//latex.artofproblemsolving.com/e/d/6/ed66420129e21c0c0e72c21558750e7c04b3b89e.png" class="latex" alt="$\log f(r)$" style="vertical-align: -4px" width="58" height="18" /> is smallest. And as <img src="//latex.artofproblemsolving.com/e/d/6/ed66420129e21c0c0e72c21558750e7c04b3b89e.png" class="latex" alt="$\log f(r)$" style="vertical-align: -4px" width="58" height="18" /> is an increasing function, this is the option for which <img src="//latex.artofproblemsolving.com/4/d/4/4d404a4bf3e67e6fd88a82c768979219e455e3e5.png" class="latex" alt="$f(r)$" style="vertical-align: -4px" width="32" height="18" /> is smallest.
</p><p>We now get the following easier problem: Given that <img src="//latex.artofproblemsolving.com/6/5/f/65f8fd8b742a09b42eb620851526ff44732a448c.png" class="latex" alt="$0&lt;r&lt;3$" style="vertical-align: 0px" width="75" height="13" />, find the smallest value in the set <img src="//latex.artofproblemsolving.com/1/8/6/1864771825af631fcfa9126e3e195c2620ef26dc.png" class="latex" alt="$\{ 1+r, 1+r/10, 1+2r, 1+\sqrt r, 1+1/r\}$" style="vertical-align: -4px" width="317" height="19" />.
</p><p>Clearly <img src="//latex.artofproblemsolving.com/1/1/6/116e6ecec919528befe7e814a12b484bdcb97435.png" class="latex" alt="$1+r/10$" style="vertical-align: -4px" width="66" height="18" /> is smaller than the first and the third option. 
</p><p>We have <img src="//latex.artofproblemsolving.com/a/1/9/a192e72acd73a37b62dda4ba63b320cbabb8ab45.png" class="latex" alt="$r^2 &lt; 10$" style="vertical-align: 0px" width="58" height="15" />, dividing both sides by <img src="//latex.artofproblemsolving.com/4/9/d/49de2adfaf910759ea05607eb07c363009ec392c.png" class="latex" alt="$10r$" style="vertical-align: 0px" width="26" height="13" /> we get <img src="//latex.artofproblemsolving.com/7/1/2/712735ad2549e79652b1eb859575d7484a73f54b.png" class="latex" alt="$r/10 &lt; 1/r$" style="vertical-align: -4px" width="86" height="18" />. 
</p><p>And finally, <img src="//latex.artofproblemsolving.com/2/8/9/289e784712c93240a3e76184dcd45efbdc84583e.png" class="latex" alt="$r/100 &lt; 1$" style="vertical-align: -4px" width="77" height="18" />, therefore <img src="//latex.artofproblemsolving.com/2/4/0/240f7a3036653d855f07f39c205718f18b6fe5fa.png" class="latex" alt="$r^2/100 &lt; r$" style="vertical-align: -4px" width="85" height="19" />, and as both sides are positive, we can take the square root and get <img src="//latex.artofproblemsolving.com/c/f/1/cf13860b38908c9930b2d9e04390f076bbfa68c3.png" class="latex" alt="$r/10 &lt; \sqrt r$" style="vertical-align: -4px" width="84" height="19" />.
</p><p>Thus the answer is <img src="//latex.artofproblemsolving.com/4/2/9/429a820f96d5a9de225c7c3202add78938ab5469.png" class="latex" alt="$\boxed{\textbf{(B) } 3\left(1+\frac{r}{10}\right)^x = 7}$" style="vertical-align: -18px" width="175" height="45" />.
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>As stated in the first solution, <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> will be largest when the value in the parenthesis is smallest.
</p><p>We can plug in a value for <img src="//latex.artofproblemsolving.com/b/5/5/b55ca7a0aa88ab7d58f4fc035317fdac39b17861.png" class="latex" alt="$r$" width="8" height="8" /> (for instance, <img src="//latex.artofproblemsolving.com/a/0/7/a0736cc24657e4fbe87410091de1c4ebd44e26db.png" class="latex" alt="$r=2$" width="41" height="12" />) and see which parenthesis has the smallest value. Doing so, we see that B is the answer.
</p><p>-Made_in_2004
</p>
