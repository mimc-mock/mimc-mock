<p>What is the area of the region enclosed by the graph of the equation <img src="//latex.artofproblemsolving.com/8/6/4/864191ae88a08202ee04fd5d9d1ddd4b594b4e94.png" class="latex" alt="$x^2+y^2=|x|+|y|?$" style="vertical-align: -4px" width="151" height="19" />
</p><p><img src="//latex.artofproblemsolving.com/8/c/2/8c2a3131093aa24e41f2c6f016bdf3cf574d84f2.png" class="latex" alt="$\textbf{(A)}\ \pi+\sqrt{2}\qquad\textbf{(B)}\ \pi+2\qquad\textbf{(C)}\ \pi+2\sqrt{2}\qquad\textbf{(D)}\ 2\pi+\sqrt{2}\qquad\textbf{(E)}\ 2\pi+2\sqrt{2}$" style="vertical-align: -4px" width="613" height="21" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>Without loss of generality (WLOG) note that if a point in the first quadrant satisfies the equation, so do its corresponding points in the other three quadrants. Therefore, we can assume that <img src="//latex.artofproblemsolving.com/3/5/a/35a6b0a8286d4b943c1c07eaa2efbd9bc5e100d0.png" class="latex" alt="$x, y \ge 0$" style="vertical-align: -3px" width="61" height="16" />, which implies that <img src="//latex.artofproblemsolving.com/e/f/d/efd7084dd690b95ccfc16b539645bea69168d54e.png" class="latex" alt="$|x|=x$" style="vertical-align: -4px" width="54" height="18" /> and <img src="//latex.artofproblemsolving.com/a/1/d/a1d193301712ebb118f59eff4b12f614f21bfbe6.png" class="latex" alt="$|y|=y$" style="vertical-align: -4px" width="53" height="18" />, and multiply by <img src="//latex.artofproblemsolving.com/c/7/c/c7cab1a05e1e0c1d51a6a219d96577a16b7abf9d.png" class="latex" alt="$4$" style="vertical-align: 0px" width="9" height="12" /> at the end.
</p><p>We can rearrange the equation to get <img src="//latex.artofproblemsolving.com/0/d/3/0d3d9475006805ac5ebd15d77f13f5267db59030.png" class="latex" alt="$x^2-x+y^2-y=0 \Rightarrow (x-\tfrac12)^2+(y-\tfrac12)^2=(\tfrac{\sqrt2}{2})^2$" style="vertical-align: -6px" width="400" height="24" />, which describes a circle with center <img src="//latex.artofproblemsolving.com/3/8/8/3889b6be2bb0b112c99948973dba491a8e3c37c0.png" class="latex" alt="$(\tfrac12, \tfrac12)$" style="vertical-align: -6px" width="42" height="21" /> and radius <img src="//latex.artofproblemsolving.com/6/b/7/6b712fdbf0daf3dd2839265f39ba8d04f23fbc31.png" class="latex" alt="$\tfrac{\sqrt2}{2}.$" style="vertical-align: -6px" width="25" height="24" /> It's clear we now want to find the union of four equal areas.
</p><p><img src="//latex.artofproblemsolving.com/a/0/8/a0815e5c87cf231fd4778fdfa3824fe489191f98.png" class="latexcenter" alt="[asy]draw((0,-1.5)--(0,1.5),EndArrow);draw((-1.5,0)--(1.5,0),EndArrow);draw((0,1)--(1,0)--(0,-1)--(-1,0)--cycle,dotted); for(int i=0;i&lt;4;++i){draw(rotate(i*90,(0,0))*arc((1/2,1/2),sqrt(1/2),-45,135));dot(rotate(i*90,(0,0))*(1/2,1/2));}[/asy]" width="252" height="252" />
There are several ways to find the area, but note that if you connect <img src="//latex.artofproblemsolving.com/6/4/5/6457ca2dcd40dc21d9aa0db18c66bbac9af4d899.png" class="latex" alt="$(0, 1)$" style="vertical-align: -4px" width="38" height="18" /> to its other three respective points in the other three quadrants, you get a square of area <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" />, along with four half-circles of diameter <img src="//latex.artofproblemsolving.com/2/c/4/2c4dc1409208ff6a3039f4871d85c16763334983.png" class="latex" alt="$\sqrt{2}$" style="vertical-align: -1px" width="24" height="18" />, for a total area of <img src="//latex.artofproblemsolving.com/6/2/3/623f05c576bb9d9a0b2fe41b79af612658829131.png" class="latex" alt="$2+2\cdot(\tfrac{\sqrt2}{2})^2\pi = \pi + 2$" style="vertical-align: -6px" width="173" height="24" /> which is <img src="//latex.artofproblemsolving.com/a/2/1/a2194b81820e8380be586c1564c3f5d03d5c100d.png" class="latex" alt="$\boxed{\textbf{(B)}}$" style="vertical-align: -10px" width="40" height="28" />.
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>Another way to solve this problem is using cases. 
Though this may seem tedious, we only have to do one case since the area enclosed is symmetrical.
The equation for this figure is <img src="//latex.artofproblemsolving.com/5/a/6/5a6700699995139c6fb34b87a687bb9ba2fbfbca.png" class="latex" alt="$x^2+y^2=|x|+|y|$" style="vertical-align: -4px" width="141" height="19" /> To make this as easy as possible,
we can make both <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> and <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" /> positive. Simplifying the equation for <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> and <img src="//latex.artofproblemsolving.com/0/9/2/092e364e1d9d19ad5fffb0b46ef4cc7f2da02c1c.png" class="latex" alt="$y$" style="vertical-align: -3px" width="9" height="11" /> being positive,
we get the equation <img src="//latex.artofproblemsolving.com/7/a/5/7a503b637b4b7669fb6cb61814d84e43250033c5.png" class="latex" alt="$x^{2} +y^{2} -x-y = 0.$" style="vertical-align: -3px" width="158" height="18" /> 
</p><p>Using the complete the square method, we get <img src="//latex.artofproblemsolving.com/2/4/7/2470e4eef0b5a7c7a0e6fa784ffa9853340bae49.png" class="latex" alt="$\left(x-\frac{1}{2}\right)^{2} + \left(y-\frac{1}{2}\right)^{2}=\frac{1}{2}$" style="vertical-align: -17px" width="215" height="46" />
</p><p>Therefore, the origin of this section of the shape is at <img src="//latex.artofproblemsolving.com/8/d/d/8ddb4d19b9e7708b2b8994015ee286ca37d7a575.png" class="latex" alt="$\left(\frac{1}{2}, \frac{1}{2}\right).$" style="vertical-align: -17px" width="67" height="43" />
</p><p>Using the equation we can also see that the radius has a length of <img src="//latex.artofproblemsolving.com/c/c/7/cc7c4b51e06b26289d792878cbf9efac28aac903.png" class="latex" alt="$\frac{\sqrt{2}}{2}$" style="vertical-align: -12px" width="26" height="41" /> .
</p><p>With this shape we see that this shape can be cut into a right triangle and a semicircle.
The length of the hypotenuse of the triangle is <img src="//latex.artofproblemsolving.com/2/c/4/2c4dc1409208ff6a3039f4871d85c16763334983.png" class="latex" alt="$\sqrt{2}$" style="vertical-align: -1px" width="24" height="18" /> so using special right triangles, we see that
the area of the triangle is <img src="//latex.artofproblemsolving.com/2/f/9/2f960094315d60883495f9b74148e17487ee9584.png" class="latex" alt="$\frac{1}{2}$" style="vertical-align: -12px" width="11" height="37" /> . The semicircle has the area of <img src="//latex.artofproblemsolving.com/d/5/d/d5de598861d2ee80dc5debded662bb289995ddf7.png" class="latex" alt="$\frac{1}{4}\pi$" style="vertical-align: -13px" width="23" height="37" />.
</p><p>But this is only <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" /> case. There are <img src="//latex.artofproblemsolving.com/c/7/c/c7cab1a05e1e0c1d51a6a219d96577a16b7abf9d.png" class="latex" alt="$4$" style="vertical-align: 0px" width="9" height="12" /> cases in total so we have to multiply 
<img src="//latex.artofproblemsolving.com/b/b/2/bb2a34ff03b1bfccb4ca318e87fa418b47411b65.png" class="latex" alt="$\frac{1}{2}+\frac{1}{4}\pi$" style="vertical-align: -13px" width="59" height="37" /> by <img src="//latex.artofproblemsolving.com/c/7/c/c7cab1a05e1e0c1d51a6a219d96577a16b7abf9d.png" class="latex" alt="$4$" style="vertical-align: 0px" width="9" height="12" />. 
</p><p>After multiplying, our answer is: <img src="//latex.artofproblemsolving.com/4/2/b/42b633ae2a7b617ff80a7a63489e218ac3818e0c.png" class="latexcenter" alt="\[\boxed{\textbf{(B)}  \pi+2}.\]" width="85" height="28" />
</p>
<h2><span class="mw-headline" id="Solution_3:_Analytic_geometry">Solution 3: Analytic geometry</span></h2>
<p>We solve with cases. The cases are:
Case 1: <img src="//latex.artofproblemsolving.com/5/2/1/521d1fb704211cad3c2ed5adee17228c8c7b1c51.png" class="latex" alt="$x\geq0, y\geq0.$" style="vertical-align: -4px" width="102" height="17" />
Case 2: <img src="//latex.artofproblemsolving.com/7/4/3/743b256ed27983eb3818440bbf31a1fc175f969d.png" class="latex" alt="$x\geq0, y&lt;0.$" style="vertical-align: -4px" width="102" height="17" />
Case 3: <img src="//latex.artofproblemsolving.com/f/f/b/ffb3bdc3a1a29375ae122eae3f2982de5fa1c6f3.png" class="latex" alt="$x&lt;0, y\geq0.$" style="vertical-align: -4px" width="102" height="17" />
Case 4: <img src="//latex.artofproblemsolving.com/f/2/e/f2e6814677ae8407c0e122e9d28ea6ba08802abe.png" class="latex" alt="$x&lt;0, y&lt;0.$" style="vertical-align: -4px" width="102" height="17" />
</p><p>We can quickly realize that the whole figure is symmetrical; so when you figure out the first case, you get the first part is <img src="//latex.artofproblemsolving.com/d/b/1/db1a16d97676fc630481bc98628f01b6800d572a.png" class="latex" alt="$\left(x-\dfrac12\right)^2+\left(x-\dfrac12\right)^2$" style="vertical-align: -18px" width="183" height="48" /> you can figure out the whole figure: (scaled 8x).
<img src="//latex.artofproblemsolving.com/a/b/2/ab2ee4d5698c396d94a230f8e55ae5ba42412552.png" class="latexcenter" alt="[asy] size(400); import TrigMacros;  rr_cartesian_axes(-10,10,-10,10,complexplane=false, usegrid = true);    draw(circle((4,4), 4*1.41421356237)); draw(circle((4,-4), 4*1.41421356237)); draw(circle((-4,4), 4*1.41421356237)); draw(circle((-4,-4), 4*1.41421356237));   [/asy]" width="668" height="662" />
The way we figure out the area is by splitting it the following way:
<img src="//latex.artofproblemsolving.com/0/0/5/0052669c89d7a2e1442cc2b51ddd8bd37640827d.png" class="latexcenter" alt="[asy] size(400); import TrigMacros;  rr_cartesian_axes(-10,10,-10,10,complexplane=false, usegrid = true);    draw(circle((4,4), 4*1.41421356237)); draw(circle((4,-4), 4*1.41421356237)); draw(circle((-4,4), 4*1.41421356237)); draw(circle((-4,-4), 4*1.41421356237));  real f(real x) { return -x+8; } draw(graph(f,0,8), red+linewidth(1.5));   real g(real x) { return x+8; } draw(graph(g,0,-8), red+linewidth(1.5));  real f(real x) { return -x+8; } draw(graph(f,0,8), red+linewidth(1.5));  real h(real x) { return -x-8; } draw(graph(h,-8,0), red+linewidth(1.5));   real z(real x) { return x-8; } draw(graph(z,0,8), red+linewidth(1.5));   pair A,B,C,D;  A = (8,0); B = (0,8); C = (-8,0); D = (0,-8); fill(A--B--C--D--cycle, red); [/asy]" width="668" height="662" />
</p><p>We know each of the red lines is a diameter of the circle which is <img src="//latex.artofproblemsolving.com/1/1/4/114e91200cd4c857d233c7fed6cbe7451f3a82fc.png" class="latex" alt="$\sqrt2$" style="vertical-align: -1px" width="24" height="18" />. So the area of the red is 2. We know that the area of each semicircle is <img src="//latex.artofproblemsolving.com/4/7/e/47eb6c50cd8c7a6eb285ecb3bce124145b9b87d6.png" class="latex" alt="$\dfrac14 \pi$" style="vertical-align: -13px" width="26" height="38" /> so the area of the semicircles combines is <img src="//latex.artofproblemsolving.com/f/2/c/f2ca003a7da0de4994b4733e203b74ff52d42553.png" class="latex" alt="$\pi$" width="10" height="8" />. Thus we get  <img src="//latex.artofproblemsolving.com/6/3/0/6303ff41b85e1118e4830bb532daab3b091df628.png" class="latex" alt="$\boxed{\textbf{(B)}  \pi+2}.$" style="vertical-align: -11px" width="89" height="30" />
</p>
