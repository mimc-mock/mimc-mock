<p>How many solutions does the equation <img src="//latex.artofproblemsolving.com/3/0/f/30f3377d5d3de02ba154a81878c7b6d501119720.png" class="latex" alt="$\tan(2x)=\cos(\tfrac{x}{2})$" style="vertical-align: -6px" width="132" height="20" /> have on the interval <img src="//latex.artofproblemsolving.com/e/c/c/ecccefb4c460857891d99273237d9deb9ecf41c1.png" class="latex" alt="$[0,2\pi]?$" style="vertical-align: -5px" width="55" height="18" />
</p><p><img src="//latex.artofproblemsolving.com/9/9/5/9959d47fffb760b6cf89894df252b99436e4d8f5.png" class="latex" alt="$\textbf{(A)}\ 1\qquad\textbf{(B)}\ 2\qquad\textbf{(C)}\ 3\qquad\textbf{(D)}\ 4\qquad\textbf{(E)}\ 5$" style="vertical-align: -4px" width="348" height="17" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>We count the intersections of the graphs of <img src="//latex.artofproblemsolving.com/0/2/b/02bc38defcfd3aecca03c576c8738c7140bc0c2b.png" class="latex" alt="$y=\tan(2x)$" style="vertical-align: -5px" width="96" height="20" /> and <img src="//latex.artofproblemsolving.com/8/b/1/8b1efdf68ff14e94ab2e7bc76fae386e0175bc35.png" class="latex" alt="$y=\cos\left(\frac x2\right):$" style="vertical-align: -13px" width="109" height="35" />
</p>
<ol style="margin-left: 1.5em;">
  <li>The graph of <img src="//latex.artofproblemsolving.com/0/2/b/02bc38defcfd3aecca03c576c8738c7140bc0c2b.png" class="latex" alt="$y=\tan(2x)$" style="vertical-align: -5px" width="96" height="20" /> has a period of <img src="//latex.artofproblemsolving.com/e/2/5/e25523ad869caf448d31d6f06cdbca4d6d60070f.png" class="latex" alt="$\frac{\pi}{2},$" style="vertical-align: -12px" width="19" height="33" /> asymptotes at <img src="//latex.artofproblemsolving.com/c/5/f/c5f203746531f578cebb08830469f8bbb93efacd.png" class="latex" alt="$x=\frac{\pi}{4}+\frac{k\pi}{2},$" style="vertical-align: -13px" width="104" height="39" /> and zeros at <img src="//latex.artofproblemsolving.com/3/3/e/33e07e1a8698b550c7fc4835f1adf607801614b3.png" class="latex" alt="$x=\frac{k\pi}{2}$" style="vertical-align: -13px" width="62" height="39" /> for some integer <img src="//latex.artofproblemsolving.com/5/d/4/5d48e0968a64d782fe75bca620ce749938b7d0bb.png" class="latex" alt="$k.$" width="13" height="12" /> <p>
On the interval <img src="//latex.artofproblemsolving.com/b/e/2/be287a7495ea16e7d9471055df3954dac7b0b0d8.png" class="latex" alt="$[0,2\pi],$" style="vertical-align: -5px" width="51" height="18" /> the graph has five branches: <img src="//latex.artofproblemsolving.com/0/f/6/0f61c4e0ed515f1a18498e7b2922905c5f1e2a81.png" class="latexcenter" alt="\[\biggl[0,\frac{\pi}{4}\biggr),\left(\frac{\pi}{4},\frac{3\pi}{4}\right),\left(\frac{3\pi}{4},\frac{5\pi}{4}\right),\left(\frac{5\pi}{4},\frac{7\pi}{4}\right),\left(\frac{7\pi}{4},2\pi\right].\]" width="411" height="43" />
Note that <img src="//latex.artofproblemsolving.com/4/b/4/4b47cc490dd6adb8ff4aac89bf231f3c932aaaf3.png" class="latex" alt="$\tan(2x)\in[0,\infty)$" style="vertical-align: -5px" width="131" height="20" /> for the first branch, <img src="//latex.artofproblemsolving.com/2/c/2/2c25a3fc0e0584677596158e394133ccd734fa48.png" class="latex" alt="$\tan(2x)\in(-\infty,\infty)$" style="vertical-align: -5px" width="157" height="20" /> for the three middle branches, and <img src="//latex.artofproblemsolving.com/5/b/8/5b8d849c9e05ed072772ac711429ff9f30bb763d.png" class="latex" alt="$\tan(2x)\in(-\infty,0]$" style="vertical-align: -5px" width="146" height="20" /> for the last branch. Moreover, all branches are strictly increasing.
</p></li><p class="mw-empty-elt">
  </p><li>The graph of <img src="//latex.artofproblemsolving.com/6/9/e/69e1e66698fb2c2329223aaff51a29400272eb9f.png" class="latex" alt="$y=\cos\left(\frac x2\right)$" style="vertical-align: -13px" width="99" height="35" /> has a period of <img src="//latex.artofproblemsolving.com/a/7/2/a72a1caa0cb9e0f72f414ebd8180ea676808568f.png" class="latex" alt="$4\pi$" style="vertical-align: 0px" width="19" height="12" /> and zeros at <img src="//latex.artofproblemsolving.com/d/2/c/d2cff5896f886af31e6ab132b5e4ff33e37a3fe1.png" class="latex" alt="$x=\pi+2k\pi$" style="vertical-align: -2px" width="100" height="16" /> for some integer <img src="//latex.artofproblemsolving.com/5/d/4/5d48e0968a64d782fe75bca620ce749938b7d0bb.png" class="latex" alt="$k.$" width="13" height="12" /> <p>
On the interval <img src="//latex.artofproblemsolving.com/b/e/2/be287a7495ea16e7d9471055df3954dac7b0b0d8.png" class="latex" alt="$[0,2\pi],$" style="vertical-align: -5px" width="51" height="18" /> note that <img src="//latex.artofproblemsolving.com/b/4/e/b4ed836dbb1efbe3523d09e1a13e10d81f07d20d.png" class="latex" alt="$\cos\left(\frac x2\right)\in[-1,1].$" style="vertical-align: -13px" width="143" height="35" /> Moreover, the graph is strictly decreasing.</p></li><p class="mw-empty-elt">
</p></ol>
<p>The graphs of <img src="//latex.artofproblemsolving.com/0/2/b/02bc38defcfd3aecca03c576c8738c7140bc0c2b.png" class="latex" alt="$y=\tan(2x)$" style="vertical-align: -5px" width="96" height="20" /> and <img src="//latex.artofproblemsolving.com/6/9/e/69e1e66698fb2c2329223aaff51a29400272eb9f.png" class="latex" alt="$y=\cos\left(\frac x2\right)$" style="vertical-align: -13px" width="99" height="35" /> intersect once on each of the five branches of <img src="//latex.artofproblemsolving.com/b/5/3/b53066956af2d4fc7ddebb90c2eb5c8bf7d15181.png" class="latex" alt="$y=\tan(2x),$" style="vertical-align: -5px" width="101" height="20" /> as shown below:
<img src="//latex.artofproblemsolving.com/c/1/4/c14719331d5c1b4c94bdc7825e40af9d20180888.png" class="latexcenter" alt="[asy] /* Made by MRENTHUSIASM */ size(800,200);   real f(real x) { return tan(2*x); }  real g(real x) { return cos(x/2); }  draw(graph(f,0,atan(3)/2),red,&quot;$y=\tan(2x)$&quot;); draw(graph(f,-atan(3)/2+pi/2,atan(3)/2+pi/2),red); draw(graph(f,-atan(3)/2+2*pi/2,atan(3)/2+2*pi/2),red); draw(graph(f,-atan(3)/2+3*pi/2,atan(3)/2+3*pi/2),red); draw(graph(f,-atan(3)/2+4*pi/2,2*pi),red); draw(graph(g,0,2pi),blue,&quot;$y=\cos\left(\frac x2\right)$&quot;);  real xMin = 0; real xMax = 9/4*pi; real yMin = -3; real yMax = 3;  //Draws the horizontal gridlines void horizontalLines() {   for (real i = yMin+1; i &lt; yMax; ++i)   {     draw((xMin,i)--(xMax,i), mediumgray+linewidth(0.4));   } }  //Draws the vertical gridlines void verticalLines() {   for (real i = xMin+pi/2; i &lt; xMax; i+=pi/2)   {     draw((i,yMin)--(i,yMax), mediumgray+linewidth(0.4));   } }  //Draws the horizontal ticks void horizontalTicks() {   for (real i = yMin+1; i &lt; yMax; ++i)   {     draw((-1/8,i)--(1/8,i), black+linewidth(1));   } }  //Draws the vertical ticks void verticalTicks() {   for (real i = xMin+pi/2; i &lt; xMax; i+=pi/2)   {     draw((i,-1/8)--(i,1/8), black+linewidth(1));   } }  horizontalLines(); verticalLines(); horizontalTicks(); verticalTicks(); draw((xMin,0)--(xMax,0),black+linewidth(1.5),EndArrow(5)); draw((0,yMin)--(0,yMax),black+linewidth(1.5),EndArrow(5)); label(&quot;$x$&quot;,(xMax,0),(2,0)); label(&quot;$y$&quot;,(0,yMax),(0,2));  pair A[], B[]; A[0] = (2pi,0); A[1] = (0,2); A[2] = (0,0); A[3] = (0,-2); B[0] = intersectionpoints(graph(f,0,atan(3)/2),graph(g,0,2pi))[0]; B[1] = intersectionpoints(graph(f,-atan(3)/2+pi/2,atan(3)/2+pi/2),graph(g,0,2pi))[0]; B[2] = intersectionpoints(graph(f,-atan(3)/2+2*pi/2,atan(3)/2+2*pi/2),graph(g,0,2pi))[0]; B[3] = intersectionpoints(graph(f,-atan(3)/2+3*pi/2,atan(3)/2+3*pi/2),graph(g,0,2pi))[0]; B[4] = intersectionpoints(graph(f,-atan(3)/2+4*pi/2,atan(3)/2+4*pi/2),graph(g,0,2pi))[0];  label(&quot;$2\pi$&quot;,A[0],(0,-2.5)); label(&quot;$2$&quot;,A[1],(-2.5,0)); label(&quot;$0$&quot;,A[2],(-2.5,0)); label(&quot;$-2$&quot;,A[3],(-2.5,0));  for (int i = 0; i &lt; 5; ++i) { 	dot(B[i],black+linewidth(5)); }  add(legend(),point(E),60E,UnFill); [/asy]" width="732" height="335" />
Therefore, the answer is <img src="//latex.artofproblemsolving.com/b/9/6/b96ddb77e36f43901ed14f2791125d045ae9198f.png" class="latex" alt="$\boxed{\textbf{(E)}\ 5}.$" style="vertical-align: -11px" width="60" height="30" />
