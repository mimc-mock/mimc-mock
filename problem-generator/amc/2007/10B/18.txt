<p>A circle of radius <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" /> is surrounded by <img src="//latex.artofproblemsolving.com/c/7/c/c7cab1a05e1e0c1d51a6a219d96577a16b7abf9d.png" class="latex" alt="$4$" style="vertical-align: 0px" width="9" height="12" /> circles of radius <img src="//latex.artofproblemsolving.com/b/5/5/b55ca7a0aa88ab7d58f4fc035317fdac39b17861.png" class="latex" alt="$r$" width="8" height="8" /> as shown. What is <img src="//latex.artofproblemsolving.com/b/5/5/b55ca7a0aa88ab7d58f4fc035317fdac39b17861.png" class="latex" alt="$r$" width="8" height="8" />?
</p>
<center><img src="//latex.artofproblemsolving.com/0/9/1/0916dbd96f1ec23e5b075b5070e690e9970968fa.png" class="latexcenter" alt="[asy] unitsize(5mm); defaultpen(linewidth(.8pt)+fontsize(7pt)); dotfactor=4;  real r1=1, r2=1+sqrt(2); pair A=(0,0), B=(1+sqrt(2),1+sqrt(2)), C=(-1-sqrt(2),1+sqrt(2)), D=(-1-sqrt(2),-1-sqrt(2)), E=(1+sqrt(2),-1-sqrt(2)); pair A1=(1,0), B1=(2+2sqrt(2),1+sqrt(2)), C1=(0,1+sqrt(2)), D1=(0,-1-sqrt(2)), E1=(2+2sqrt(2),-1-sqrt(2)); path circleA=Circle(A,r1); path circleB=Circle(B,r2); path circleC=Circle(C,r2); path circleD=Circle(D,r2); path circleE=Circle(E,r2); draw(circleA); draw(circleB); draw(circleC); draw(circleD); draw(circleE); draw(A--A1); draw(B--B1); draw(C--C1); draw(D--D1); draw(E--E1);  label(&quot;$1$&quot;,midpoint(A--A1),N); label(&quot;$r$&quot;,midpoint(B--B1),N); label(&quot;$r$&quot;,midpoint(C--C1),N); label(&quot;$r$&quot;,midpoint(D--D1),N); label(&quot;$r$&quot;,midpoint(E--E1),N); [/asy]" width="232" height="232" /></center>
<p><img src="//latex.artofproblemsolving.com/2/3/6/2361d38ed8afbfb86a5c02aef4d0c940ed394e48.png" class="latex" alt="$\textbf{(A) } \sqrt{2} \qquad\textbf{(B) } 1+\sqrt{2} \qquad\textbf{(C) } \sqrt{6} \qquad\textbf{(D) } 3 \qquad\textbf{(E) } 2+\sqrt{2}$" style="vertical-align: -4px" width="477" height="21" />
</p>
<h2><span class="mw-headline" id="Solutions">Solutions</span></h2>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>You can express the line connecting the centers of an outer circle and the inner circle in two different ways. You can add the radius of both circles to get <img src="//latex.artofproblemsolving.com/f/7/f/f7fd3b9bd3a51464397537cb3adb60f8eda7aefa.png" class="latex" alt="$r+1.$" style="vertical-align: -1px" width="43" height="13" /> You can also add the radius of two outer circles and use a <img src="//latex.artofproblemsolving.com/2/2/2/22231a5cdd4fe05a11f4b577e3b81221f3017a85.png" class="latex" alt="$45-45-90$" style="vertical-align: 0px" width="98" height="13" /> triangle to get <img src="//latex.artofproblemsolving.com/5/8/9/58984ecde73ce7bcefa48a53d8ffc21fffaea2d0.png" class="latex" alt="$\frac{2r}{\sqrt{2}} = r\sqrt{2}.$" style="vertical-align: -17px" width="89" height="41" /> Since both expressions represent the same length, you can set them equal to each other.
<img src="//latex.artofproblemsolving.com/7/a/6/7a6c0a79bd7faef974678cfc0067043cda580ac2.png" class="latexcenter" alt="\begin{align*} r+1&amp;=r\sqrt{2}\\ 1&amp;=r(\sqrt{2}-1)\end{align*}" width="140" height="51" />
<img src="//latex.artofproblemsolving.com/b/3/5/b35282f84928a8198b71bb4355abf2ba9d1437e5.png" class="latexcenter" alt="\[r = \frac{1}{\sqrt{2}-1} = \frac{1(\sqrt{2}+1)}{(\sqrt{2}-1)(\sqrt{2}+1)} = \frac{\sqrt{2}+1}{2-1} = \boxed{\mathrm{(B) \ } 1 + \sqrt{2}}\]" width="467" height="47" />
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>You can solve this problem by setting up a simple equation with the Pythagorean Theorem. The hypotenuse would be a segment that includes the radius of two circles on opposite corners and the diameter of the middle circle. This results in a segment of length <img src="//latex.artofproblemsolving.com/8/c/4/8c4fad7d86478236c5fcd62b5ddef78860bc031a.png" class="latex" alt="$2r+2$" style="vertical-align: -1px" width="48" height="13" />. The two legs are each the length between the centers of two large, adjacent circles, thus they are each equal to <img src="//latex.artofproblemsolving.com/0/a/5/0a5745df50c8607bdc9b8de4968315633a3047dc.png" class="latex" alt="$2r$" width="17" height="12" />. 
Using the Pythagorean Theorem:
<img src="//latex.artofproblemsolving.com/f/1/7/f17cd1baf8a602810b58471181b258346d7b6c8b.png" class="latexcenter" alt="\begin{align*} (2r+2)^2 = 2(2r)^2\\ 4r^2+8r+4=8r^2\\ r^2+2r+1=2r^2\\ r^2-2r-1=0\\ r=\frac{2+\sqrt{4-(-4)}}{2}=\frac{2+2\sqrt{2}}{2}=\boxed{\mathrm{(B) \ } 1 + \sqrt{2}} \end{align*}" width="371" height="152" />
</p>
