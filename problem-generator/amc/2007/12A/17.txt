<p>Suppose that <img src="//latex.artofproblemsolving.com/4/1/0/410349baf2b0e584b6b8f81dc1e0e766bfd6021a.png" class="latex" alt="$\sin a + \sin b = \sqrt{\frac{5}{3}}$" style="vertical-align: -14px" width="146" height="43" /> and <img src="//latex.artofproblemsolving.com/b/5/0/b50b84521e9b492bb345a9d89c375ea06d180123.png" class="latex" alt="$\cos a + \cos b = 1$" style="vertical-align: -1px" width="126" height="14" />. What is <img src="//latex.artofproblemsolving.com/d/a/4/da4f2ad5e79d2561ffce68e018d9cd72d67a2de6.png" class="latex" alt="$\cos (a - b)$" style="vertical-align: -4px" width="76" height="18" />?
</p><p><img src="//latex.artofproblemsolving.com/3/a/6/3a66922efb141e4bc0797cd28600a04a7729a56b.png" class="latex" alt="$\mathrm{(A)}\ \sqrt{\frac{5}{3}} - 1\qquad \mathrm{(B)}\ \frac 13\qquad \mathrm{(C)}\ \frac 12\qquad \mathrm{(D)}\ \frac 23\qquad \mathrm{(E)}\ 1$" style="vertical-align: -14px" width="406" height="43" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>We can make use the of the trigonometric <a href="/wiki/index.php/Trigonometric_identities#Pythagorean_Identities" title="Trigonometric identities">Pythagorean identities</a>: square both equations and add them up:
</p>
<div style="text-align:center;"><img src="//latex.artofproblemsolving.com/5/a/6/5a6d5de20ab0bc919b7a7858544f89155541d275.png" class="latex" alt="$\sin^2 a + \sin^2 b + 2\sin a \sin b + \cos^2 a + \cos^2 b + 2\cos a \cos b = \frac{5}{3} + 1$" style="vertical-align: -12px" width="519" height="37" /><br /><img src="//latex.artofproblemsolving.com/f/2/f/f2f7e9b21268fc2f419dabe37d0440e8e82c6ed3.png" class="latex" alt="$2 + 2\sin a \sin b + 2\cos a \cos b = \frac{8}{3}$" style="vertical-align: -12px" width="260" height="37" /><br /><img src="//latex.artofproblemsolving.com/6/f/6/6f689d46f91ed11c75ff9271ff4741e35ba035df.png" class="latex" alt="$2(\cos a \cos b + \sin a \sin b) = \frac{2}{3}$" style="vertical-align: -12px" width="227" height="37" /></div>
<p>This is just the cosine difference identity, which simplifies to <img src="//latex.artofproblemsolving.com/2/e/7/2e7dd65f67e7babbefe32f7298be14e86d64ec45.png" class="latex" alt="$\cos (a - b) = \frac{1}{3} \Longrightarrow \mathrm{(B)}$" style="vertical-align: -12px" width="180" height="37" />
</p>
