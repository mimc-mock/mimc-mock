<p>The numbers 1, 2, 3, 4, 5 are to be arranged in a circle. An arrangement is <img src="//latex.artofproblemsolving.com/0/0/7/007141660ad610cb926f64c795d09f379a525e04.png" class="latex" alt="$bad$" width="27" height="12" /> if it is not true that for every <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> from <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" /> to <img src="//latex.artofproblemsolving.com/f/1/9/f1965fae079a9ba2c0726c307070c2355dfcb213.png" class="latex" alt="$15$" style="vertical-align: 0px" width="17" height="13" /> one can find a subset of the numbers that appear consecutively on the circle that sum to <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" />. Arrangements that differ only by a rotation or a reflection are considered the same. How many different bad arrangements are there?
</p><p><img src="//latex.artofproblemsolving.com/3/1/0/3107145add6467b53856f3368b15266721dd3b92.png" class="latex" alt="$\textbf {(A) } 1 \qquad \textbf {(B) } 2 \qquad \textbf {(C) } 3 \qquad \textbf {(D) } 4 \qquad \textbf {(E) } 5$" style="vertical-align: -4px" width="353" height="17" />
</p>
<h2><span class="mw-headline" id="Solutions">Solutions</span></h2>
<h3><span class="mw-headline" id="Solution_1">Solution 1</span></h3>
<p>We see that there are <img src="//latex.artofproblemsolving.com/2/9/3/2933f407c21ceb3f3ec8370f43cd5b939250ffca.png" class="latex" alt="$5!$" width="12" height="13" /> total ways to arrange the numbers. However, we can always rotate these numbers so that, for example, the number 1 is always at the top of the circle. Thus, there are only <img src="//latex.artofproblemsolving.com/2/d/a/2da866442e69c040daf44260b4728d37e98de313.png" class="latex" alt="$4!$" style="vertical-align: 0px" width="12" height="13" /> ways under rotation, which is not difficult to list out. We systematically list out all <img src="//latex.artofproblemsolving.com/e/2/9/e290f2cba0ba0e11c0b91630da15a408da2fcf1c.png" class="latex" alt="$24$" style="vertical-align: 0px" width="18" height="12" /> cases. 
</p><p>Now, we must examine if they satisfy the conditions. We can see that by choosing one number at a time, we can always obtain subsets with sums 1, 2, 3, 4, and 5. By choosing the full circle, we can obtain 15. By choosing everything except for 1, 2, 3, 4, and 5, we can obtain subsets with sums of 10, 11, 12, 13, and 14. 
</p><p>This means that we now only need to check for 6, 7, 8, and 9. However, once we have found a set summing to 6, we can choose everything else and obtain a set summing to 9, and similarly for 7 and 8. Thus, we only need to check each case for whether or not we can obtain 6 or 7. 
</p><p>We find that there are only 4 arrangements that satisfy these conditions. However, each of these is a reflection of another. We divide by 2 for these reflections to obtain a final answer of <img src="//latex.artofproblemsolving.com/6/c/0/6c04c9b0a66734ae63a0bb81d39d4247e4ad560c.png" class="latex" alt="$\boxed{\textbf {(B) }2}$" style="vertical-align: -10px" width="55" height="28" />.
</p>
