<p>Let <img src="//latex.artofproblemsolving.com/2/3/6/2363d70c4ce423ea23c71fa750e064ccd2be6f2b.png" class="latex" alt="$f_0(x)=x+|x-100|-|x+100|$" style="vertical-align: -4px" width="257" height="18" />, and for <img src="//latex.artofproblemsolving.com/9/c/c/9ccd696a5d0cbab8337ceb8a4cdfa202a54a6eae.png" class="latex" alt="$n\geq 1$" style="vertical-align: -2px" width="43" height="14" />, let <img src="//latex.artofproblemsolving.com/1/8/9/189e1d4544e54606ff57245e567518abb7bc07bc.png" class="latex" alt="$f_n(x)=|f_{n-1}(x)|-1$" style="vertical-align: -4px" width="166" height="18" />.  For how many values of <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> is <img src="//latex.artofproblemsolving.com/3/2/2/322f5e1a015ad9faae91ad85c818799e184d8d5a.png" class="latex" alt="$f_{100}(x)=0$" style="vertical-align: -4px" width="87" height="18" />?
</p><p><img src="//latex.artofproblemsolving.com/3/0/f/30ff65b595a7db51815e8a885476c0f483fff607.png" class="latex" alt="$\textbf{(A) }299\qquad \textbf{(B) }300\qquad \textbf{(C) }301\qquad \textbf{(D) }302\qquad \textbf{(E) }303\qquad$" style="vertical-align: -4px" width="443" height="17" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>1. Draw the graph of <img src="//latex.artofproblemsolving.com/c/f/2/cf2a7aed8e14bfdbd4184d2ad8b79d1c8187cf71.png" class="latex" alt="$f_0(x)$" style="vertical-align: -4px" width="39" height="18" /> by dividing the domain into three parts.
<img src="//latex.artofproblemsolving.com/d/4/6/d46ed802921cb9890bad75d864d89c4458aeb756.png" class="latexcenter" alt="[asy] unitsize(0);  int w = 250; int h = 125;  xaxis(-w,w,Ticks(100.0),Arrows); yaxis(-h,h,Ticks(100.0),Arrows);  draw((-100,-h)--(-100,h),dashed); draw((100,-h)--(100,h),dashed);  real f0(real x) { return x + abs(x-100) - abs(x+100); } draw(graph(f0,-w,w),Arrows); label(&quot;$f_0$&quot;,(-w,f0(-w)),W); [/asy]" width="668" height="322" /> 
</p><p>2. Apply the recursive rule a few times to find the pattern.
</p><p><b>Note:</b> <img src="//latex.artofproblemsolving.com/b/3/5/b3571b80083849d2f9cb950f1bdd60ff3ea82298.png" class="latex" alt="$f_n(x) = |f_{n-1}(x)| - 10$" style="vertical-align: -5px" width="179" height="20" /> is used to enlarge the difference, but the reasoning is the same.
<img src="//latex.artofproblemsolving.com/1/e/9/1e945622bdb3aa7f664a2d8a9722284a768b4940.png" class="latexcenter" alt="[asy] unitsize(0);  int w = 350; int h = 125;  xaxis(-w,w,Ticks(100.0),Arrows); yaxis(-h,h,Ticks(100.0),Arrows);  draw((-100,-h)--(-100,h),dashed); draw((100,-h)--(100,h),dashed);  int s = 10;  real f0(real x) { return x + abs(x-100) - abs(x+100); } real f(real x, int k) { real a = abs(f0(x))-s*k; return a &gt;= -s&#160;? a&#160;: k%2 == 0&#160;? abs(x%(2*s)-s)-s&#160;: -abs(x%(2*s)-s); }  real f1(real x) { return f(x,1); } real f2(real x) { return f(x,2); } real f3(real x) { return f(x,3); } real f4(real x) { return f(x,4); }  draw(graph(f0,-w,w,w*2#s),Arrows); draw(graph(f1,-w,w,w*2#s),red,Arrows); draw(graph(f2,-w,w,w*2#s),orange,Arrows); draw(graph(f3,-w,w,w*2#s),lightolive,Arrows);  label(&quot;$f_0$&quot;,(-w,f0(-w)),W); label(&quot;$f_1$&quot;,(-w,f1(-w)),NW,red); label(&quot;$f_2$&quot;,(-w,f2(-w)),W,orange); label(&quot;$f_3$&quot;,(-w,f3(-w)),SW,lightolive); [/asy]" width="668" height="302" />
</p><p>3. Extrapolate to <img src="//latex.artofproblemsolving.com/7/6/2/762528c54833c81f3398803e5b0fb51b07eca52a.png" class="latex" alt="$f_{100}$" style="vertical-align: -3px" width="28" height="16" />. Notice that the summits start <img src="//latex.artofproblemsolving.com/e/5/9/e59e2c6e83eb78cca610a5fd4070ae01c8d4ae60.png" class="latex" alt="$100$" style="vertical-align: 0px" width="26" height="13" /> away from <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" /> and get <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" /> closer each iteration, so they reach <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" /> exactly at <img src="//latex.artofproblemsolving.com/7/6/2/762528c54833c81f3398803e5b0fb51b07eca52a.png" class="latex" alt="$f_{100}$" style="vertical-align: -3px" width="28" height="16" />.
<img src="//latex.artofproblemsolving.com/c/c/c/ccce360f042201e7edd30341d03920382658764d.png" class="latexcenter" alt="[asy] unitsize(0);  int w = 350; int h = 125;  xaxis(-w,w,Ticks(100.0),Arrows); yaxis(-h,h,Ticks(100.0),Arrows);  draw((-100,-h)--(-100,h),dashed); draw((100,-h)--(100,h),dashed);  int s = 10;  real f0(real x) { return x + abs(x-100) - abs(x+100); } real f(real x, int k) { real a = abs(f0(x))-s*k; return a &gt;= -s&#160;? a&#160;: k%2 == 0&#160;? abs(x%(2*s)-s)-s&#160;: -abs(x%(2*s)-s); }  real f1(real x) { return f(x,1); } real f2(real x) { return f(x,2); } real f3(real x) { return f(x,3); } real f4(real x) { return f(x,4); } real f98(real x) { return f(x,100#s-2); } real f99(real x) { return f(x,100#s-1); } real f100(real x) { return f(x,100#s); }  draw(graph(f0,-w,w,w*2#s),Arrows); draw(graph(f1,-w,w,w*2#s),red,Arrows); draw(graph(f2,-w,w,w*2#s),orange,Arrows); draw(graph(f3,-w,w,w*2#s),lightolive,Arrows); draw(graph(f98,-w,w,w*2#s),heavygreen,Arrows); draw(graph(f99,-w,w,w*2#s),blue,Arrows); draw(graph(f100,-w,w,w*2#s),purple,Arrows);  label(&quot;$f_0$&quot;,(-w,f0(-w)),W); label(&quot;$f_1$&quot;,(-w,f1(-w)),NW,red); label(&quot;$f_2$&quot;,(-w,f2(-w)),W,orange); label(&quot;$f_3$&quot;,(-w,f3(-w)),SW,lightolive); label(&quot;$f_{98}$&quot;,(-w,f98(-w)),NW,heavygreen); label(&quot;$f_{99}$&quot;,(-w,f99(-w)),W,blue); label(&quot;$f_{100}$&quot;,(-w,f100(-w)),SW,purple); [/asy]" width="668" height="298" />
</p><p><img src="//latex.artofproblemsolving.com/e/7/8/e78c570c393090f53c72a03d27ab97d03ee5fa96.png" class="latex" alt="$f_{100}(x)$" style="vertical-align: -4px" width="52" height="18" /> reaches <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" /> at <img src="//latex.artofproblemsolving.com/8/6/8/868bd609188f5da1d42caa671a3d32609bdf947e.png" class="latex" alt="$x = -300$" style="vertical-align: -2px" width="78" height="15" />, then zigzags between <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" /> and <img src="//latex.artofproblemsolving.com/b/a/e/bae5aba07d37ff6ff813107e76260fb31ad5794e.png" class="latex" alt="$-1$" style="vertical-align: 0px" width="22" height="12" />, hitting <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" /> at every even <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" />, before leaving <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" /> at <img src="//latex.artofproblemsolving.com/7/c/1/7c109e16aee0eb782eb0803775d4d217b87ecc95.png" class="latex" alt="$x = 300$" width="61" height="12" />.
</p><p>This means that <img src="//latex.artofproblemsolving.com/e/a/3/ea33ecdb938550f2d47427ac23673a4bbc8b26d8.png" class="latex" alt="$f_{100}(x) = 0$" style="vertical-align: -5px" width="89" height="20" /> at all even <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> where <img src="//latex.artofproblemsolving.com/f/8/2/f82c70667188da6ae8f73a47249d1dcd8548d184.png" class="latex" alt="$-300 \le x \le 300$" style="vertical-align: -2px" width="127" height="15" />. This is a <img src="//latex.artofproblemsolving.com/1/0/a/10a590444099e67335096609a93d85df88282653.png" class="latex" alt="$701$" style="vertical-align: 0px" width="26" height="13" />-integer odd-size range with even numbers at the endpoints, so just over half of the integers are even, or <img src="//latex.artofproblemsolving.com/d/7/0/d7006dea2b962c09e35630697222c7e4289b4ce3.png" class="latex" alt="$\frac{701+1}{2} = \boxed{\textbf{(C) }301}$" style="vertical-align: -13px" width="162" height="38" />.
(Revised by Flamedragon &amp; Jason,C &amp; <a href="/wiki/index.php/User:Emerald_block" title="User:Emerald block">emerald_block</a>)
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>First, notice that the recursive rule moves the current value <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" /> closer to <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" />. Upon reaching <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" />, it alternates between <img src="//latex.artofproblemsolving.com/b/a/e/bae5aba07d37ff6ff813107e76260fb31ad5794e.png" class="latex" alt="$-1$" style="vertical-align: 0px" width="22" height="12" /> and <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" />. This means that <img src="//latex.artofproblemsolving.com/e/a/3/ea33ecdb938550f2d47427ac23673a4bbc8b26d8.png" class="latex" alt="$f_{100}(x) = 0$" style="vertical-align: -5px" width="89" height="20" /> exactly when <img src="//latex.artofproblemsolving.com/e/3/4/e3455ecc658712973239a2271bc1e86caea11963.png" class="latex" alt="$|f_0(x)| \le 100$" style="vertical-align: -5px" width="104" height="20" /> (to reach <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" /> in time) and <img src="//latex.artofproblemsolving.com/c/f/2/cf2a7aed8e14bfdbd4184d2ad8b79d1c8187cf71.png" class="latex" alt="$f_0(x)$" style="vertical-align: -4px" width="39" height="18" /> is even (so <img src="//latex.artofproblemsolving.com/4/a/3/4a34f0d907c5c1e6d0e62b70cb0824ddb39c69c3.png" class="latex" alt="$f_{100}(x) \ne -1$" style="vertical-align: -5px" width="103" height="20" />).
</p><p>Casework each part of <img src="//latex.artofproblemsolving.com/c/f/2/cf2a7aed8e14bfdbd4184d2ad8b79d1c8187cf71.png" class="latex" alt="$f_0(x)$" style="vertical-align: -4px" width="39" height="18" /> (where the expressions in the absolute values do not change sign):
<img src="//latex.artofproblemsolving.com/f/a/5/fa5f3d1f7eb2c9f2f374b25bf2077ee1b8bdb4fe.png" class="latexcenter" alt="\[x \le -100 \implies f_0(x) = x-(x-100)+(x+100) = x+200\]" width="476" height="18" />
so even <img src="//latex.artofproblemsolving.com/d/f/6/df6ae6e931ea07c6bf8ac72a583d9692fd8b50d1.png" class="latex" alt="$-300 \le x \le -100$" style="vertical-align: -3px" width="144" height="16" /> work.
<img src="//latex.artofproblemsolving.com/3/1/6/316cbc48789ad4d5e5fa2413952e57906e075829.png" class="latexcenter" alt="\[-100 \le x \le 100 \implies f_0(x) = x-(x-100)-(x+100) = -x\]" width="478" height="18" />
so even <img src="//latex.artofproblemsolving.com/8/4/f/84f0eedd6a81073bd009de6d79532694636539a8.png" class="latex" alt="$-100 \le x \le 100$" style="vertical-align: -2px" width="127" height="15" /> work.
<img src="//latex.artofproblemsolving.com/8/2/c/82ced3d879d47a831bc23c8c05cba900e574099a.png" class="latexcenter" alt="\[100 \le x \implies f_0(x) = x+(x-100)-(x+100) = x-200\]" width="461" height="18" />
so even <img src="//latex.artofproblemsolving.com/8/2/6/8263807b4fbd817a925ebe6d3063bec09e0acb5c.png" class="latex" alt="$100 \le x \le 300$" style="vertical-align: -3px" width="115" height="16" /> work.
</p><p>Putting these together, all even <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> where <img src="//latex.artofproblemsolving.com/f/8/2/f82c70667188da6ae8f73a47249d1dcd8548d184.png" class="latex" alt="$-300 \le x \le 300$" style="vertical-align: -2px" width="127" height="15" /> work. So, the answer is <img src="//latex.artofproblemsolving.com/a/6/1/a6169077b0ef53ff4c38a56f6493858c10efdc07.png" class="latex" alt="$2\cdot150+1 = \boxed{\textbf{(C)}\ 301}$" style="vertical-align: -11px" width="180" height="30" />.
~revised by <a href="/wiki/index.php/User:Emerald_block" title="User:Emerald block">emerald_block</a>
</p>
