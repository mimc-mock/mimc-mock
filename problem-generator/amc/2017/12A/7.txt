<p>Define a function on the positive integers recursively by <img src="//latex.artofproblemsolving.com/e/b/e/ebe113a800b519c0a4b80803d4fa2be2546015f4.png" class="latex" alt="$f(1) = 2$" style="vertical-align: -4px" width="67" height="18" />, <img src="//latex.artofproblemsolving.com/b/c/4/bc4f22f633f9490f2e4afd3b2ecace6e45aab63d.png" class="latex" alt="$f(n) = f(n-1) + 1$" style="vertical-align: -4px" width="157" height="18" /> if <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> is even, and <img src="//latex.artofproblemsolving.com/e/a/3/ea3d473063509951b57269a3a82871d1832bd051.png" class="latex" alt="$f(n) = f(n-2) + 2$" style="vertical-align: -4px" width="158" height="18" /> if <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> is odd and greater than <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" />. What is <img src="//latex.artofproblemsolving.com/a/4/a/a4a29ca4e2bcde44d859e40455cace721c96c7e6.png" class="latex" alt="$f(2017)$" style="vertical-align: -4px" width="59" height="18" />?
</p><p><img src="//latex.artofproblemsolving.com/6/3/e/63e48a67ce512bc639f4b4995c859876e6b15d05.png" class="latex" alt="$\textbf{(A)}\ 2017 \qquad\textbf{(B)}\ 2018 \qquad\textbf{(C)}\ 4034 \qquad\textbf{(D)}\ 4035 \qquad\textbf{(E)}\ 4036$" style="vertical-align: -4px" width="484" height="17" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>This is a recursive function, which means the function refers back to itself to calculate subsequent terms. To solve this, we must identify the base case, <img src="//latex.artofproblemsolving.com/e/4/a/e4a16b9dffeb3757d936f3a92e8f8f85c2bf6745.png" class="latex" alt="$f(1)=2$" style="vertical-align: -4px" width="67" height="18" />. We also know that when <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> is odd, <img src="//latex.artofproblemsolving.com/0/e/c/0ece07e7f551b5cac10cf200fad4e4b9190ff924.png" class="latex" alt="$f(n)=f(n-2)+2$" style="vertical-align: -4px" width="158" height="18" />. Thus we know that <img src="//latex.artofproblemsolving.com/d/8/6/d864a2725e33895e3b600c09e2383a6ce191a2db.png" class="latex" alt="$f(2017)=f(2015)+2$" style="vertical-align: -4px" width="177" height="18" />. Thus we know that n will always be odd in the recursion of <img src="//latex.artofproblemsolving.com/a/4/a/a4a29ca4e2bcde44d859e40455cace721c96c7e6.png" class="latex" alt="$f(2017)$" style="vertical-align: -4px" width="59" height="18" />, and we add <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" /> each recursive cycle, which there are <img src="//latex.artofproblemsolving.com/f/2/3/f233b4bd39c27b96921ff522ca6f43eff8813674.png" class="latex" alt="$1008$" style="vertical-align: 0px" width="35" height="13" /> of. Thus the answer is <img src="//latex.artofproblemsolving.com/a/2/d/a2d2febfdc545560a1d845652c35a6042c0476f6.png" class="latex" alt="$1008*2+2=2018$" style="vertical-align: -1px" width="153" height="14" />, which is answer
<img src="//latex.artofproblemsolving.com/a/2/1/a2194b81820e8380be586c1564c3f5d03d5c100d.png" class="latex" alt="$\boxed{\textbf{(B)}}$" style="vertical-align: -10px" width="40" height="28" />.
Note that when you write out a few numbers, you find that <img src="//latex.artofproblemsolving.com/b/6/7/b679f174299d13f6d2440ca60539e061f9d5e822.png" class="latex" alt="$f(n)=n+1$" style="vertical-align: -4px" width="101" height="18" /> for any <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" />, so <img src="//latex.artofproblemsolving.com/0/a/d/0ad4a5c84bac21bd38bcf5ff0b134cac73f78ed5.png" class="latex" alt="$f(2017)=2018$" style="vertical-align: -4px" width="121" height="18" />
</p>
