<p>The data set <img src="//latex.artofproblemsolving.com/c/1/f/c1f79e4f4be0f8520c3451e684ea03f3f42d9817.png" class="latex" alt="$[6, 19, 33, 33, 39, 41, 41, 43, 51, 57]$" style="vertical-align: -5px" width="253" height="18" /> has median <img src="//latex.artofproblemsolving.com/a/e/f/aefcf57aef873e70c66e39f720a01468b25f5615.png" class="latex" alt="$Q_2 = 40$" style="vertical-align: -3px" width="63" height="16" />, first quartile <img src="//latex.artofproblemsolving.com/6/4/b/64b72792554fa9f86fde1abed5813ee7f59e31eb.png" class="latex" alt="$Q_1 = 33$" style="vertical-align: -3px" width="63" height="16" />, and third quartile <img src="//latex.artofproblemsolving.com/f/8/c/f8cb7ab0723826ef7d46f766907cf590c4e032c6.png" class="latex" alt="$Q_3 = 43$" style="vertical-align: -3px" width="63" height="16" />. An outlier in a data set is a value that is more than <img src="//latex.artofproblemsolving.com/1/5/7/15788e198a53ced87268aea78fcde68a84de92c0.png" class="latex" alt="$1.5$" style="vertical-align: 0px" width="22" height="13" /> times the interquartile range below the first quartle (<img src="//latex.artofproblemsolving.com/c/1/6/c16e423c8f499e0bef8c20078b04126aeb9417bc.png" class="latex" alt="$Q_1$" style="vertical-align: -3px" width="20" height="16" />) or more than <img src="//latex.artofproblemsolving.com/1/5/7/15788e198a53ced87268aea78fcde68a84de92c0.png" class="latex" alt="$1.5$" style="vertical-align: 0px" width="22" height="13" /> times the interquartile range above the third quartile (<img src="//latex.artofproblemsolving.com/1/d/2/1d2aa7380ca1674e2159cbb686c162838f61a2a4.png" class="latex" alt="$Q_3$" style="vertical-align: -3px" width="20" height="16" />), where the interquartile range is defined as <img src="//latex.artofproblemsolving.com/b/1/3/b13e048358c8751b3c42253b1a64bf99cbcdf5f9.png" class="latex" alt="$Q_3 - Q_1$" style="vertical-align: -3px" width="64" height="16" />. How many outliers does this data set have?
</p><p><img src="//latex.artofproblemsolving.com/3/a/d/3ad07ba45d9a4dcd46fbd971320abb2585f7ce0e.png" class="latex" alt="$\textbf{(A)}\ 0\qquad\textbf{(B)}\ 1\qquad\textbf{(C)}\ 2\qquad\textbf{(D)}\ 3\qquad\textbf{(E)}\ 4$" style="vertical-align: -4px" width="349" height="17" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>The interquartile range is defined as <img src="//latex.artofproblemsolving.com/2/e/b/2eb6f7c6a1445d3617a7f996090227ae16138e92.png" class="latex" alt="$Q3 - Q1$" style="vertical-align: -3px" width="68" height="16" />, which is <img src="//latex.artofproblemsolving.com/5/d/3/5d3be9a267c4836e9f773b22cb46f2c621ae1f9d.png" class="latex" alt="$43 - 33 = 10$" style="vertical-align: 0px" width="100" height="13" />. <img src="//latex.artofproblemsolving.com/1/5/7/15788e198a53ced87268aea78fcde68a84de92c0.png" class="latex" alt="$1.5$" style="vertical-align: 0px" width="22" height="13" /> times this value is <img src="//latex.artofproblemsolving.com/f/1/9/f1965fae079a9ba2c0726c307070c2355dfcb213.png" class="latex" alt="$15$" style="vertical-align: 0px" width="17" height="13" />, so all values more than <img src="//latex.artofproblemsolving.com/f/1/9/f1965fae079a9ba2c0726c307070c2355dfcb213.png" class="latex" alt="$15$" style="vertical-align: 0px" width="17" height="13" /> below <img src="//latex.artofproblemsolving.com/b/7/d/b7dcfb8e9ee6ef907801ecdc34fb58b444913763.png" class="latex" alt="$Q1$" style="vertical-align: -3px" width="22" height="16" /> = <img src="//latex.artofproblemsolving.com/4/4/f/44f42857a6f506c19528497ec7ebd1eba9bfe728.png" class="latex" alt="$33 - 15 = 18$" style="vertical-align: 0px" width="100" height="13" /> is an outlier. The only one that fits this is <img src="//latex.artofproblemsolving.com/6/0/1/601a7806cbfad68196c43a4665871f8c3186802e.png" class="latex" alt="$6$" width="8" height="12" />. All values more than <img src="//latex.artofproblemsolving.com/f/1/9/f1965fae079a9ba2c0726c307070c2355dfcb213.png" class="latex" alt="$15$" style="vertical-align: 0px" width="17" height="13" /> above <img src="//latex.artofproblemsolving.com/c/c/c/ccc47477f0451bde684ffbb1582270a7e484e10e.png" class="latex" alt="$Q3 = 43 + 15 = 58$" style="vertical-align: -4px" width="151" height="18" /> are also outliers, of which there are none so there is only <img src="//latex.artofproblemsolving.com/7/8/2/78239690c6863df7c4c76cb7408bfb720373cb62.png" class="latex" alt="$\boxed{\textbf{(B) 1}}$" style="vertical-align: -10px" width="56" height="28" /> outlier in total.
</p>
