<p>Which of the following equations does NOT have a solution?
</p><p><img src="//latex.artofproblemsolving.com/c/5/c/c5c50664ce09ee339598425619c37e9f17a35651.png" class="latex" alt="$\text{(A)}\:(x+7)^2=0$" style="vertical-align: -4px" width="125" height="19" />
</p><p><img src="//latex.artofproblemsolving.com/b/6/6/b66f5b12cf1022b441699e00181e482a08ca1cbb.png" class="latex" alt="$\text{(B)}\:|-3x|+5=0$" style="vertical-align: -4px" width="144" height="18" />
</p><p><img src="//latex.artofproblemsolving.com/c/8/0/c80d05e3ffccd51cd8db94d196312246a9eb8ed5.png" class="latex" alt="$\text{(C)}\:\sqrt{-x}-2=0$" style="vertical-align: -4px" width="132" height="19" />
</p><p><img src="//latex.artofproblemsolving.com/7/e/b/7ebd85e123d9ba5a7f4f0803e3bacec54c9c41e0.png" class="latex" alt="$\text{(D)}\:\sqrt{x}-8=0$" style="vertical-align: -4px" width="119" height="19" />
</p><p><img src="//latex.artofproblemsolving.com/7/4/7/74702a4bb0654417ded3c6c68d31a6a771d84d28.png" class="latex" alt="$\text{(E)}\:|-3x|-4=0$" style="vertical-align: -4px" width="144" height="18" />
</p><p><br />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p><img src="//latex.artofproblemsolving.com/b/9/d/b9d4585d25446873cb275259748b872107c94b83.png" class="latex" alt="$|-3x|+5=0$" style="vertical-align: -4px" width="116" height="18" /> has no solution because absolute values output numbers greater than or equal to 0 and this equation implies that the absolute value could output a negative.
</p><p>Further:
<img src="//latex.artofproblemsolving.com/2/a/b/2ab8c59f298443c24a48306f4cacbd01b2f2bb7a.png" class="latex" alt="$(x+7)^2 = 0$" style="vertical-align: -4px" width="96" height="19" /> is true for <img src="//latex.artofproblemsolving.com/6/9/3/6939833425734b3d8fa4a0112e06f226ff67b15f.png" class="latex" alt="$x = -7$" width="57" height="12" /> 
</p><p><img src="//latex.artofproblemsolving.com/8/a/1/8a1068da21bdfc4fbe5ccebdc3378f6efcb99d22.png" class="latex" alt="$\sqrt{-x}-2=0$" style="vertical-align: -3px" width="103" height="18" /> is true for <img src="//latex.artofproblemsolving.com/b/7/d/b7d924f912a4072521e9bb4c86d868591e01d0f5.png" class="latex" alt="$x = -4$" style="vertical-align: 0px" width="57" height="12" />
</p><p><img src="//latex.artofproblemsolving.com/f/8/7/f8797d90aaf5c82eac2d06514229d6b25b9f0cef.png" class="latex" alt="$\sqrt{x}-8=0$" style="vertical-align: -3px" width="89" height="18" /> is true for <img src="//latex.artofproblemsolving.com/0/6/5/065a6008b7a6b06f5d8b4e45b9be22d3eac01ba9.png" class="latex" alt="$x = 64$" style="vertical-align: 0px" width="52" height="13" />
</p><p><img src="//latex.artofproblemsolving.com/0/8/d/08dc580f0d16cfe7c5fccc9eebe61109e014777c.png" class="latex" alt="$|-3x|-4=0$" style="vertical-align: -4px" width="116" height="18" /> is true for <img src="//latex.artofproblemsolving.com/5/b/c/5bc416a63f2959766c39928506373ccdf33c5cee.png" class="latex" alt="$x = \frac{4}{3}, -\frac{4}{3}$" style="vertical-align: -12px" width="81" height="37" />
</p><p>Therefore, the answer is <img src="//latex.artofproblemsolving.com/c/7/2/c72a08c9de59fb963e17912f4b2127a20340346a.png" class="latex" alt="$\boxed{\mathrm{(B)}}$" style="vertical-align: -10px" width="38" height="29" />.
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>Instead of solving, we can just categorize and solve.
</p><p>Section 1:
This contains A,C,D as they are all squares or square roots.
From skimming, we can get an answer as maybe C
</p><p>Section 2:
This contains B and E
From skimming we can get we can get answer as maybe B
</p><p>Now we can analyze and we see <img src="//latex.artofproblemsolving.com/5/2/9/52969fdeeb90f7e24cd2a2f1a6cfe124776fee2a.png" class="latex" alt="$-x$" width="24" height="8" /> can become <img src="//latex.artofproblemsolving.com/2/6/e/26eeb5258ca5099acf8fe96b2a1049c48c89a5e6.png" class="latex" alt="$x$" width="10" height="8" /> if <img src="//latex.artofproblemsolving.com/5/2/c/52c6f440a0a502b9742ddfe66649ed66b834d8ef.png" class="latex" alt="$x=-y$" style="vertical-align: -3px" width="57" height="11" /> and absolute value inequalities cannot be negative, so the answer is <img src="//latex.artofproblemsolving.com/c/7/2/c72a08c9de59fb963e17912f4b2127a20340346a.png" class="latex" alt="$\boxed{\mathrm{(B)}}$" style="vertical-align: -10px" width="38" height="29" />
</p>
