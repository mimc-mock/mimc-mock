<p>For every positive integer <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" />, let <img src="//latex.artofproblemsolving.com/1/9/3/1936697fee7a7eae7e12e03d1a7f8d59a1481069.png" class="latex" alt="$\text{mod}_5 (n)$" style="vertical-align: -4px" width="63" height="18" /> be the remainder obtained when <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> is divided by 5. Define a function <img src="//latex.artofproblemsolving.com/5/8/9/589de9edefcc1e5f1e87c7d997162287ea7cf573.png" class="latex" alt="$f: \{0,1,2,3,\dots\} \times \{0,1,2,3,4\} \to \{0,1,2,3,4\}$" style="vertical-align: -4px" width="378" height="18" /> recursively as follows:
</p><p><img src="//latex.artofproblemsolving.com/1/e/f/1efa0d34cf7ae66e852b1687a71b18515cda058b.png" class="latexcenter" alt="\[f(i,j) = \begin{cases}\text{mod}_5 (j+1) &amp; \text{ if } i = 0 \text{ and } 0 \le j \le 4 \text{,}\\ f(i-1,1) &amp; \text{ if } i \ge 1 \text{ and } j = 0 \text{, and} \\ f(i-1, f(i,j-1)) &amp; \text{ if } i \ge 1 \text{ and } 1 \le j \le 4. \end{cases}\]" width="430" height="74" />  
</p><p>What is <img src="//latex.artofproblemsolving.com/5/8/a/58abcab7f44397366f1c1962289eef3e0ac1c5c2.png" class="latex" alt="$f(2015,2)$" style="vertical-align: -4px" width="77" height="18" />?
</p><p><img src="//latex.artofproblemsolving.com/4/4/a/44a0eed8bcaa15ee1b45f9a2eb6c1bad801f7e5d.png" class="latex" alt="$\textbf{(A)}\; 0 \qquad\textbf{(B)}\; 1 \qquad\textbf{(C)}\; 2 \qquad\textbf{(D)}\; 3 \qquad\textbf{(E)}\; 4$" style="vertical-align: -4px" width="346" height="17" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>Simply take some time to draw a table of values of <img src="//latex.artofproblemsolving.com/5/2/9/529a30a91b68ec2b50b539e622e58bd62dff66ab.png" class="latex" alt="$f(i,j)$" style="vertical-align: -4px" width="46" height="18" /> for the first few values of <img src="//latex.artofproblemsolving.com/3/4/8/34857b3ba74ce5cd8607f3ebd23e9015908ada71.png" class="latex" alt="$i$" width="5" height="12" />:
</p><p><img src="//latex.artofproblemsolving.com/8/6/a/86a0e49932036b4f8ac2202543492e3c25dc10c6.png" class="latexcenter" alt="\[\begin{array}{|c || c | c | c | c | c |} \hline i \text{\ \textbackslash\ } j &amp; 0 &amp; 1 &amp; 2 &amp; 3 &amp; 4\\ \hline\hline 0 &amp; 1 &amp; 2 &amp; 3 &amp; 4 &amp; 0\\ \hline 1 &amp; 2 &amp; 3 &amp; 4 &amp; 0 &amp; 1\\ \hline 2 &amp; 3 &amp; 0 &amp; 2 &amp; 4 &amp; 1\\ \hline 3 &amp; 0 &amp; 3 &amp; 4 &amp; 1 &amp; 0\\ \hline 4 &amp; 3 &amp; 1 &amp; 3 &amp; 1 &amp; 3\\ \hline 5 &amp; 1 &amp; 1 &amp; 1 &amp; 1 &amp; 1\\ \hline \end{array}\]" width="183" height="148" />
</p><p>Now we claim that for <img src="//latex.artofproblemsolving.com/e/b/f/ebfed880c3d5c165227b2e8fe88fb927f9d23c49.png" class="latex" alt="$i \ge 5$" style="vertical-align: -2px" width="39" height="15" />, <img src="//latex.artofproblemsolving.com/5/c/1/5c105f93316dc8f640f4a5afd91e63bc9708983d.png" class="latex" alt="$f(i,j) = 1$" style="vertical-align: -4px" width="80" height="18" /> for all values <img src="//latex.artofproblemsolving.com/0/f/d/0fd3736cb4d4d972c317412726c771787e597021.png" class="latex" alt="$0 \le j \le 4$" style="vertical-align: -3px" width="75" height="16" />.  We will prove this by induction on <img src="//latex.artofproblemsolving.com/3/4/8/34857b3ba74ce5cd8607f3ebd23e9015908ada71.png" class="latex" alt="$i$" width="5" height="12" /> and <img src="//latex.artofproblemsolving.com/8/1/2/8122aa89ea6e80784c6513d22787ad86e36ad0cc.png" class="latex" alt="$j$" style="vertical-align: -3px" width="8" height="16" />.  The base cases for <img src="//latex.artofproblemsolving.com/c/1/2/c1236ac7c7cb59f16237e0e51cf9c626005edc95.png" class="latex" alt="$i = 5$" width="39" height="12" />, have already been proven.
</p><p>For our inductive step, we must show that for all valid values of <img src="//latex.artofproblemsolving.com/8/1/2/8122aa89ea6e80784c6513d22787ad86e36ad0cc.png" class="latex" alt="$j$" style="vertical-align: -3px" width="8" height="16" />, <img src="//latex.artofproblemsolving.com/d/4/5/d45798c5cc582a24ec33f0969225446515689be1.png" class="latex" alt="$f(i, j) = 1$" style="vertical-align: -4px" width="80" height="18" /> if for all valid values of <img src="//latex.artofproblemsolving.com/8/1/2/8122aa89ea6e80784c6513d22787ad86e36ad0cc.png" class="latex" alt="$j$" style="vertical-align: -3px" width="8" height="16" />, <img src="//latex.artofproblemsolving.com/0/0/b/00b938123df2a4a4aa219df4989b93980e76577e.png" class="latex" alt="$f(i - 1, j) = 1$" style="vertical-align: -4px" width="111" height="18" />.
</p><p>We prove this itself by induction on <img src="//latex.artofproblemsolving.com/8/1/2/8122aa89ea6e80784c6513d22787ad86e36ad0cc.png" class="latex" alt="$j$" style="vertical-align: -3px" width="8" height="16" />.  For the base case, <img src="//latex.artofproblemsolving.com/8/4/9/8491cdd7734fa4e048d0883176399057c4badc6f.png" class="latex" alt="$j=0$" style="vertical-align: -3px" width="42" height="16" />, <img src="//latex.artofproblemsolving.com/0/8/c/08cd3eb0f943745f280d2da111c067e34c2fa794.png" class="latex" alt="$f(i, 0) = f(i-1, 1) = 1$" style="vertical-align: -4px" width="184" height="18" />.  For the inductive step, we need <img src="//latex.artofproblemsolving.com/d/4/5/d45798c5cc582a24ec33f0969225446515689be1.png" class="latex" alt="$f(i, j) = 1$" style="vertical-align: -4px" width="80" height="18" /> if <img src="//latex.artofproblemsolving.com/e/2/2/e22ce95c781876100376cc4f18590e2130add384.png" class="latex" alt="$f(i, j-1) = 1$" style="vertical-align: -4px" width="111" height="18" />.  Then, <img src="//latex.artofproblemsolving.com/1/4/1/141df99291da1c07adccf1d8f521924696474781.png" class="latex" alt="$f(i, j) = f(i-1, f(i, j-1)).$" style="vertical-align: -4px" width="225" height="18" />  <img src="//latex.artofproblemsolving.com/e/2/2/e22ce95c781876100376cc4f18590e2130add384.png" class="latex" alt="$f(i, j-1) = 1$" style="vertical-align: -4px" width="111" height="18" /> by our inductive hypothesis from our inner induction and <img src="//latex.artofproblemsolving.com/f/a/c/facce027027ab8bc365f0578310843fea1008d6b.png" class="latex" alt="$f(i-1, 1) = 1$" style="vertical-align: -4px" width="112" height="18" /> from our outer inductive hypothesis.  Thus, <img src="//latex.artofproblemsolving.com/d/4/5/d45798c5cc582a24ec33f0969225446515689be1.png" class="latex" alt="$f(i, j) = 1$" style="vertical-align: -4px" width="80" height="18" />, completing the proof.
</p><p>It is now clear that for <img src="//latex.artofproblemsolving.com/e/b/f/ebfed880c3d5c165227b2e8fe88fb927f9d23c49.png" class="latex" alt="$i \ge 5$" style="vertical-align: -2px" width="39" height="15" />, <img src="//latex.artofproblemsolving.com/5/c/1/5c105f93316dc8f640f4a5afd91e63bc9708983d.png" class="latex" alt="$f(i,j) = 1$" style="vertical-align: -4px" width="80" height="18" /> for all values <img src="//latex.artofproblemsolving.com/0/f/d/0fd3736cb4d4d972c317412726c771787e597021.png" class="latex" alt="$0 \le j \le 4$" style="vertical-align: -3px" width="75" height="16" />.
</p><p>Thus, <img src="//latex.artofproblemsolving.com/d/1/d/d1d8df8db5eaa1a717dd79b9a229b5111b7ab548.png" class="latex" alt="$f(2015,2) = \boxed{\textbf{(B)} \; 1}$" style="vertical-align: -10px" width="157" height="28" />.
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>We are given that <img src="//latex.artofproblemsolving.com/8/1/b/81b442f685442974d8bda6a1083494d8a356698f.png" class="latexcenter" alt="\[f(0,n) \equiv n+1\pmod{5} .\]" width="204" height="18" />
Then, <img src="//latex.artofproblemsolving.com/8/b/d/8bd50446aa863ff3b05dd51b5b1e6c995fd34fa5.png" class="latex" alt="$f(1,n) = f(0, f(1, n-1)) \equiv f(1,n-1) + 1\pmod{5}$" style="vertical-align: -4px" width="414" height="18" />. Thus <img src="//latex.artofproblemsolving.com/1/a/f/1af8205d5a83c571986d8fe6b9621155f56b60a2.png" class="latex" alt="$f(1,n) \equiv n+f(1,0)\pmod{5}$" style="vertical-align: -4px" width="232" height="18" />. Since <img src="//latex.artofproblemsolving.com/9/1/1/911fae87120c413062055d18647ed45f9a5bb99f.png" class="latex" alt="$f(1,0)=f(0,1)=2$" style="vertical-align: -4px" width="159" height="18" />, we get 
<img src="//latex.artofproblemsolving.com/1/2/0/12061afac715c251d9be2ef12fbfbd75d5d98457.png" class="latexcenter" alt="\[f(1,n) \equiv n+2\pmod{5} .\]" width="204" height="18" />
Then, <img src="//latex.artofproblemsolving.com/0/0/6/00617cb95fd6ee271851a20379d339dbd67d8cd0.png" class="latex" alt="$f(2,n) = f(1, f(2, n-1)) \equiv f(2,n-1) + 2\pmod{5}$" style="vertical-align: -4px" width="414" height="18" />. Thus <img src="//latex.artofproblemsolving.com/e/b/3/eb340479ac9afff49377cc712ecdceb202c08c15.png" class="latex" alt="$f(2,n) \equiv 2n+f(2,0)\pmod{5}$" style="vertical-align: -4px" width="241" height="18" />. Since <img src="//latex.artofproblemsolving.com/8/f/3/8f3c7234fb76285ff6eca766393ad4b69235c12a.png" class="latex" alt="$f(2,0)=f(1,1)=3$" style="vertical-align: -4px" width="159" height="18" />, we get 
<img src="//latex.artofproblemsolving.com/4/c/4/4c4ed01c790a02086e7e3424af12b12b1e9362f5.png" class="latexcenter" alt="\[f(2,n) \equiv 2n+3\pmod{5} .\]" width="213" height="18" />
Now <img src="//latex.artofproblemsolving.com/2/9/5/295b906a1ed5162c3a741e8646b636b700f90581.png" class="latex" alt="$f(3,n) = f(2, f(3, n-1)) \equiv 2f(3,n-1) + 3\pmod{5}$" style="vertical-align: -4px" width="423" height="18" />. Thus 
<img src="//latex.artofproblemsolving.com/6/f/2/6f207ae5a390cce99fc8fd9cc3febd52595a2b0f.png" class="latexcenter" alt="\begin{align*}     f(3,n) &amp;\equiv 2f(3,n-1) + 3 &amp;\pmod{5} \\     2f(3,n-1) &amp;\equiv 2^2f(3,n-2) + 3\cdot 2 &amp;\pmod{5} \\     \vdots \qquad &amp;\quad\vdots \quad\qquad \vdots \qquad\qquad \vdots &amp;\\     2^{n-1}f(3,1) &amp;\equiv 2^nf(3,0) + 3\cdot 2^{n-1} &amp;\pmod{5}  \end{align*}" width="394" height="113" />
Adding them all up we get
<img src="//latex.artofproblemsolving.com/9/b/5/9b565242f8e6d4b0d4a2d1180220d37843de6ed8.png" class="latexcenter" alt="\[f(3,n) \equiv 3(2^n-1)\pmod{5} .\]" width="235" height="18" />
This means that <img src="//latex.artofproblemsolving.com/1/2/c/12c8fa0c3a5c370bc66c4c441d753556150955bf.png" class="latex" alt="$f(3,0)=0$" style="vertical-align: -4px" width="84" height="18" />, <img src="//latex.artofproblemsolving.com/5/b/7/5b78d5d99e7ed2ae253f13243957004e14062dc5.png" class="latex" alt="$f(3,1)=3$" style="vertical-align: -4px" width="84" height="18" />, <img src="//latex.artofproblemsolving.com/e/f/d/efd03cc70103c4eb2778ef4f640114d340ef459b.png" class="latex" alt="$f(3,2)=4$" style="vertical-align: -4px" width="84" height="18" />, <img src="//latex.artofproblemsolving.com/c/6/6/c66672b993c1d0fe582011f247b37be564ed8fdb.png" class="latex" alt="$f(3,3)=1$" style="vertical-align: -4px" width="83" height="18" />, and <img src="//latex.artofproblemsolving.com/6/a/5/6a55db8469ea2924de0753cbf29bf3f67cabb9e1.png" class="latex" alt="$f(3,4)=0$" style="vertical-align: -4px" width="84" height="18" />. Thus <img src="//latex.artofproblemsolving.com/4/3/7/437143471eaf49c94c07d37ee8d950285f6dcc62.png" class="latex" alt="$f(3,n)$" style="vertical-align: -4px" width="51" height="18" /> never takes the value 2. 
</p><p>Since <img src="//latex.artofproblemsolving.com/6/b/1/6b1943b2ed44bdcd9490a4e7f14068474874407a.png" class="latex" alt="$f(4,n)=f(3,f(4,n-1))$" style="vertical-align: -4px" width="202" height="18" />, this implies that <img src="//latex.artofproblemsolving.com/e/7/f/e7f6ed015647bb1e06ef8ca659b17d53af43c804.png" class="latex" alt="$f(4,n) \neq 2$" style="vertical-align: -4px" width="86" height="18" /> for any <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" />. By induction, <img src="//latex.artofproblemsolving.com/1/8/6/18668a430d5851dde50faa0206c6bc15d36c0775.png" class="latex" alt="$f(3,n) \neq f(3,2) = 4$" style="vertical-align: -4px" width="162" height="18" /> for any <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" />. It follows that <img src="//latex.artofproblemsolving.com/2/4/a/24a919adda9d78b4acf384220fe3cff88c015f40.png" class="latex" alt="$f(3,n) \neq f(3,4) = 0$" style="vertical-align: -4px" width="161" height="18" /> for any <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" />. Thus <img src="//latex.artofproblemsolving.com/7/c/f/7cf7ab41696118c23a89b2e4b99e2369bd682f1f.png" class="latex" alt="$f(4,n)$" style="vertical-align: -4px" width="51" height="18" /> only takes values in <img src="//latex.artofproblemsolving.com/8/7/6/876b8d6a743c118b5e3bb67d0002cc82050a77a0.png" class="latex" alt="$\{1,3\}$" style="vertical-align: -4px" width="43" height="18" />. In fact, it alternates between 1 and 3: <img src="//latex.artofproblemsolving.com/d/b/b/dbb26ee599c273147b2f4f052be47e96492ee424.png" class="latex" alt="$f(4,0)=f(3,1)=3$" style="vertical-align: -4px" width="159" height="18" />, then  <img src="//latex.artofproblemsolving.com/5/8/4/584f5470c836a6a43e8ab6a10df66ffd161b7f51.png" class="latex" alt="$f(4,1)=f(3,f(4,0))=f(3,3)=1$" style="vertical-align: -4px" width="276" height="18" />, then  <img src="//latex.artofproblemsolving.com/2/8/8/288c9194d486e3e57e04b65da68aad63c9455585.png" class="latex" alt="$f(4,2)=f(3,f(4,1))=f(3,1)=3$" style="vertical-align: -4px" width="277" height="18" />, and so on. 
</p><p>Repeating the argument above, we see that <img src="//latex.artofproblemsolving.com/e/5/0/e50a56eb87de7d59e4f3184d917124ccb779e0ad.png" class="latex" alt="$f(5,n) = f(4, f(5,n-1))$" style="vertical-align: -4px" width="202" height="18" /> can only take values in <img src="//latex.artofproblemsolving.com/8/7/6/876b8d6a743c118b5e3bb67d0002cc82050a77a0.png" class="latex" alt="$\{1,3\}$" style="vertical-align: -4px" width="43" height="18" />. However, <img src="//latex.artofproblemsolving.com/7/6/f/76f7a479d31c5d983ab6f9f64f4dec5cc0ca33bf.png" class="latex" alt="$f(5,n-1)\neq 0$" style="vertical-align: -4px" width="117" height="18" /> for any <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> implies that <img src="//latex.artofproblemsolving.com/c/3/f/c3f90abf30842ff09d6e858b3247d301c49a4caa.png" class="latex" alt="$f(5,n)\neq f(4,0)=3$" style="vertical-align: -4px" width="161" height="18" /> for any <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" />. Thus <img src="//latex.artofproblemsolving.com/3/6/8/36857551318e7b2d372364e777d042d5bb1da36f.png" class="latex" alt="$f(5,n)=1$" style="vertical-align: -4px" width="85" height="18" /> for all <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" />. We can easily verify this: <img src="//latex.artofproblemsolving.com/1/e/b/1ebe38fa61ba4aab30b1ae44225a80c8cf031874.png" class="latex" alt="$f(5,0)=f(4,1)=1$" style="vertical-align: -4px" width="159" height="18" />, then <img src="//latex.artofproblemsolving.com/2/a/2/2a29699f9af6cf824b385a417d4e160a5e35d85b.png" class="latex" alt="$f(5,1)=f(4,f(5,0))=f(4,1)=1$" style="vertical-align: -4px" width="276" height="18" />, then <img src="//latex.artofproblemsolving.com/9/c/8/9c850f7cb54b9e86365daa7df68c8337950df753.png" class="latex" alt="$f(5,2)=f(4,f(5,1))=f(4,1)=1$" style="vertical-align: -4px" width="276" height="18" />, and so on.
</p><p>Then <img src="//latex.artofproblemsolving.com/1/6/5/16556ca206d4d848d567e4b7c9f40b01a34ba700.png" class="latex" alt="$f(6,0)=f(5,1)=1$" style="vertical-align: -4px" width="159" height="18" />. Moreover, <img src="//latex.artofproblemsolving.com/e/c/c/ecc525dc83f35b7c3cde858c9d7446beb74221e3.png" class="latex" alt="$f(6,n) = f(5,f(6,n-1)) = 1$" style="vertical-align: -4px" width="236" height="18" /> for all <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" />. Continuing in this manner we see that <img src="//latex.artofproblemsolving.com/f/a/b/fab1c36f75fc1ec649351c7e9facd27f28b6fe87.png" class="latex" alt="$f(m,n)=1$" style="vertical-align: -4px" width="92" height="18" /> for all <img src="//latex.artofproblemsolving.com/1/c/5/1c51b8eebced7fe79c2c17e5f7a3b0062f57cdf2.png" class="latex" alt="$m\ge 5$" style="vertical-align: -2px" width="48" height="15" />. 
</p><p>In particular, <img src="//latex.artofproblemsolving.com/d/1/d/d1d8df8db5eaa1a717dd79b9a229b5111b7ab548.png" class="latex" alt="$f(2015,2) = \boxed{\textbf{(B)} \; 1}$" style="vertical-align: -10px" width="157" height="28" />.
</p>
