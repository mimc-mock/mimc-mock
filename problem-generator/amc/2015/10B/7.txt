<p>Consider the operation "minus the reciprocal of," defined by <img src="//latex.artofproblemsolving.com/0/d/9/0d97aced07f7efeb77b7b85f77ea9b1333cc02ed.png" class="latex" alt="$a\diamond b=a-\frac{1}{b}$" style="vertical-align: -12px" width="102" height="37" />. What is <img src="//latex.artofproblemsolving.com/3/c/1/3c1151d7db248af97ba22f812e68de042d5b0d3b.png" class="latex" alt="$((1\diamond2)\diamond3)-(1\diamond(2\diamond3))$" style="vertical-align: -4px" width="200" height="18" />?
</p><p><img src="//latex.artofproblemsolving.com/9/a/d/9ad7d98c07e13f32892142baaeff70487ff38779.png" class="latex" alt="$\textbf{(A) } -\dfrac{7}{30} \qquad\textbf{(B) } -\dfrac{1}{6} \qquad\textbf{(C) } 0 \qquad\textbf{(D) } \dfrac{1}{6} \qquad\textbf{(E) } \dfrac{7}{30}$" style="vertical-align: -12px" width="430" height="37" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p><img src="//latex.artofproblemsolving.com/1/4/e/14e2d3d67f6ac174d2b6713529165cb315e31350.png" class="latex" alt="$1\diamond2=1-\dfrac{1}{2}=\dfrac{1}{2}$" style="vertical-align: -12px" width="139" height="37" />, so <img src="//latex.artofproblemsolving.com/f/6/f/f6fc92317ca281da20b68a990c4fc99172296c0e.png" class="latex" alt="$(1\diamond2)\diamond3=\dfrac{1}{2}\diamond3=\dfrac{1}{2}-\dfrac{1}{3}=\dfrac{1}{6}$" style="vertical-align: -12px" width="247" height="37" />. Also, <img src="//latex.artofproblemsolving.com/4/b/b/4bb100a9a1cf4f0f2a23d16f928a5f598f4f4c06.png" class="latex" alt="$2\diamond3=2-\dfrac{1}{3}=\dfrac{5}{3}$" style="vertical-align: -12px" width="139" height="37" />, so <img src="//latex.artofproblemsolving.com/f/c/e/fce804143ba71e8af439cfd73a0fd5586eefba97.png" class="latex" alt="$1\diamond(2\diamond3)=1-\dfrac{1}{5/3}=1-\dfrac{3}{5}=\dfrac{2}{5}$" style="vertical-align: -17px" width="267" height="42" />. Thus, <img src="//latex.artofproblemsolving.com/2/2/c/22c1d59f7f7584db22fe0fa95e90cd58631521e0.png" class="latex" alt="$((1\diamond2)\diamond3)-(1\diamond(2\diamond3))=\dfrac{1}{6}-\dfrac{2}{5}=\boxed{\mathbf{(A)}\ -\dfrac{7}{30}}$" style="vertical-align: -18px" width="390" height="48" />
</p>
