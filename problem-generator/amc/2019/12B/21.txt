<p>How many quadratic polynomials with real coefficients are there such that the set of roots equals the set of coefficients? (For clarification: If the polynomial is <img src="//latex.artofproblemsolving.com/e/e/f/eef17be51ce22bb33687cd468e96c2eca1bf5ed3.png" class="latex" alt="$ax^2+bx+c,a\neq 0,$" style="vertical-align: -4px" width="153" height="19" /> and the roots are <img src="//latex.artofproblemsolving.com/b/5/5/b55ca7a0aa88ab7d58f4fc035317fdac39b17861.png" class="latex" alt="$r$" width="8" height="8" /> and <img src="//latex.artofproblemsolving.com/5/4/7/547263eb631ff2f1ef193c447d92ecb966213e1f.png" class="latex" alt="$s,$" style="vertical-align: -3px" width="12" height="11" /> then the requirement is that <img src="//latex.artofproblemsolving.com/e/3/4/e34a3244a3676af89d4102beb2eb77610af6fe59.png" class="latex" alt="$\{a,b,c\}=\{r,s\}$" style="vertical-align: -4px" width="125" height="18" />.)
</p><p><img src="//latex.artofproblemsolving.com/d/5/f/d5fe7baaf52282b50c4ba7e241378b4194eae50e.png" class="latex" alt="$\textbf{(A) } 3 \qquad\textbf{(B) } 4 \qquad\textbf{(C) } 5 \qquad\textbf{(D) } 6 \qquad\textbf{(E) } \text{infinitely many}$" style="vertical-align: -4px" width="453" height="17" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>Firstly, if <img src="//latex.artofproblemsolving.com/8/7/d/87dfc8219efcc266a55f7c3ffdf3279e823a9ba2.png" class="latex" alt="$r=s$" width="41" height="8" />, then <img src="//latex.artofproblemsolving.com/5/6/6/566b9136144a62fac113ecbc8065793a74123238.png" class="latex" alt="$a=b=c$" width="74" height="12" />, so the equation becomes <img src="//latex.artofproblemsolving.com/a/8/4/a848acbfb0ceae543a1324e355b315f5b86adbe4.png" class="latex" alt="$ax^2 + ax + a = 0 \Rightarrow x^2 + x + 1=0$" style="vertical-align: -1px" width="277" height="16" />, which has no real roots.
</p><p>Hence there are three cases we need to consider:
</p><p><b>Case 1</b>: <img src="//latex.artofproblemsolving.com/b/6/c/b6c3d2d364bdeaa68a520dc3a18218c05bace3a8.png" class="latex" alt="$a=b=r$" width="74" height="12" /> and <img src="//latex.artofproblemsolving.com/a/0/a/a0a888230cd17a88f0b67d0d48a57f69dd3ccca1.png" class="latex" alt="$c=s \neq r$" style="vertical-align: -4px" width="73" height="17" />:
The equation becomes <img src="//latex.artofproblemsolving.com/2/f/c/2fc1645d109e0b3726788f353d5d1e92d665a740.png" class="latex" alt="$ax^2+ax+c=0$" style="vertical-align: -1px" width="132" height="16" />, and by Vieta's Formulas, we have <img src="//latex.artofproblemsolving.com/f/c/9/fc987effcd880cd7fc6fa9e2828dad44206ff484.png" class="latex" alt="$a+c=-1$" style="vertical-align: -1px" width="86" height="13" /> and <img src="//latex.artofproblemsolving.com/0/5/0/05033dd3aac5a53b804b71cd81c882c55b7c806b.png" class="latex" alt="$ac = \frac{c}{a}$" style="vertical-align: -12px" width="53" height="33" />. This second equation becomes <img src="//latex.artofproblemsolving.com/b/5/a/b5a920b9309142760f759ed9e3c9832d0dcfffbe.png" class="latex" alt="$(a^2-1)c=0$" style="vertical-align: -4px" width="103" height="19" />. Hence one possibility is <img src="//latex.artofproblemsolving.com/a/8/0/a80d7a3ca845e6cae14b9db710b64331219ec3f7.png" class="latex" alt="$c=0$" width="40" height="12" />, in which case <img src="//latex.artofproblemsolving.com/8/1/7/817555099cf3c1ef68bd2e5b1f4bd27d48cbbb6c.png" class="latex" alt="$a=-1$" style="vertical-align: 0px" width="56" height="12" />, giving the equation <img src="//latex.artofproblemsolving.com/b/d/6/bd64472c7b9ce21352be95b951c6d192f0aae58e.png" class="latex" alt="$-x^2 - x = 0$" width="97" height="15" />, which has roots <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" /> and <img src="//latex.artofproblemsolving.com/d/0/d/d0dadbb52721adf7e774797fad55c1bd3c3a67cd.png" class="latex" alt="${-}1$" style="vertical-align: 0px" width="22" height="12" />. This gives one valid solution. On the other hand, if <img src="//latex.artofproblemsolving.com/2/e/7/2e703c9317484c02d00315965247fc020225788c.png" class="latex" alt="$c\neq 0$" style="vertical-align: -4px" width="40" height="17" />, then <img src="//latex.artofproblemsolving.com/b/a/0/ba0ad9c49dc2a8b3549ab1e7b3adeced5e534b68.png" class="latex" alt="$a^2-1=0$" style="vertical-align: 0px" width="81" height="15" />, so <img src="//latex.artofproblemsolving.com/b/4/3/b43ba1f77810a10b7b09b67669c0b6b15223ca14.png" class="latex" alt="$a = \pm 1$" style="vertical-align: 0px" width="56" height="12" />. If <img src="//latex.artofproblemsolving.com/5/0/2/502e059d1a020b368aee3a99c5669d88331d321f.png" class="latex" alt="$a=1$" style="vertical-align: 0px" width="42" height="12" />, we have <img src="//latex.artofproblemsolving.com/5/2/5/52566cda76e94a472e3d567b038aff381ea43375.png" class="latex" alt="$c=-2$" width="54" height="12" />, and the equation is <img src="//latex.artofproblemsolving.com/f/1/a/f1a4b91ea925d2f427dca2b7c61bfa8a3aa40781.png" class="latex" alt="$x^2 + x - 2 = 0$" style="vertical-align: -1px" width="114" height="16" />, which clearly works, giving a second valid solution. If <img src="//latex.artofproblemsolving.com/8/1/7/817555099cf3c1ef68bd2e5b1f4bd27d48cbbb6c.png" class="latex" alt="$a=-1$" style="vertical-align: 0px" width="56" height="12" />, then we have <img src="//latex.artofproblemsolving.com/a/8/0/a80d7a3ca845e6cae14b9db710b64331219ec3f7.png" class="latex" alt="$c=0$" width="40" height="12" />, which has already been considered, so this possibility gives no further valid solutions.
</p><p><b>Case 2</b>: <img src="//latex.artofproblemsolving.com/4/1/a/41a7080087bd1be69690bfbc0054b7af46215d51.png" class="latex" alt="$a=c=r$" width="74" height="8" />, <img src="//latex.artofproblemsolving.com/c/0/7/c079685d148b8e6dc54d1b55dae7a2aa88f6e557.png" class="latex" alt="$b=s \neq r$" style="vertical-align: -4px" width="73" height="17" />:
The equation becomes <img src="//latex.artofproblemsolving.com/8/9/4/894f7ea7d7f74d5e9ebeb6fa1c5cc19a71c25268.png" class="latex" alt="$ax^2 + bx + a = 0$" style="vertical-align: -1px" width="132" height="16" />, so by Vieta's Formulas, we have <img src="//latex.artofproblemsolving.com/9/b/6/9b6a87be3e9bebf90a2706dae4ff1725b2c058c9.png" class="latex" alt="$a+b = -\frac{b}{a}$" style="vertical-align: -12px" width="90" height="37" /> and <img src="//latex.artofproblemsolving.com/7/8/f/78fed10b7a7e08e2c85c0eb365f15cc759885edb.png" class="latex" alt="$ab = 1$" style="vertical-align: 0px" width="49" height="13" />. These equations reduce to <img src="//latex.artofproblemsolving.com/0/7/1/071c292c004da3c023e548998e0f0397fb5ec90a.png" class="latex" alt="$a^3 + a + 1 = 0$" style="vertical-align: -1px" width="113" height="16" />. By sketching a graph of this function, we see that there is exactly one real root. (Alternatively, note that as it is of odd degree, there is at least one real root, and by differentiation, it has no stationary points, so there is at most one real root. Combining these gives exactly one real root.) This gives a third valid solution.
</p><p><b>Case 3</b>: <img src="//latex.artofproblemsolving.com/0/e/1/0e1489f44949b5da418be8e4f5152aca1d723ba5.png" class="latex" alt="$a=r$" width="42" height="8" />, <img src="//latex.artofproblemsolving.com/2/8/4/28498dba86a085774cbc6fbdbef232df4bc4355e.png" class="latex" alt="$b=c=s \neq r$" style="vertical-align: -4px" width="105" height="17" />: 
The equation becomes <img src="//latex.artofproblemsolving.com/5/c/d/5cdc608976c8ad74501315f724e95544d4e6b10f.png" class="latex" alt="$ax^2 + bx+b=0$" style="vertical-align: -1px" width="130" height="16" />, so by Vieta's Formulas, we have <img src="//latex.artofproblemsolving.com/9/b/6/9b6a87be3e9bebf90a2706dae4ff1725b2c058c9.png" class="latex" alt="$a+b = -\frac{b}{a}$" style="vertical-align: -12px" width="90" height="37" /> and <img src="//latex.artofproblemsolving.com/c/f/c/cfc702a0013d6813d2d678fd44194b05b1e9f626.png" class="latex" alt="$ab = \frac{b}{a}$" style="vertical-align: -12px" width="53" height="37" />.
Observe that <img src="//latex.artofproblemsolving.com/d/0/9/d09016ae7d4bb0ccd869b43b3da544e9bacc08e0.png" class="latex" alt="$b \neq 0$" style="vertical-align: -4px" width="40" height="17" />, as if it were <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" />, the equation would just have one real root, <img src="//latex.artofproblemsolving.com/b/c/1/bc1f9d9bf8a1b606a4188b5ce9a2af1809e27a89.png" class="latex" alt="$0$" width="8" height="12" />, so this would not give a valid solution. Thus, taking the second equation and dividing both sides by <img src="//latex.artofproblemsolving.com/8/1/3/8136a7ef6a03334a7246df9097e5bcc31ba33fd2.png" class="latex" alt="$b$" width="8" height="12" />, we deduce have <img src="//latex.artofproblemsolving.com/b/9/b/b9bbd5c9d564d1ec978b03095be438ec311c52d9.png" class="latex" alt="$a=\frac{1}{a}$" style="vertical-align: -12px" width="46" height="37" />, so <img src="//latex.artofproblemsolving.com/c/0/9/c09e5c61278fa8840ebdf9ea0dfb3fc264056dcb.png" class="latex" alt="$a=\pm 1$" style="vertical-align: 0px" width="56" height="12" />. If <img src="//latex.artofproblemsolving.com/5/0/2/502e059d1a020b368aee3a99c5669d88331d321f.png" class="latex" alt="$a=1$" style="vertical-align: 0px" width="42" height="12" />, we have <img src="//latex.artofproblemsolving.com/7/3/e/73eaa322b2df33ba1ea3dc5e287bdb48dcc69981.png" class="latex" alt="$1+b=-b$" style="vertical-align: -1px" width="85" height="14" />, giving <img src="//latex.artofproblemsolving.com/5/e/9/5e9a475229473a3fe74fbf742f80e45e30cacc96.png" class="latex" alt="$b=-\frac{1}{2}$" style="vertical-align: -12px" width="57" height="37" />, so the equation is <img src="//latex.artofproblemsolving.com/5/1/9/5190a3096f9bb702426c29e2599299eac7a8b018.png" class="latex" alt="$x^2 - \frac{1}{2}x - \frac{1}{2} = 0$" style="vertical-align: -12px" width="131" height="37" />, which is a fourth valid solution. If <img src="//latex.artofproblemsolving.com/8/1/7/817555099cf3c1ef68bd2e5b1f4bd27d48cbbb6c.png" class="latex" alt="$a=-1$" style="vertical-align: 0px" width="56" height="12" />, we have <img src="//latex.artofproblemsolving.com/e/e/7/ee7beeed8eca268bf62e228447fb15d4c743c714.png" class="latex" alt="$1+b=b$" style="vertical-align: -1px" width="71" height="14" />, which is a contradiction, so this case gives no further valid solutions.
</p><p>Hence the total number of valid solutions is <img src="//latex.artofproblemsolving.com/d/f/8/df87dd1e1ac938c57d4fb05f78e1f147791dfe36.png" class="latex" alt="$\boxed{\textbf{(B) } 4}$" style="vertical-align: -10px" width="55" height="28" />.
</p>
