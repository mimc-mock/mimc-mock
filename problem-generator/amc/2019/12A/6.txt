<p>The figure below shows line <img src="//latex.artofproblemsolving.com/6/3/c/63c17c295325f731666c7d74952b563a01e00fcc.png" class="latex" alt="$\ell$" style="vertical-align: 0px" width="7" height="13" /> with a regular, infinite, recurring pattern of squares and line segments.
</p><p><img src="//latex.artofproblemsolving.com/a/2/f/a2f165e74d6ddc1b021c8af4c74374a1a78d2815.png" class="latexcenter" alt="[asy] size(300); defaultpen(linewidth(0.8)); real r = 0.35; path P = (0,0)--(0,1)--(1,1)--(1,0), Q = (1,1)--(1+r,1+r); path Pp = (0,0)--(0,-1)--(1,-1)--(1,0), Qp = (-1,-1)--(-1-r,-1-r); for(int i=0;i &lt;= 4;i=i+1) { draw(shift((4*i,0)) * P); draw(shift((4*i,0)) * Q); } for(int i=1;i &lt;= 4;i=i+1) { draw(shift((4*i-2,0)) * Pp); draw(shift((4*i-1,0)) * Qp); } draw((-1,0)--(18.5,0),Arrows(TeXHead)); [/asy]" width="502" height="72" />
</p><p>How many of the following four kinds of rigid motion transformations of the plane in which this figure is drawn, other than the identity transformation, will transform this figure into itself?
</p>
<ul><li>some rotation around a point of line <img src="//latex.artofproblemsolving.com/6/3/c/63c17c295325f731666c7d74952b563a01e00fcc.png" class="latex" alt="$\ell$" style="vertical-align: 0px" width="7" height="13" /></li>
<li>some translation in the direction parallel to line <img src="//latex.artofproblemsolving.com/6/3/c/63c17c295325f731666c7d74952b563a01e00fcc.png" class="latex" alt="$\ell$" style="vertical-align: 0px" width="7" height="13" /></li>
<li>the reflection across line <img src="//latex.artofproblemsolving.com/6/3/c/63c17c295325f731666c7d74952b563a01e00fcc.png" class="latex" alt="$\ell$" style="vertical-align: 0px" width="7" height="13" /></li>
<li>some reflection across a line perpendicular to line <img src="//latex.artofproblemsolving.com/6/3/c/63c17c295325f731666c7d74952b563a01e00fcc.png" class="latex" alt="$\ell$" style="vertical-align: 0px" width="7" height="13" /></li></ul>
<p><img src="//latex.artofproblemsolving.com/7/9/0/790fd2f1643b47466e5d327bc932fe70d7503210.png" class="latex" alt="$\textbf{(A) } 0 \qquad\textbf{(B) } 1 \qquad\textbf{(C) } 2 \qquad\textbf{(D) } 3 \qquad\textbf{(E) } 4$" style="vertical-align: -4px" width="353" height="17" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>Statement <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" /> is true. A <img src="//latex.artofproblemsolving.com/3/f/0/3f0cd95f95238159d02dbcea0aca632c2928991c.png" class="latex" alt="$180^{\circ}$" style="vertical-align: 0px" width="33" height="13" /> rotation about the point half way between an up-facing square and a down-facing square will yield the same figure. 
</p><p>Statement <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" /> is also true. A translation to the left or right will place the image onto itself when the figures above and below the line realign (the figure goes on infinitely in both directions). 
</p><p>Statement <img src="//latex.artofproblemsolving.com/7/c/d/7cde695f2e4542fd01f860a89189f47a27143b66.png" class="latex" alt="$3$" width="8" height="12" /> is false. A reflection across line <img src="//latex.artofproblemsolving.com/6/3/c/63c17c295325f731666c7d74952b563a01e00fcc.png" class="latex" alt="$\ell$" style="vertical-align: 0px" width="7" height="13" /> will change the up-facing squares to down-facing squares and vice versa. 
</p><p>Finally, statement <img src="//latex.artofproblemsolving.com/c/7/c/c7cab1a05e1e0c1d51a6a219d96577a16b7abf9d.png" class="latex" alt="$4$" style="vertical-align: 0px" width="9" height="12" /> is also false because it will cause the diagonal lines extending from the squares to switch direction. 
</p><p>Thus, only <img src="//latex.artofproblemsolving.com/6/e/b/6eb000de4c312a8163426664affb66720f748471.png" class="latex" alt="$\fbox {\textbf{(C)}  2}$" style="vertical-align: -11px" width="56" height="30" /> statements are true.
</p>
