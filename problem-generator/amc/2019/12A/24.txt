<p>For how many integers <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> between <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" /> and <img src="//latex.artofproblemsolving.com/9/0/6/906734e2f1010e9a171ffee8050cbc0649cea37c.png" class="latex" alt="$50$" width="17" height="12" />, inclusive, is <img src="//latex.artofproblemsolving.com/2/f/b/2fb0a7fcc7e8ea99426bae70e641eac53ea8d7e6.png" class="latexcenter" alt="\[\frac{(n^2-1)!}{(n!)^n}\]" width="69" height="43" /> an integer? (Recall that <img src="//latex.artofproblemsolving.com/0/1/9/019528532ecd654053bdee316fce3f77351d4989.png" class="latex" alt="$0! = 1$" style="vertical-align: 0px" width="46" height="13" />.)
</p><p><img src="//latex.artofproblemsolving.com/e/b/6/eb6602925116edbf22be1df3f33f9ffed3895bf0.png" class="latex" alt="$\textbf{(A) } 31 \qquad \textbf{(B) } 32 \qquad \textbf{(C) } 33 \qquad \textbf{(D) } 34 \qquad \textbf{(E) } 35$" style="vertical-align: -4px" width="398" height="17" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>The main insight is that 
</p><p><img src="//latex.artofproblemsolving.com/e/4/5/e45e8bb08e31a5d89f1542589cbc346b1949bf8d.png" class="latexcenter" alt="\[\frac{(n^2)!}{(n!)^{n+1}}\]" width="56" height="43" /> 
</p><p>is always an integer. This is true because it is precisely the number of ways to split up <img src="//latex.artofproblemsolving.com/6/0/0/6006bba637eef443381fbaff768fa62d9ff41e62.png" class="latex" alt="$n^2$" width="17" height="15" /> objects into <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> unordered groups of size <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" />. Thus,
</p><p><img src="//latex.artofproblemsolving.com/7/2/7/727e7d51f941f9e450a61f08646b5a1e5e270350.png" class="latexcenter" alt="\[\frac{(n^2-1)!}{(n!)^n}=\frac{(n^2)!}{(n!)^{n+1}}\cdot\frac{n!}{n^2}\]" width="189" height="43" />
</p><p>is an integer if <img src="//latex.artofproblemsolving.com/3/0/f/30f2492ddcae167e222a00c431a59aeb822d92d3.png" class="latex" alt="$n^2 \mid n!$" style="vertical-align: -4px" width="48" height="19" />, or in other words, if <img src="//latex.artofproblemsolving.com/a/2/6/a26384c388f66713c2e082bce3a8513037f02f36.png" class="latex" alt="$\frac{(n-1)!}{n}$" style="vertical-align: -13px" width="67" height="40" />, is an integer. This condition is false precisely when <img src="//latex.artofproblemsolving.com/3/9/c/39cf3e35a4981583288d2a7c4b34989916fb7360.png" class="latex" alt="$n=4$" style="vertical-align: 0px" width="44" height="12" /> or <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> is prime, by <a href="/wiki/index.php/Wilson%27s_Theorem" title="Wilson&#39;s Theorem">Wilson's Theorem</a>. There are <img src="//latex.artofproblemsolving.com/f/1/9/f1965fae079a9ba2c0726c307070c2355dfcb213.png" class="latex" alt="$15$" style="vertical-align: 0px" width="17" height="13" /> primes between <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" /> and <img src="//latex.artofproblemsolving.com/9/0/6/906734e2f1010e9a171ffee8050cbc0649cea37c.png" class="latex" alt="$50$" width="17" height="12" />, inclusive, so there are <img src="//latex.artofproblemsolving.com/c/3/9/c39b25c45ce78af7a14ed41edeedcbb61fa7c22c.png" class="latex" alt="$15 + 1 = 16$" style="vertical-align: -1px" width="91" height="14" /> terms for which
</p><p><img src="//latex.artofproblemsolving.com/c/0/9/c09ec519b3d2598b4d89348c3999cd0e715c9f8b.png" class="latexcenter" alt="\[\frac{(n^2-1)!}{(n!)^{n}}\]" width="69" height="43" />
</p><p>is potentially not an integer. It can be easily verified that the above expression is not an integer for <img src="//latex.artofproblemsolving.com/3/9/c/39cf3e35a4981583288d2a7c4b34989916fb7360.png" class="latex" alt="$n=4$" style="vertical-align: 0px" width="44" height="12" /> as there are more factors of <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" /> in the denominator than the numerator. Similarly, it can be verified that the above expression is not an integer for any prime <img src="//latex.artofproblemsolving.com/7/4/f/74fd85afeb25627a961382162e87d1969b921837.png" class="latex" alt="$n=p$" style="vertical-align: -3px" width="44" height="11" />, as there are more factors of p in the denominator than the numerator. Thus all <img src="//latex.artofproblemsolving.com/9/a/5/9a5b4928c8fe50ce3c2428da3bee3505e891b788.png" class="latex" alt="$16$" style="vertical-align: 0px" width="17" height="13" /> values of n make the expression not an integer and the answer is <img src="//latex.artofproblemsolving.com/b/b/8/bb874c177d4ef7d4f13a75207d954fed87fd1ecd.png" class="latex" alt="$50-16=\boxed{\textbf{(D)}\ 34}$" style="vertical-align: -10px" width="148" height="28" />.
</p><p>SideNote: Another method to prove that <img src="//latex.artofproblemsolving.com/e/4/5/e45e8bb08e31a5d89f1542589cbc346b1949bf8d.png" class="latexcenter" alt="\[\frac{(n^2)!}{(n!)^{n+1}}\]" width="56" height="43" /> is always an integer is instead as follows using Number Theory. Notice that <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> will divide the numerator <img src="//latex.artofproblemsolving.com/0/e/3/0e3efd9b14723a92c2ae891fe27780d5f8e2b215.png" class="latex" alt="$n+1$" style="vertical-align: -1px" width="41" height="13" /> times, since <img src="//latex.artofproblemsolving.com/1/9/d/19d5bcf7eeed4f4c6ec39e4b0772393853008bc9.png" class="latex" alt="$n^2 = n \times n$" style="vertical-align: -2px" width="88" height="18" /> contains not one but two factors of <img src="//latex.artofproblemsolving.com/b/d/9/bd923e41635af40a1abe58f230732f2280b0723a.png" class="latex" alt="$n.$" width="14" height="8" /> Also, for <img src="//latex.artofproblemsolving.com/b/e/f/bef1b888f2f5f0c49cf0d9c0553402590445a9f5.png" class="latex" alt="$a &lt; n,$" style="vertical-align: -4px" width="52" height="15" /> notice that <img src="//latex.artofproblemsolving.com/c/7/d/c7d457e388298246adb06c587bccd419ea67f7e8.png" class="latex" alt="$a$" width="9" height="8" /> divides <img src="//latex.artofproblemsolving.com/c/3/1/c313160d1bd1a39d5923afbe568db7dcead2f212.png" class="latex" alt="$(n^2)!$" style="vertical-align: -4px" width="36" height="19" /> at least <img src="//latex.artofproblemsolving.com/b/b/2/bb2eca5503e08e653dc705c5c8a553c239b67b84.png" class="latexcenter" alt="\[\lfloor \frac{n^2}{a} \rfloor \ge \lfloor \frac{n^2}{n-1} \rfloor \ge \lfloor \frac{n^2 - 1}{n-1} \rfloor \ge n+1\]" width="281" height="39" /> times. Thus, each integer from <img src="//latex.artofproblemsolving.com/d/c/e/dce34f4dfb2406144304ad0d6106c5382ddd1446.png" class="latex" alt="$1$" style="vertical-align: 0px" width="8" height="12" /> to <img src="//latex.artofproblemsolving.com/1/7/4/174fadd07fd54c9afe288e96558c92e0c1da733a.png" class="latex" alt="$n$" width="10" height="8" /> will divide <img src="//latex.artofproblemsolving.com/c/3/1/c313160d1bd1a39d5923afbe568db7dcead2f212.png" class="latex" alt="$(n^2)!$" style="vertical-align: -4px" width="36" height="19" /> at least <img src="//latex.artofproblemsolving.com/0/e/3/0e3efd9b14723a92c2ae891fe27780d5f8e2b215.png" class="latex" alt="$n+1$" style="vertical-align: -1px" width="41" height="13" /> times, which proves such a lemma. <img src="//latex.artofproblemsolving.com/d/0/3/d03964589d6c9d48bbb4ed36acd2e568c899f62a.png" class="latex" alt="$\square{}$" width="13" height="12" />
</p>
<h2><span class="mw-headline" id="Solution_2">Solution 2</span></h2>
<p>We can use the P-Adic Valuation (more info could be found here: <a href="/wiki/index.php/Mathematicial_notation" title="Mathematicial notation">Mathematicial notation</a>) of n to solve this problem (recall the P-Adic Valuation of 'n' is denoted by <img src="//latex.artofproblemsolving.com/5/4/c/54c5d91852fe42bcab76ef23453deea15ce1feb4.png" class="latex" alt="$v_p (n)$" style="vertical-align: -4px" width="40" height="18" /> and is defined as the greatest power of some prime 'p' that divides n. For example, <img src="//latex.artofproblemsolving.com/f/1/d/f1dbe982850daeca3a69630285d236e75ce7ec59.png" class="latex" alt="$v_2 (6)=1$" style="vertical-align: -4px" width="71" height="18" /> or <img src="//latex.artofproblemsolving.com/e/c/8/ec8111573c1930d88294da215664236b29527d61.png" class="latex" alt="$v_7 (245)=2$" style="vertical-align: -4px" width="90" height="18" /> .) Using Legendre's formula, we know that&#160;:
</p><p><img src="//latex.artofproblemsolving.com/5/9/b/59b94976138ce9346d27b8be4ba9529e42437486.png" class="latexcenter" alt="\[v_p (n!)= \sum_{i=1}^\infty   \lfloor \frac {n}{p^i} \rfloor\]" width="127" height="48" />
</p><p>Seeing factorials involved in the problem, this prompts us to use <a href="/wiki/index.php/Legendre%27s_Formula" title="Legendre&#39;s Formula">Legendre's Formula</a> where n is a power of a prime.
</p><p>We also know that , <img src="//latex.artofproblemsolving.com/5/7/1/571a81a29d240a04f98616bf92ea767a0a94d87b.png" class="latex" alt="$v_p (m^n) = n \cdot v_p (m)$" style="vertical-align: -4px" width="149" height="18" /> .
Knowing that <img src="//latex.artofproblemsolving.com/e/d/9/ed9b9d632661064469851ff5a8363effa1522168.png" class="latex" alt="$a\mid b$" style="vertical-align: -4px" width="33" height="18" /> if <img src="//latex.artofproblemsolving.com/e/e/f/eef121d8d5c044088a2038fff7be5ad390400ad8.png" class="latex" alt="$v_p (a) \le v_p (b)$" style="vertical-align: -4px" width="101" height="18" /> , we have that&#160;:
</p><p><img src="//latex.artofproblemsolving.com/b/e/2/be25e97702f26f1bb5616de64e27d309e5d2e55c.png" class="latexcenter" alt="\[n \cdot v_p (n!) \le v_p ((n^2 -1 )!)\]" width="192" height="20" /> and we must find all n for which this is true.
</p><p>If we plug in <img src="//latex.artofproblemsolving.com/7/4/f/74fd85afeb25627a961382162e87d1969b921837.png" class="latex" alt="$n=p$" style="vertical-align: -3px" width="44" height="11" />, by Legendre's we get two equations:
</p><p><img src="//latex.artofproblemsolving.com/6/9/4/6945baa2ed2ee721dd31ebe1c75534360cabeb71.png" class="latexcenter" alt="\[v_p ((n^2 -1)!) = \sum_{i=1}^\infty  \lfloor \frac {n^2 -1}{p^i} \rfloor = (p-1)+0+...+0 = p-1\]" width="461" height="48" />
</p><p>And we also get&#160;:
</p><p><img src="//latex.artofproblemsolving.com/f/3/a/f3ae2368b0cb061d35d1af0e4a789fcac0b66433.png" class="latexcenter" alt="\[v_p ((n!)^n) = n \cdot v_p (n!)= n \cdot \sum_{i=1}^\infty   \lfloor \frac {n}{p^i} \rfloor = p \cdot ( 1+0+...0) = p\]" width="453" height="48" />
</p><p>But we are asked to prove that <img src="//latex.artofproblemsolving.com/4/5/5/4553ea5162554e8dbc72fd98e7c4a1b47cee46fb.png" class="latex" alt="$n \cdot v_p (n!) \le v_p ((n^2 -1 )!) \Longrightarrow p \le p-1$" style="vertical-align: -4px" width="306" height="19" /> which is false for all 'n' where n is prime.
</p><p>Now we try the same for <img src="//latex.artofproblemsolving.com/5/c/a/5cafe2ec47609fea69da05e31308d5f9642c2e7c.png" class="latex" alt="$n=p^2$" style="vertical-align: -3px" width="50" height="18" /> , where p is a prime. By Legendre we arrive at:
</p><p><img src="//latex.artofproblemsolving.com/0/b/7/0b760ce26be5e064a5912d42ef46822a2eb681e7.png" class="latexcenter" alt="\[v_p ((p^4 -1)!) = p^3 + p^2 + p -3\]" width="239" height="20" /> 
</p><p>(as <img src="//latex.artofproblemsolving.com/6/8/b/68b2be99f2a1e3ddf0268149f1013df76171ea8d.png" class="latex" alt="$v_p(p^4!) = p^3 + p^2 + p + 1$" style="vertical-align: -6px" width="196" height="22" /> and <img src="//latex.artofproblemsolving.com/4/e/e/4eeda7a7a8630f80b8b345d1bc47e9fdfc43c597.png" class="latex" alt="$p^4$" style="vertical-align: -3px" width="16" height="18" /> contains 4 factors of <img src="//latex.artofproblemsolving.com/3/6/f/36f73fc1312ee0349b3f3a0f3bd9eb5504339011.png" class="latex" alt="$p$" style="vertical-align: -3px" width="10" height="11" />) and 
</p><p><img src="//latex.artofproblemsolving.com/2/8/7/2870a4b3d30456f301d44d2da53eeb2b3dcba633.png" class="latexcenter" alt="\[p^2 \cdot v_p (p^2&#160;!) = p^3 + p^2\]" width="161" height="20" />
</p><p>Then we get:
</p><p><img src="//latex.artofproblemsolving.com/6/6/9/669b81d6a29fef54261e7f2f5704d9db204599b0.png" class="latexcenter" alt="\[p^2 \cdot v_p (p!) \le v_p ((n^4 -1)!) \Longrightarrow p^3 + p^2 \le p^3 + p^2 + p -3\]" width="436" height="20" /> Which is true for all primes except for 2, so <img src="//latex.artofproblemsolving.com/8/c/b/8cb0e6e64d73bacef9c8869039bc21581f1884e6.png" class="latex" alt="$2^2 = 4$" style="vertical-align: 0px" width="50" height="15" /> doesn't work. It can easily be verified that for all <img src="//latex.artofproblemsolving.com/2/6/c/26c2d2440acaf99df7539011d3388920f0b32e37.png" class="latex" alt="$n=p^i$" style="vertical-align: -3px" width="48" height="18" /> where <img src="//latex.artofproblemsolving.com/3/4/8/34857b3ba74ce5cd8607f3ebd23e9015908ada71.png" class="latex" alt="$i$" width="5" height="12" /> is an integer greater than 2, satisfies the inequality&#160;:<img src="//latex.artofproblemsolving.com/c/1/0/c10e68debc4a0ca1448ffe81f0b82714033f9059.png" class="latexcenter" alt="\[n \cdot v_p (n!) \le v_p ((n^2 -1 )!).\]" width="197" height="20" />
</p><p>Therefore, there are 16 values that don't work and <img src="//latex.artofproblemsolving.com/e/4/6/e46979acbfe776cb91045b431dde10b2cdf9975c.png" class="latex" alt="$50-16 = \boxed{\mathbf{(D)}\ 34}$" style="vertical-align: -10px" width="148" height="29" /> values that work.
</p><p>~qwertysri987
</p>
