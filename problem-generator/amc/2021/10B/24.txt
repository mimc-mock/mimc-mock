<p>Arjun and Beth play a game in which they take turns removing one brick or two adjacent bricks from one "wall" among a set of several walls of bricks, with gaps possibly creating new walls. The walls are one brick tall. For example, a set of walls of sizes <img src="//latex.artofproblemsolving.com/c/7/c/c7cab1a05e1e0c1d51a6a219d96577a16b7abf9d.png" class="latex" alt="$4$" style="vertical-align: 0px" width="9" height="12" /> and <img src="//latex.artofproblemsolving.com/4/1/c/41c544263a265ff15498ee45f7392c5f86c6d151.png" class="latex" alt="$2$" width="8" height="12" /> can be changed into any of the following by one move: <img src="//latex.artofproblemsolving.com/3/8/f/38f3e5ebe894921c1595df33f8a08e5f31d58b34.png" class="latex" alt="$(3,2),(2,1,2),(4),(4,1),(2,2),$" style="vertical-align: -4px" width="238" height="18" /> or <img src="//latex.artofproblemsolving.com/f/2/e/f2eeef264787a92d49f6c6f239c538b3baa64749.png" class="latex" alt="$(1,1,2).$" style="vertical-align: -4px" width="61" height="18" />
</p><p><img src="//latex.artofproblemsolving.com/1/a/5/1a5938c90db67cc867d798bb9693e149b9438071.png" class="latexcenter" alt="[asy] unitsize(4mm); real[] boxes = {0,1,2,3,5,6,13,14,15,17,18,21,22,24,26,27,30,31,32,33}; for(real i:boxes){ 	draw(box((i,0),(i+1,3))); } draw((8,1.5)--(12,1.5),Arrow()); defaultpen(fontsize(20pt)); label(&quot;,&quot;,(20,0)); label(&quot;,&quot;,(29,0)); label(&quot;,...&quot;,(35.5,0)); [/asy]" width="695" height="68" />
</p><p>Arjun plays first, and the player who removes the last brick wins. For which starting configuration is there a strategy that guarantees a win for Beth?
</p><p><img src="//latex.artofproblemsolving.com/9/3/e/93edd1f3df0ee06e42e4e92ab12b30704da29172.png" class="latex" alt="$\textbf{(A) }(6,1,1) \qquad \textbf{(B) }(6,2,1) \qquad \textbf{(C) }(6,2,2)\qquad \textbf{(D) }(6,3,1) \qquad \textbf{(E) }(6,3,2)$" style="vertical-align: -4px" width="594" height="18" />
</p>
<h2><span class="mw-headline" id="Solution_1">Solution 1</span></h2>
<p>We say that a game state is an N-position if it is winning for the next player (the player to move), and a P-position if it is winning for the other player. We are trying to find which of the given states is a P-position.
</p><p>First we note that symmetrical positions are P-positions, as the second player can win by mirroring the first player's moves. It follows that <img src="//latex.artofproblemsolving.com/a/6/4/a64d5d84be1b22d642dc8d1ff6438b8c428465eb.png" class="latex" alt="$(6, 1, 1)$" style="vertical-align: -4px" width="56" height="18" /> is an N-position, since we can win by moving to <img src="//latex.artofproblemsolving.com/0/4/7/047ada8f1946ea02e6d32d999a9b7692551e2ba7.png" class="latex" alt="$(2, 2, 1, 1)$" style="vertical-align: -4px" width="73" height="18" />; this rules out <img src="//latex.artofproblemsolving.com/a/4/6/a46f6af01fcf3bb5f6cac2316bc4de9bb3906bec.png" class="latex" alt="$\textbf{(A)}$" style="vertical-align: -4px" width="28" height="17" />. We next look at <img src="//latex.artofproblemsolving.com/6/6/5/6656f2f0d37aa8d94541b3a1494c8096f78fd4aa.png" class="latex" alt="$(6, 2, 1)$" style="vertical-align: -4px" width="56" height="18" />. The possible next states are
<img src="//latex.artofproblemsolving.com/3/0/4/3041bb84300c95e62b41a0e3472cbb2525a5a952.png" class="latexcenter" alt="\[(6, 2), \enskip (6, 1, 1), \enskip (6, 1), \enskip (5, 2, 1), \enskip (4, 2, 1, 1), \enskip (4, 2, 1), \enskip (3, 2, 2, 1), \enskip (3, 2, 1, 1), \enskip (2, 2, 2, 1).\]" width="685" height="18" />
None of these are symmetrical, so we might reasonably suspect that they are all N-positions. Indeed, it just so happens that for all of these states except <img src="//latex.artofproblemsolving.com/c/1/7/c17384a6c2a4aee0ea47051a974ad1cb5967d47f.png" class="latex" alt="$(6, 2)$" style="vertical-align: -4px" width="38" height="18" /> and <img src="//latex.artofproblemsolving.com/5/a/0/5a0994ca0283f051e07f29844f7af73f4624eef1.png" class="latex" alt="$(6, 1)$" style="vertical-align: -4px" width="38" height="18" />, we can win by moving to <img src="//latex.artofproblemsolving.com/0/4/7/047ada8f1946ea02e6d32d999a9b7692551e2ba7.png" class="latex" alt="$(2, 2, 1, 1)$" style="vertical-align: -4px" width="73" height="18" />; it remains to check that <img src="//latex.artofproblemsolving.com/c/1/7/c17384a6c2a4aee0ea47051a974ad1cb5967d47f.png" class="latex" alt="$(6, 2)$" style="vertical-align: -4px" width="38" height="18" /> and <img src="//latex.artofproblemsolving.com/5/a/0/5a0994ca0283f051e07f29844f7af73f4624eef1.png" class="latex" alt="$(6, 1)$" style="vertical-align: -4px" width="38" height="18" /> are N-positions.
</p><p>To save ourselves work, it would be nice if we could find a single P-position directly reachable from both <img src="//latex.artofproblemsolving.com/c/1/7/c17384a6c2a4aee0ea47051a974ad1cb5967d47f.png" class="latex" alt="$(6, 2)$" style="vertical-align: -4px" width="38" height="18" /> and <img src="//latex.artofproblemsolving.com/5/a/0/5a0994ca0283f051e07f29844f7af73f4624eef1.png" class="latex" alt="$(6, 1)$" style="vertical-align: -4px" width="38" height="18" />. We notice that <img src="//latex.artofproblemsolving.com/e/2/7/e27589f15c8fce6e2ef60cce98d7f014b5e02e4d.png" class="latex" alt="$(3, 2, 1)$" style="vertical-align: -4px" width="56" height="18" /> is directly reachable from both states, so it would suffice to show that <img src="//latex.artofproblemsolving.com/e/2/7/e27589f15c8fce6e2ef60cce98d7f014b5e02e4d.png" class="latex" alt="$(3, 2, 1)$" style="vertical-align: -4px" width="56" height="18" /> is a P-position. Indeed, the possible next states are
<img src="//latex.artofproblemsolving.com/9/d/a/9dafefba4b2631e3ec5d7eb6c82fe826f8e65cdf.png" class="latexcenter" alt="\[(3, 2), \enskip (3, 1, 1), \enskip (3, 1), \enskip (2, 2, 1), \enskip (2, 1, 1, 1), \enskip (2, 1, 1),\]" width="412" height="18" />
which allow for the following refutations:
</p><p><img src="//latex.artofproblemsolving.com/c/d/4/cd421104c81bc83688ab57254ad31886ab62851d.png" class="latexcenter" alt="\begin{align*} &amp;(3, 2) \to (2, 2), &amp;&amp; &amp;&amp;(3, 1, 1) \to (1, 1, 1, 1), &amp;&amp; &amp;&amp;(3, 1) \to (1, 1), \\ &amp;(2, 2, 1) \to (2, 2), &amp;&amp; &amp;&amp;(2, 1, 1, 1) \to (1, 1, 1, 1), &amp;&amp; &amp;&amp;(2, 1, 1) \to (1, 1). \end{align*}" width="508" height="43" />
</p><p>Hence, <img src="//latex.artofproblemsolving.com/e/2/7/e27589f15c8fce6e2ef60cce98d7f014b5e02e4d.png" class="latex" alt="$(3, 2, 1)$" style="vertical-align: -4px" width="56" height="18" /> is a P-position, so <img src="//latex.artofproblemsolving.com/c/1/7/c17384a6c2a4aee0ea47051a974ad1cb5967d47f.png" class="latex" alt="$(6, 2)$" style="vertical-align: -4px" width="38" height="18" /> and <img src="//latex.artofproblemsolving.com/5/a/0/5a0994ca0283f051e07f29844f7af73f4624eef1.png" class="latex" alt="$(6, 1)$" style="vertical-align: -4px" width="38" height="18" /> are both N-positions, along with all other possible next states from <img src="//latex.artofproblemsolving.com/6/6/5/6656f2f0d37aa8d94541b3a1494c8096f78fd4aa.png" class="latex" alt="$(6, 2, 1)$" style="vertical-align: -4px" width="56" height="18" /> as noted before. Thus, <img src="//latex.artofproblemsolving.com/6/6/5/6656f2f0d37aa8d94541b3a1494c8096f78fd4aa.png" class="latex" alt="$(6, 2, 1)$" style="vertical-align: -4px" width="56" height="18" /> is a P-position, so our answer is <img src="//latex.artofproblemsolving.com/a/2/1/a2194b81820e8380be586c1564c3f5d03d5c100d.png" class="latex" alt="$\boxed{\textbf{(B)}}$" style="vertical-align: -10px" width="40" height="28" />. (For completeness, we could also rule out <img src="//latex.artofproblemsolving.com/1/6/9/169b8d69bd74a91e72ec43123070d770633a340a.png" class="latex" alt="$\textbf{(C)}$" style="vertical-align: -4px" width="27" height="17" />, <img src="//latex.artofproblemsolving.com/b/3/0/b305929a95b53f14a0ad53eba1d8560626e1f28b.png" class="latex" alt="$\textbf{(D)}$" style="vertical-align: -4px" width="28" height="17" /> and <img src="//latex.artofproblemsolving.com/7/5/e/75e41e3cd48cee2c6866cfbb4386a8e99d3482f5.png" class="latex" alt="$\textbf{(E)}$" style="vertical-align: -4px" width="26" height="17" /> as in Solution 2.)
</p><p>-Note: In general, this game is very complicated. For example <img src="//latex.artofproblemsolving.com/9/4/6/9467354d286f2df469cd9cf966fd4a352ce83b6d.png" class="latex" alt="$(8, 7, 5, 3, 2)$" style="vertical-align: -4px" width="90" height="18" /> is winning for the first player but good luck showing that.
</p>
