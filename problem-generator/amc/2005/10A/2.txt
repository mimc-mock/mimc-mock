<p>For each pair of real numbers <img src="//latex.artofproblemsolving.com/a/f/4/af40dd462cfac9a98f63c84a474236829d0b3c25.png" class="latex" alt="$a \neq b$" style="vertical-align: -4px" width="42" height="17" />, define the <a href="/wiki/index.php/Operation" title="Operation">operation</a> <img src="//latex.artofproblemsolving.com/d/1/7/d17950ad2ab63c764778b133f95a03dcf2969d57.png" class="latex" alt="$\star$" width="9" height="9" /> as
</p><p><img src="//latex.artofproblemsolving.com/5/4/4/544e898c2610cdfe1b7e459dd14276cb5ab625a1.png" class="latex" alt="$(a \star b) = \frac{a+b}{a-b}$" style="vertical-align: -12px" width="115" height="37" />.
</p><p>What is the value of <img src="//latex.artofproblemsolving.com/5/c/9/5c96d2265fd92d9330d54468f19c32fc061e0f65.png" class="latex" alt="$((1 \star 2) \star 3)$" style="vertical-align: -4px" width="88" height="18" />?
</p><p><img src="//latex.artofproblemsolving.com/d/3/5/d353a13b68d0c32a1e32cb387610f339e40295db.png" class="latex" alt="$\textbf{(A) } -\frac{2}{3}\qquad \textbf{(B) } -\frac{1}{5}\qquad \textbf{(C) } 0\qquad \textbf{(D) } \frac{1}{2}\qquad \textbf{(E) } \textrm{This\, value\, is\, not\, defined.}$" style="vertical-align: -13px" width="597" height="38" />
</p>
