<p>Which of the following is the same as
</p><p><img src="//latex.artofproblemsolving.com/d/d/a/ddaa33db347793c4318c6123185c74195fe22c74.png" class="latexcenter" alt="\[\frac{2-4+6-8+10-12+14}{3-6+9-12+15-18+21}?\]" width="243" height="39" />
</p><p><img src="//latex.artofproblemsolving.com/0/5/2/05251bde871a7ae30ded735bb9326d2d8edaa84b.png" class="latex" alt="$\text {(A) } -1 \qquad \text {(B) } -\frac{2}{3} \qquad \text {(C) } \frac{2}{3} \qquad \text {(D) } 1 \qquad \text {(E) } \frac{14}{3}$" style="vertical-align: -12px" width="394" height="37" />
</p>
<h2><span class="mw-headline" id="Solution">Solution</span></h2>
<p>The numbers in the numerator and denominator can be grouped like this:
</p><p><img src="//latex.artofproblemsolving.com/1/2/d/12d2b50845a4de5a37bc01ba150dcc28f10429dd.png" class="latexcenter" alt="\begin{align*} 2+(-4+6)+(-8+10)+(-12+14)&amp;=2*4\\ 3+(-6+9)+(-12+15)+(-18+21)&amp;=3*4\\ \frac{2*4}{3*4}&amp;=\frac{2}{3} \Rightarrow \text {(C)} \end{align*}" width="407" height="88" />
</p><p>Alternatively, notice that each term in the numerator is <img src="//latex.artofproblemsolving.com/7/5/4/7547747b133453d3c209cff6cfbf129aa67a6947.png" class="latex" alt="$\frac{2}{3}$" style="vertical-align: -12px" width="11" height="37" /> of a term in the denominator, so the quotient has to be <img src="//latex.artofproblemsolving.com/7/5/4/7547747b133453d3c209cff6cfbf129aa67a6947.png" class="latex" alt="$\frac{2}{3}$" style="vertical-align: -12px" width="11" height="37" />.
</p>
