You can view and practice with all of our contests, past and present, here with various modes.

Mode Descriptions

Competition Mode: The competition will be timed with the timer starting immediately after starting the test.

Challenger Mode: The problem will show up one at a time. No skipping is allowed, and only correctly answering a problem will proceed to the next. If an answer is incorrect, it will end immediately and reset the number of problems correct.

Practice Mode: The problem will show up to answer in an interactive page, but there are no time limits. Only available after the contest has ended.

Printable Version: This will redirect to a pdf of the test that you can print out. Only available after the contest has ended.
